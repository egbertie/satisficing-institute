# 满意解研究所 · 五路图腾方法体系 V1.3

---

## 一、方法体系概述

### 核心思想

五路图腾决策法是一套融合东西方智慧的合伙人决策框架，帮助创业者在信息不完备、时间有限的情况下，做出有据可依的合伙人选择决策。

### 理论基础

- **满意解理论**（赫伯特·西蒙）：在有限理性下追求"足够好"而非"最优"
- **儒商哲学**（黎红雷）：仁义礼智信在现代商业中的应用
- **脑科学**（方翊沣）：直觉与理性的协同决策

## 二、五路图腾详解

### 第一路：LIU（刘禹锡）· 惟吾德馨

**五行**：土  
**核心精神**：根基与信任  
**工整对仗**：聚贤才为伍，引智士同行  
**决策维度**：价值观对齐

**校准问题**：
1. 你当初创业的初心是什么？
2. 你最看重的合伙人品质是什么？
3. 你们的核心价值观是否一致？
4. 在利益冲突时，对方会如何选择？

**检查清单**：
- [ ] 彼此了解对方的创业动机
- [ ] 核心价值观无明显冲突
- [ ] 对"成功"的定义基本一致
- [ ] 人品和诚信度有可靠背书

---

### 第二路：SIMON（司马贺）· 满意解

**五行**：金  
**核心精神**：理性决策  
**工整对仗**：不求最优，但求最适；结果为本，满意为尺  
**决策维度**：理性评估

**校准问题**：
1. 你对合伙人的"满意标准"是什么？
2. 哪些条件是必须具备的？哪些是加分项？
3. 时间、资金、机会的约束条件是什么？
4. 什么情况下你会接受"足够好"而非"完美"？

**检查清单**：
- [ ] 明确了"必须具备"的条件
- [ ] 明确了"可以接受"的底线
- [ ] 充分考虑了资源约束
- [ ] 避免了分析瘫痪

---

### 第三路：GUANYIN（观自在）· 自在从容

**五行**：水  
**核心精神**：洞察与应变  
**工整对仗**：居方寸之地，以价值致远  
**决策维度**：状态校准

**校准问题**：
1. 你现在的身心状态如何？
2. 是否有足够的时间做决策？
3. 你是否受到了情绪或压力的影响？
4. 这个决策是否被过度焦虑或过度乐观所扭曲？

**检查清单**：
- [ ] 身体状态良好，非疲惫/生病时决策
- [ ] 有充足的时间思考，非匆忙决定
- [ ] 情绪平稳，非大喜大悲时决策
- [ ] 已排除外界过度干扰

---

### 第四路：CONFUCIUS（孔子）· 仁者爱人

**五行**：木  
**核心精神**：团队伦理  
**工整对仗**：（儒家伦理）  
**决策维度**：伦理契合

**校准问题**：
1. 你们的分工是否清晰？
2. 股权分配是否体现贡献和价值？
3. 决策权如何分配？
4. 退出机制是否提前约定？

**检查清单**：
- [ ] 股权、决策权、分红权有明确约定
- [ ] 退出机制已前置设计
- [ ] 冲突解决机制已明确
- [ ] 各方权利义务有书面确认

---

### 第五路：HUINENG（六祖慧能）· 顿悟

**五行**：火  
**核心精神**：直觉与创新  
**工整对仗**：（顿悟）  
**决策维度**：压力测试

**校准问题**：
1. 当你想到和这个人合伙时，直觉感受如何？
2. 如果明天公司遇到困难，你是否愿意和TA共渡难关？
3. 你们是否做过压力测试（如72小时极限挑战）？
4. 最坏情况下，你是否能接受这个选择？

**检查清单**：
- [ ] 直觉感受整体正向
- [ ] 已设想最坏情况并接受
- [ ] 已完成压力测试或深度磨合
- [ ] 准备立即行动而非继续犹豫

## 三、五步决策流程

```
Step 1: LIU校准（土）
    ↓
Step 2: SIMON校准（金）
    ↓
Step 3: GUANYIN校准（水）
    ↓
Step 4: CONFUCIUS校准（木）
    ↓
Step 5: HUINENG校准（火）
    ↓
【决策】
```

### 流程说明

1. **LIU校准**：先看人，再看事。价值观不对，一切免谈。
2. **SIMON校准**：明确标准，设定底线。知道什么是"足够好"。
3. **GUANYIN校准**：检查状态，排除干扰。不在状态差时做重大决策。
4. **CONFUCIUS校准**：理清规则，预防冲突。丑话说在前头。
5. **HUINENG校准**：直觉验证，准备行动。相信理性分析后的直觉。

## 四、应用场景

### 场景1：初次选择合伙人

按五步走完全流程，特别关注LIU（价值观）和CONFUCIUS（规则）。

### 场景2：合伙人冲突调解

重点看CONFUCIUS（规则是否清晰）和GUANYIN（情绪是否平稳）。

### 场景3：合伙人退出决策

重点看SIMON（满意解标准）和HUINENG（最坏情况接受度）。

### 场景4：新合伙人加入

全流程再走一遍，特别关注新老合伙人的价值观契合。

## 五、常见误区

### 误区1：只看能力，不看价值观
> **纠正**：能力可以培养，价值观难以改变。LIU校准是第一关。

### 误区2：追求完美合伙人
> **纠正**：没有完美的人，只有"足够好"的组合。SIMON校准帮你设定现实标准。

### 误区3：情绪冲动时决策
> **纠正**：大喜大悲不做重大决策。GUANYIN校准检查状态。

### 误区4：只谈感情，不谈规则
> **纠正**：先小人后君子。CONFUCIUS校准确保规则清晰。

### 误区5：过度分析，不敢行动
> **纠正**：分析是为了行动，不是为了逃避。HUINENG校准推动执行。
