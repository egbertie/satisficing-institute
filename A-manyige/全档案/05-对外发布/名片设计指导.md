# 名片设计指导 · 双面

## 整体规格
- 尺寸：90mm × 54mm（标准名片）
- 出血：各边3mm
- 分辨率：300dpi
- 色彩模式：CMYK

---

## 正面设计（满意解研究所为主）

### 布局结构
```
┌─────────────────────────────┐
│                             │
│    【鼎玉公司Logo】          │  ← 左上角，较小
│         (占位)              │
│                             │
│                             │
│                             │
│         【满意解             │  ← 中央偏上，最大
│           Logo】             │
│          (主体)             │
│                             │
│                             │
│   "吹尽黄沙始到金"          │  ← 中央，金句
│                             │
│                             │
│   满意解研究所               │  ← 底部，机构名
│   Satisfactory Solution Lab │
│                             │
│   专注深港硬科技合伙人决策教练 │  ← 最底部，定位
│                             │
└─────────────────────────────┘
```

### 元素说明

**1. 鼎玉公司Logo（左上角）**
- 尺寸：约12mm × 12mm
- 位置：距左边缘8mm，距上边缘8mm
- 透明度：80%（弱化，不抢主体）
- 标注："法律主体：鼎玉公司"（小字，8pt）

**2. 满意解Logo（中央主体）**
- 尺寸：约35mm × 35mm
- 位置：中央偏上
- 色彩：品牌主色（深海蓝#1a365d）
- 效果：简洁线条，可有轻微立体感

**3. 金句（中央）**
- 文字："吹尽黄沙始到金"
- 字体：书法体或宋体（站酷文艺体/方正清刻本悦宋）
- 字号：14pt
- 颜色：暖金色#d4af37
- 位置：Logo下方8mm

**4. 机构名（底部）**
- 中文："满意解研究所"
- 英文："Satisfactory Solution Lab"
- 中文字号：12pt，思源黑体Medium
- 英文字号：9pt，Avenir Light
- 颜色：深海蓝#1a365d

**5. 定位语（最底部）**
- 文字："专注深港硬科技合伙人决策教练"
- 字号：8pt
- 颜色：深灰#4a5568

## 背面设计（个人信息+联系方式）

### 布局结构
```
┌─────────────────────────────┐
│                             │
│   Egbertie                  │  ← 顶部，姓名
│   创始人 · 合伙人决策教练     │
│                             │
│   ─────────────────────     │  ← 分隔线
│                             │
│   【资质背书】               │  ← 中部
│   CBIIP │ CDAA              │
│                             │
│   ─────────────────────     │  ← 分隔线
│                             │
│   【联系方式】               │  ← 底部
│   📍 深圳市南山区XXX        │
│   📞 1XX-XXXX-XXXX          │
│   ✉️  contact@xxx.com       │
│                             │
└─────────────────────────────┘
```

### 元素说明

**1. 姓名（顶部）**
- 文字："Egbertie"
- 字体：Avenir Medium / 思源黑体Bold
- 字号：16pt
- 颜色：深海蓝#1a365d

**2. 职位**
- 文字："创始人 · 合伙人决策教练"
- 字号：10pt
- 颜色：深灰#4a5568

**3. 资质背书（中部）**
- 呈现方式：只放核心双认证
- 文字："CBIIP │ CDAA" 或 "商业直觉 · 数字资产"
- 字号：9pt
- 颜色：暖金色#d4af37
- 备注：其他资质口头介绍，名片上不放（避免拥挤）

**4. 联系方式（底部）**
- 地址："📍 深圳市南山区XXX"
- 电话："📞 1XX-XXXX-XXXX"
- 邮箱："✉️  contact@xxx.com"
- 字号：8pt
- 颜色：深灰#4a5568
- 图标：可用emoji或简洁线条图标

**5. 分隔线**
- 细线，0.5pt
- 颜色：浅灰#e2e8f0
- 长度：占宽度70%，居中
