## 二、详细操作步骤

### 步骤1：制作封面图

**方法A：使用AI生成**
1. 打开任意AI绘图工具（Midjourney/即梦/可灵等）
2. 输入以下提示词：
```
极简设计，干净排版，深海蓝和暖金色配色，
文字"22年双系统经验｜合伙人决策教练"，
专业商务风格，留白充足，无人物照片
```
3. 生成后下载，用美图秀秀裁剪为3:4比例

**方法B：使用Canva/稿定设计**
1. 打开Canva APP或网页
2. 搜索"小红书封面"模板
3. 选择简约商务风格
4. 修改文字为：
   - 主标题：吹尽黄沙始到金
   - 副标题：22年双系统经验｜合伙人决策教练
   - 底部：满意解研究所
5. 配色：深蓝#1a365d + 金色#d4af37
6. 导出3:4比例图片

### 步骤2：制作正文配图（4-6张）

**图1：22年经历对比**
- 左半：银行11年（图标：建筑/保险箱）
- 右半：创业11年（图标：火箭/图表）
- 底部：吹尽黄沙始到金

**图2：痛点数据**
- 大字：67%
- 小字：初创企业第二年合伙人冲突率
- 配色：红色警示

**图3：服务介绍**
- 标题：2小时1V1深度咨询
- 副标题：首批5个免费名额
- 底部：试运营启动

**图4：五路图腾**
- 五个图标排成一排
- 文字：惟吾德馨｜满意解｜自在从容｜仁者爱人｜红莲淬火

**图5：行动召唤**
- 大字：如果你面临合伙人选择困境
- 小字：评论区简要说明情况
- 底部：满意解研究所

**图6：个人形象（可选）**
- 专业形象照
- 文字：Egbertie｜合伙人决策教练

### 步骤3：发布笔记

1. 打开小红书APP
2. 点击底部"+"号
3. 选择"图文笔记"
4. 选择已制作好的图片（先选封面）
5. 点击"下一步"
6. 在文案区域粘贴以下文字：

```
【吹尽黄沙始到金｜试运营启动】

去年做了两件事：
✓ CBIIP商业直觉认证（右脑）
✓ CDAA数字资产分析师（左脑）

发现：AI替代不了的——人与人的关系，和最终决策。

【22年双系统】
前11年：银行体系→风控严谨性
后11年：创业实战→合伙人成败见证

【我能做什么】
不是帮你找人，是帮你做决策。
在AI时代，感知方案背后的"人"。

【试运营】
服务对象：技术已验证、产品已成型的硬科技创业者
首批5个名额｜2小时1V1｜邀约审核制

拟邀相关方向专家作为指导顾问。

如果你面临合伙人选择的关键决策，
评论区简要说明情况。

#合伙人决策 #硬科技创业 #创业教练 #试运营启动 #满意解
```

7. 添加话题标签（点击#号）：
   - #合伙人决策
   - #硬科技创业
   - #创业教练
   - #试运营启动
   - #满意解
   - #深圳创业
   - #合伙人选择

8. 点击"发布"

## 三、发布后跟进

**场景1：询问详情**
```
感谢关注！目前处于试运营阶段，首批开放5个免费体验名额。
如果您是技术已验证、产品已成型的硬科技创业者，
可以私信简要说明您的情况，我们会安排初步沟通。
```

```
感谢关注！目前试运营主要面向TRL 4-7阶段的硬科技创业者，
可能暂时不太适合您的情况。可以关注后续正式版发布。
```

**场景3：直接申请**
```
收到！请私信发送：
1. 当前创业阶段
2. 面临的核心决策
3. 期望通过这次交流获得什么

我们会尽快回复安排初步沟通。
```
