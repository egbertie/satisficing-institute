# 视频号视频脚本 · 3-5分钟版

## 完整脚本（5段式）

### 第一段：开场（30秒）

**画面**：主讲人正面，眼神看镜头

**台词**：
```
为什么同样的起点，合伙结局却天差地别？

我见过技术很牛的团队，因为合伙人选择失误，最终项目夭折。
我也见过一些"不完美的团队"，创造出惊人的价值。

我是Egbertie，满意解研究所创始人。
今天想分享我的22年观察。
```

---

### 第二段：银行视角（60秒）

**画面**：主讲人+银行相关B-roll（办公大楼、文件、数据图表）

**台词**：
```
2003年，我进入一家全国性商业银行。

11年里，我从总行到支行，完整跨越了组织层级。
参与过省级分行筹备，从0到1搭建业务体系。

那段时间，我训练出了三个本能：
第一，财务尽调的严谨性——每一个数字都要有出处。
第二，风险识别的系统性——不只看到机会，更要看到陷阱。
第三，退出机制的前置设计——在签约之前，就想好如何分手。

但银行有个局限：我们只负责说"不"，不负责说"怎么做"。
我看到太多优秀的技术创业者，因为合伙人问题最终折戟。
我当时无能为力，只能看着他们离开。
```

---

### 第三段：创业淬炼（60秒）

**画面**：主讲人+创业相关B-roll（办公室、团队、产品）

**台词**：
```
2014年，我离开银行，开始创业。

这11年，我深度参与了多个初创项目，
从种子轮到成长期，亲历了完整的创业周期。

我看到过估值3亿的成功退出，
也目睹过技术卓越但因为合伙人问题而困顿的项目。

有个项目，三个创始人一开始"兄弟情深"，
住在一起，吃在一起，每天工作16小时。
但18个月后，他们分家了。
不是因为业务不好，而是因为——
股权分配时埋下的隐患爆发了。

还有个相反的案例。
两个创始人，一个是技术大牛，一个是商业小白。
但他们从一开始就把"丑话说在前头"——
股权、决策权、退出机制，全部写清楚。
三年后，公司被收购，两人 still 是朋友。
```

---

### 第四段：方法论（60秒）

**画面**：主讲人+书籍/五路图腾图示

**台词**：
```
这些经历让我困惑了很久：
为什么同样的起点，结局如此不同？

直到我遇见赫伯特·西蒙的"满意解"理论。

西蒙说：人类决策者在信息不完备、时间有限、
认知有限的现实中，追求的是"足够好"，
而非"理论上最优"。

银行追求"最优解"，但真实世界没有最优解。
创业需要的是"满意解"——
在信息不全时，做出有据可依的决策。

这就是我开始整合五路图腾决策框架的原因。
从刘禹锡的"惟吾德馨"，到西蒙的"满意解"，
从孔子的"仁者爱人"，到六祖的"顿悟"。

东西方的智慧，在这个问题上交汇了。
```

---

### 第五段：服务介绍+结尾（30秒）

**画面**：主讲人+文字信息卡

**台词**：
```
现在，满意解研究所正式进入试运营阶段。

首批开放5个免费体验名额，
服务对象是技术已验证、产品已成型的硬科技创业者。

服务内容：2小时1V1面对面深度咨询。
获取方式：邀约制+审核制，确保彼此适合深度交流。

拟邀相关方向专家作为指导顾问。

如果你正在面临合伙人选择的关键决策，
欢迎私信我，简要说明你的情况。

我是Egbertie，
满意解研究所创始人。
```
