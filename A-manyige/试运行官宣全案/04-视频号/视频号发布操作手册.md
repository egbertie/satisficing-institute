## 二、视频制作详细步骤

### 步骤1：拍摄准备

**设备**：手机+三脚架/稳定器（建议横屏16:9）
**场景**：专业背景（办公室/书房/书架）
**着装**：商务正装或商务休闲
**灯光**：自然光+补光灯，面部明亮无阴影

### 步骤2：完整脚本（5段式）

**第一段：开场（30秒）**

画面：主讲人正面，眼神看镜头

台词：
```
为什么同样的起点，合伙结局却天差地别？

我见过技术很牛的团队，因为合伙人选择失误，最终项目夭折。
我也见过一些"不完美的团队"，创造出惊人的价值。

我是Egbertie，满意解研究所创始人。
今天想分享我的22年观察。
```

**第二段：银行视角（60秒）**

画面：可配合银行相关图片/图标

台词：
```
2003年，我进入一家全国性商业银行。

11年里，我从总行到支行，完整跨越了组织层级。
参与过省级分行筹备，从0到1搭建业务体系。

那段时间，我训练出了三个本能：
第一，财务尽调的严谨性——每一个数字都要有出处。
第二，风险识别的系统性——不只看到机会，更要看到陷阱。
第三，退出机制的前置设计——在签约之前，就想好如何分手。

但银行有个局限：我们只负责说"不"，不负责说"怎么做"。
我看到太多优秀的技术创业者，因为合伙人问题最终折戟。
我当时无能为力，只能看着他们离开。
```

**第三段：创业淬炼（60秒）**

画面：可配合创业相关图片/时间线

台词：
```
2014年，我离开银行，开始创业。

这11年，我深度参与了多个初创项目，
从种子轮到成长期，亲历了完整的创业周期。

我看到过估值3亿的成功退出，
也目睹过技术卓越但因为合伙人问题而困顿的项目。

有个项目，三个创始人一开始"兄弟情深"，
住在一起，吃在一起，每天工作16小时。
但18个月后，他们分家了。
不是因为业务不好，而是因为——
股权分配时埋下的隐患爆发了。

还有个相反的案例。
两个创始人，一个是技术大牛，一个是商业小白。
但他们从一开始就把"丑话说在前头"——
股权、决策权、退出机制，全部写清楚。
三年后，公司被收购，两人 still 是朋友。
```

**第四段：方法论（60秒）**

画面：可配合书籍封面/五路图腾图示

台词：
```
这些经历让我困惑了很久：
为什么同样的起点，结局如此不同？

直到我遇见赫伯特·西蒙的"满意解"理论。

西蒙说：人类决策者在信息不完备、时间有限、
认知有限的现实中，追求的是"足够好"，
而非"理论上最优"。

银行追求"最优解"，但真实世界没有最优解。
创业需要的是"满意解"——
在信息不全时，做出有据可依的决策。

这就是我开始整合五路图腾决策框架的原因。
从刘禹锡的"惟吾德馨"，到西蒙的"满意解"，
从孔子的"仁者爱人"，到六祖的"顿悟"。

东西方的智慧，在这个问题上交汇了。
```

**第五段：服务介绍+结尾（30秒）**

画面：主讲人+文字信息卡

台词：
```
现在，满意解研究所正式进入试运营阶段。

首批开放5个免费体验名额，
服务对象是技术已验证、产品已成型的硬科技创业者。

服务内容：2小时1V1面对面深度咨询。
获取方式：邀约制+审核制，确保彼此适合深度交流。

拟邀相关方向专家作为指导顾问。

如果你正在面临合伙人选择的关键决策，
欢迎私信我，简要说明你的情况。

我是Egbertie，
满意解研究所创始人。
```

### 步骤3：视频剪辑（使用剪映专业版）

1. 打开剪映专业版（电脑端）
2. 创建新项目，比例选择16:9
3. 导入拍摄素材

4. 粗剪：
   - 按5段式结构排列素材
   - 删除明显口误和停顿
   - 每段之间添加淡入淡出转场

5. 添加字幕：
   - 使用"识别字幕"功能
   - 字体：思源黑体 Regular
   - 字号：根据画面调整，确保清晰可读
   - 颜色：白色+黑色描边
   - 强调文字：金色#d4af37

6. 添加B-roll：
   - 银行阶段：办公大楼、文件、数据图表
   - 创业阶段：办公室、团队、产品
   - 方法论：书籍、思维导图、五路图腾图示

7. 添加BGM：
   - 音乐库搜索"商务"/"纪录片"/"轻音乐"
   - 推荐：《Experience》钢琴版
   - 音量：15%-25%，不掩盖人声

8. 添加片头片尾：
   - 片头：品牌Logo动画（2秒）
   - 片尾：联系方式+品牌Logo（3秒）

9. 调色：
   - 亮度+5
   - 对比度+3
   - 饱和度-2（更专业）

10. 导出：
    - 分辨率：1080P
    - 码率：中等
    - 格式：MP4

### 步骤4：封面制作

1. 使用Canva/稿定设计
2. 尺寸：16:9横版
3. 选择主讲人最专业的画面
4. 添加标题：
   - 主标题：吹尽黄沙始到金
   - 副标题：一个22年双系统亲历者的合伙人观察
   - 底部：满意解研究所｜Egbertie
5. 配色：深蓝#1a365d + 金色#d4af37

## 三、发布详细步骤

1. 打开微信APP
2. 点击"发现"→"视频号"
3. 点击右上角相机图标
4. 选择"从相册选择"
5. 选择已剪辑好的视频

6. 添加文案：
```
《吹尽黄沙始到金——一个22年双系统亲历者的合伙人观察》

满意解研究所试运营启动。
首批5个免费体验名额，技术已验证、产品已成型的硬科技创业者专属。

拟邀相关方向专家作为指导顾问。

如果你面临合伙人选择的关键决策，私信我简要说明情况。

#合伙人决策 #硬科技创业 #商业思维 #试运营
```

7. 添加话题（点击#号）：
   - #合伙人决策
   - #硬科技创业
   - #商业思维
   - #试运营

8. 添加位置（可选）：
   - 深圳·南山

9. 设置权限：
   - 公开发布
   - 允许评论

10. 点击"发表"
