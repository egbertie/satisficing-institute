**可选输入**：

```

```


**标准输出物**：
| 输出项 | 格式 | 内容要求 |

### 4.2 使用示例

**示例：两天一夜体验营训练设计**
```
用户输入：
"需要设计两天一夜体验营的'躯体信号识别'训练模块，目标是让学员
能够识别自己身体的直觉信号，并在决策中应用。学员为硬科技创业者，
时间约2小时。"

专家输出：
【躯体信号识别训练模块设计】

一、训练目标

认知目标：
- 理解躯体标记假说（Damasio理论）
- 识别至少3种身体信号与决策状态的关联

技能目标：
- 能够在决策前进行身体扫描
- 能够识别并记录自己的直觉信号

情感目标：
- 建立对身体智慧的信任
- 减少对纯理性分析的依赖

二、学员分析

背景特征：
- 硬科技创业者，高度理性思维训练
- 可能对"身体直觉"概念有抵触
- 时间宝贵，注重效率

学习风格：
- 偏好结构化、逻辑清晰的内容
- 需要科学证据支撑
- 重视可操作、可验证的方法

三、训练流程（2小时）

【Phase 1: 理论导入】（30分钟）
1. 开场："你的大脑在骗你"案例（5分钟）
   - 展示神经科学研究：身体比大脑更早感知风险
   
2. 躯体标记假说讲解（15分钟）
   - Damasio的 Iowa Gambling Task 研究
   - 身体信号与决策质量的关联证据
   
3. 信号类型介绍（10分钟）
   - 胃部紧缩 → 警惕/风险
   - 心口温暖 → 倾向/认同
   - 肩膀放松 → 信任/安全
   - 呼吸变浅 → 焦虑/防备

【Phase 2: 感知唤醒】（30分钟）
1. 身体扫描引导（15分钟）
   - 闭眼，从头顶到脚底依次扫描
   - 标记各部位的感受（紧张/放松/温暖/寒冷等）
   
2. 信号定位练习（15分钟）
   - 回忆一个"感觉不对劲"的决策场景
   - 识别当时身体的具体反应位置
   - 配对分享与验证

【Phase 3: 情境演练】（40分钟）
1. 决策情境模拟（20分钟）
   - 分组进行合伙人评估角色扮演
   - 过程中暂停，记录身体信号
   - 讨论信号与评估结论的关系
   
2. 信号-决策映射（20分钟）
   - 个人练习：一个真实决策场景
   - 身体扫描 → 信号识别 → 决策参考
   - 小组分享，导师反馈

【Phase 4: 整合应用】（20分钟）
1. 个人信号词典创建（10分钟）
   - 每人记录自己的3-5个独特信号
   - 标注信号含义与可信度
   
2. 行动计划（10分钟）
   - 如何在日常决策中应用
   - 信号日志的记录方法
   - 后续练习建议

四、教学方法

- 体验式学习：70%时间用于实践练习
- 同伴学习：小组分享与反馈
- 案例教学：真实创业者决策案例
- 科学支撑：每步都有研究证据

五、评估方式

过程评估：
- 参与度观察
- 练习完成质量

结果评估：
- 前测/后测：信号识别准确率
- 1周后随访：实际应用情况

六、所需资源

空间：
- 可容纳20人的安静房间
- 可调整灯光（身体扫描时需要较暗）

物料：
- 瑜伽垫/坐垫（每人一个）
- 信号记录表（打印）
- 白板与马克笔

人员：
- 主讲导师1人
- 助教1-2人（协助分组练习）

七、风险预案

可能问题：
- 学员对"身体直觉"抵触
  应对：强调科学证据，避免玄学化表述
  
- 身体扫描时情绪触发
  应对：助教巡视，提供支持；必要时暂停
  
- 时间不足
  应对：Phase 3可压缩为20分钟，确保核心练习

八、效果预期

即时效果：
- 90%学员能够识别至少3种身体信号
- 80%学员表示愿意在决策中尝试应用

短期效果（1周）：
- 60%学员开始在决策前进行身体扫描
- 信号日志完成率 > 50%

中期效果（1月）：
- 躯体信号成为决策常规参考
- 直觉决策信心提升
```

## S7: 对抗验证（失效场景测试）

### 7.1 失效场景识别

**场景一：心理触发**
```
失效模式：身体扫描触发创伤记忆或强烈情绪
识别信号：学员情绪失控、哭泣、颤抖
熔断机制：立即停止练习，提供支持，必要时转介心理咨询
```

**场景二：方法抵触**
```
失效模式：学员强烈抵触"身体直觉"概念
识别信号：公开质疑、拒绝参与、负面反馈
熔断机制：调整教学方法，增加科学证据，尊重个体边界
```

**场景三：效果不佳**
```
失效模式：多次训练后仍无感知能力提升
识别信号：持续报告无法识别身体信号
熔断机制：排查生理因素，调整训练强度，考虑个体差异
```

### 7.2 熔断机制

- 学员情绪失控 → 立即暂停，人工介入
- 涉及医学问题 → 建议医疗咨询
- 训练效果持续不佳 → 个性化诊断与调整
