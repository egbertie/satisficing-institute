## S2: 系统考虑（输入→处理→输出→反馈闭环）

### 2.1 输入规范

**必需输入**：
| 输入项 | 格式 | 说明 |
|--------|------|------|
| 睡眠状况 | 问卷+数据 | 睡眠时长、质量自评、问题描述 |
| 生理数据 | 可穿戴设备 | 心率、HRV、睡眠周期（如有） |
| 训练目标 | 枚举值 | 睡眠优化/躯体感知/压力管理 |

**可选输入**：
- 压力水平自评
- 决策场景描述
- 既往训练经历

### 2.2 处理流程

**阶段一：基线评估（感知·觉察）**
```
问卷数据 → 睡眠效率计算 → 生理指标分析 → 问题诊断
```

**阶段二：方案设计（感知·调节）**
```
诊断结果 → 环境优化建议 → 行为干预方案 → 训练计划
```

**阶段三：反馈训练（感知·提升）**
```
训练执行 → 数据监测 → 效果评估 → 方案调整
```

### 2.3 输出规范

**标准输出物**：
| 输出项 | 格式 | 内容要求 |
|--------|------|----------|
| 睡眠评估报告 | Markdown | 睡眠质量评分+问题诊断+改善建议 |
| 优化方案 | 清单 | 环境调整+行为习惯+营养补充 |
| 训练计划 | 日程表 | 每日训练内容+时间安排 |
| 进度跟踪 | 图表 | 关键指标趋势图 |

### 2.4 反馈机制

- 睡眠质量改善跟踪（每日/每周）
- 躯体信号识别准确率验证
- 决策质量主观评估

### 2.5 故障处理

| 故障类型 | 识别信号 | 处理机制 |
|----------|----------|----------|
| 方案无效 | 2周后无改善 | 调整方案强度，排查其他因素 |
| 执行困难 | 客户反馈难以坚持 | 简化方案，降低门槛 |
| 症状加重 | 睡眠/情绪恶化 | 建议医学检查 |


### 4.2 使用示例

**示例：融资前睡眠优化方案**
```
用户输入：
"下周要与投资方进行关键谈判，最近一周压力大，每天只能睡4-5小时，
入睡困难，早醒。希望能在这周调整好状态。"

专家输出：
【融资前睡眠优化方案】

一、睡眠问题诊断

主要问题：
- 入睡困难（压力性失眠）
- 睡眠时长不足（4-5小时 < 推荐7-8小时）
- 早醒（皮质醇水平过早升高）

生理机制：
压力 → 皮质醇升高 → 抑制褪黑素分泌 → 睡眠-觉醒周期紊乱

二、7天紧急优化方案

Day 1-2：环境调整期
- 卧室温度调至18-20℃（核心体温下降促进睡眠）
- 使用遮光窗帘，确保完全黑暗（促进褪黑素分泌）
- 移除电子设备（蓝光抑制褪黑素）
- 使用白噪音或粉红噪音（降低环境干扰）

Day 3-4：行为干预期
- 固定就寝时间（22:30-23:00），即使睡不着也躺下
- 睡前1小时"数字戒断"（无手机、电脑）
- 睡前30分钟温水泡脚（促进血液循环，帮助散热）
- 尝试4-7-8呼吸法（吸气4秒-屏息7秒-呼气8秒）

Day 5-7：认知优化期
- "担忧时间"设定：睡前1小时写下所有担忧与待办，心理上"存档"
- 正念身体扫描：躺下后从头到脚依次放松各部位
- 若30分钟未入睡，起床做轻度阅读（纸质书，非电子），有睡意再回床

三、营养补充建议

- 晚餐后：甘菊茶或缬草根茶（天然助眠）
- 睡前1小时：少量坚果（提供色氨酸）
- 避免：咖啡因（下午2点后）、酒精（影响睡眠结构）、高糖食物

四、预期效果

- Day 3-4：入睡时间缩短至20-30分钟
- Day 5-7：睡眠时长延长至6-7小时
- 谈判日：精力恢复度提升40-50%

五、注意事项

⚠️ 若3天后仍无改善，建议咨询睡眠专科医生
⚠️ 避免使用处方安眠药（影响认知清晰度）
⚠️ 谈判当天可适量使用咖啡因，但避免过量（焦虑反增）
```

### 4.3 限制与边界

**医学边界**：
- 不提供医学诊断
- 不替代专业医疗建议
- 严重睡眠障碍建议就医

**适用边界**：
- 建议基于一般性研究，个体差异存在
- 紧急优化方案（7天）与长期方案（3个月+）效果不同

## S7: 对抗验证（失效场景测试）

### 7.1 失效场景识别

**场景一：医学问题**
```
失效模式：客户存在未诊断的睡眠障碍（如睡眠呼吸暂停）
识别信号：严重打鼾、白天极度嗜睡、方案无效
熔断机制：建议医学检查，暂停非医学建议
```

**场景二：心理原因**
```
失效模式：失眠源于焦虑/抑郁等心理问题
识别信号：持续情绪低落、兴趣丧失、方案无效
熔断机制：建议心理咨询/精神科评估
```

### 7.2 熔断机制

- 客户报告严重症状 → 自动标注"需医学评估"
- 方案2周无效 → 建议专业医疗咨询
