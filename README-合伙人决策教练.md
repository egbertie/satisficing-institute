## 🚀 快速开始（推荐流程）

### 第一步：理解框架（30分钟）
阅读《硬件初创企业合伙人选择决策框架》，重点理解：
- 硬件创业合伙关系的特殊性
- 左脑逻辑评估的5个维度
- 右脑直觉感知的4个层面

### 第二步：准备评估（1小时）
1. 根据自身情况，调整《对比分析模板》中的权重
2. 列出候选人名单（建议 3-5 人）
3. 准备《深度访谈问题清单》，安排访谈时间

### 第三步：执行评估（2-4周）
1. **初筛**：用左脑逻辑快速排除明显不匹配者
2. **深度访谈**：与 2-3 位候选人进行 2-3 轮深度对话
3. **打分对比**：填写对比分析模板，计算综合得分
4. **直觉校验**：独处时审视对每个候选人的「身体感受」

### 第四步：决策与验证（1-3月）
1. 做出选择，进入小规模合作测试
2. 设定验证期的明确目标与退出机制
3. 测试通过后，正式确立合伙关系

---

## 📊 决策质量检查清单

在最终决策前，确认以下问题：

- [ ] 是否至少访谈了 3 位候选人？
- [ ] 左脑逻辑评分是否都有证据支撑？
- [ ] 是否进行了压力情境测试？
- [ ] 直觉感受是否与逻辑评分一致？
- [ ] 是否有任何红灯信号被忽略？
- [ ] 家人/导师是否了解这个决定？
- [ ] 退出机制是否已经明确？
- [ ] 想象5年后，还愿意和这个人共事吗？
