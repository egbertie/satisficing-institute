## 📝 使用示例

### 示例1: 现场评估参考
**输入**:
```
快速查询：评估综合分7.2分，建议什么决策？
```

**输出**:
```
📊 评估结果速查

综合评分: 7.2分
风险等级: 🟡 LOW (低风险)
建议决策: 可以推进

### 具体建议
1. **总体评价**: 合伙人匹配度良好，具备合作基础
2. **关注要点**:
   - 建议在合作中持续观察价值观一致性
   - 建议建立明确的责任分工机制
   - 建议预设退出机制
3. **风险提示**: 当前评分在安全区间，但仍需定期复评

### 下一步行动
- ✅ 可以进入详细合作条款讨论
- ✅ 建议安排深度尽调
- ⚠️ 建议1个月后进行复评
```

### 示例2: 培训讲解使用
**输入**:
```
显示7维度评估的完整说明，用于团队培训
```

**输出**:
```
# 📚 7维度评估体系 - 培训版

## 评估逻辑
满意解合伙人评估基于7个核心维度，每个维度有明确的权重和评分标准。

## 维度详解

### 1️⃣ 价值观契合度 (25%)
**为什么最重要**: 价值观冲突是合伙人分手的首要原因(67%)
**评估要点**: 创业理念、工作态度、道德底线
**关键阈值**: <5分 = 致命风险

### 2️⃣ 能力互补性 (20%)
**为什么重要**: 硬科技需要技术+商业双轮驱动
**评估要点**: 技术能力、商业能力、资源网络
**关键阈值**: <7分 = 需补强

### 3️⃣ 沟通效率 (15%)
**评估要点**: 信息透明度、响应速度、决策效率
**关键阈值**: <5分 = 中风险

### 4️⃣ 承诺可信度 (15%)
**评估要点**: 历史信用、投入意愿、兑现能力
**关键阈值**: <6分 = 高风险

### 5️⃣ 利益一致性 (10%)
**评估要点**: 股权分配、利益分配、长期利益
**注意**: 需机制保障

### 6️⃣ 退出可接受性 (10%)
**评估要点**: 退出机制、股权回购、竞业禁止
**注意**: 必须预设

### 7️⃣ 成长匹配度 (5%)
**评估要点**: 学习能力、适应性、发展潜力
**注意**: 长期观察

## 评分方法
每个维度1-10分，按权重计算加权平均分

## 实战提醒
- 7.2分的案例成功率约75%
- 6.5分是成败分界线
- 价值观<5分一票否决
```
