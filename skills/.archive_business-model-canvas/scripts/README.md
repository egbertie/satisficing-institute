# 商业模式画布 Scripts

## 脚本说明

### business-model-canvas-runner.py
商业模式画布主执行脚本。

## 功能

- create
- analyze
- validate

## 使用方法

```bash
# 执行功能
python3 business-model-canvas-runner.py run --feature create

# 查看状态
python3 business-model-canvas-runner.py status

# 生成报告
python3 business-model-canvas-runner.py report
```
