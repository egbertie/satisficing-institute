# 局限说明

## 已知局限

### 1. 语音识别局限 (S6.1)

| 局限 | 影响 | 建议 |
|------|------|------|
| 方言/口音 | 识别率下降20-40% | 提供文本校对界面 |
| 背景噪音 | 误识别增加 | 建议使用降噪音频 |
| 多人同时说话 | 转录混乱 | 标注需人工确认 |

### 2. 语言理解局限 (S6.2)

| 局限 | 示例 | 处理方式 |
|------|------|----------|
| 隐晦表达 | "那个事要处理一下" | 标注为"含义不明" |
| 讽刺语气 | "真是太好了" | 按字面意思处理 |
| 专业术语 | 行业黑话 | 支持自定义词库 |

### 3. 格式处理局限 (S6.3)

| 局限 | 说明 |
|------|------|
| 手写笔记 | 需先进行OCR识别 |
| 扫描件 | 图片质量影响识别率 |
| 复杂表格 | 表格结构可能丢失 |

### 4. 技术局限 (S6.4)

| 局限 | 说明 |
|------|------|
| 无法联网 | 离线运行，无法实时查询 |
| 上下文有限 | 单次处理，不保留历史上下文 |
| 无情感分析 | 无法判断语气、情绪 |

## 自定义词库

用户可通过 `config.json` 添加专业术语：

```json
{
  "limitations": {
    "custom_terms": {
      "OKR": "目标与关键结果",
      "KPI": "关键绩效指标",
      "ROI": "投资回报率"
    },
    "person_aliases": {
      "小李": "李明",
      "老王": "王建国"
    }
  }
}
```

## 局限标注示例

当系统检测到局限时，会在输出中标注：

```markdown
## ⚠️ 处理局限说明

1. **语音识别局限** (置信度 65%)
   - 检测到非标准口音，部分词汇可能识别错误
   
2. **模糊表述** (2处)
   - "那个文档" → 未明确具体文档
   - "跟他们说" → 未明确"他们"指谁
```
