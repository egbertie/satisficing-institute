# AI会议笔记 Scripts

## 脚本说明

### ai-meeting-notes-runner.py
AI会议笔记主执行脚本。

## 功能

- record
- transcribe
- summarize
- action

## 使用方法

```bash
# 执行功能
python3 ai-meeting-notes-runner.py run --feature record

# 查看状态
python3 ai-meeting-notes-runner.py status

# 生成报告
python3 ai-meeting-notes-runner.py report
```
