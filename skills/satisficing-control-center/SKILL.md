## 核心功能

### 1. Skill注册中心
```yaml
功能:
  - 新Skill必须通过5标准检验
  - 自动检查重复功能 (相似度>80%拒绝)
  - 分配统一ID和分类
  - 注册到知识图谱
```

### 2. Skill生命周期管理
```yaml
状态流转:
  创建 → 5标准检验 → 试运行 → 正式运行 → 监控 → 退役/归档
  
健康度检查:
  - 30天未调用: 标记闲置
  - 90天未调用: 建议归档
  - 1年未更新: 强制评估
```

### 3. Cron统一调度
```yaml
功能:
  - 所有Cron集中管理
  - 自动准点错开
  - 任务合并建议
  - 拥堵预警
```

### 4. Token预算中心
```yaml
功能:
  - 全局Token预算分配
  - 实时监控和预警
  - 成本效益分析
  - 超额熔断
```

### 5. 知识图谱集成
```yaml
功能:
  - Skill能力自动索引
  - 智能推荐最佳Skill
  - 避免重复建设
  - 能力暴露接口
```

## 治理规则 (军规15条)

### 原有军规 (1-10)
1. 绝对诚实
2. 极致精益
3. 永不停滞
4. ...

### 新增军规 (11-15)
11. **Skill不重复**: 新Skill必须通过重复检测，相似度>80%拒绝创建
12. **Cron不拥堵**: 所有定时任务必须经过统一调度中心，禁止私自配置
13. **能力不孤岛**: Skill必须注册到知识图谱，暴露能力接口
14. **Token不浪费**: 每个Skill必须标注Token消耗预估，超预算自动熔断
15. **效果不虚报**: Skill必须提供效果验证数据，无法验证视为未达标

## Commands

### Skill管理
- scc skill list - 列出所有Skill
- scc skill register [name] - 注册新Skill
- scc skill check [name] - 健康度检查
- scc skill archive [name] - 归档Skill

### Cron管理
- scc cron list - 列出所有Cron
- scc cron schedule [task] [time] - 分配时间
- scc cron optimize - 优化时间分布

### Token管理
- scc token budget - 查看预算
- scc token usage - 查看消耗
- scc token alert - 设置预警

### 知识图谱
- scc kg index - 索引所有Skill
- scc kg query [keyword] - 查询能力
- scc kg stats - 统计信息
