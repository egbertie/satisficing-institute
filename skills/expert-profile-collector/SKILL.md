# Expert Profile Collector Skill

## Purpose
收集并维护6位专家的真实资料

## 5-Standard Compliance

| Standard | Implementation |
|----------|----------------|
| 全局考虑 | 覆盖资料需求→收集→验证→归档→更新全流程 |
| 系统考虑 | 需求清单→多渠道收集→交叉验证→档案更新闭环 |
| 迭代机制 | 资料更新触发复核，定期验证准确性 |
| Skill化 | 标准化接口：identify/collect/verify/archive |
| 自动化 | 每周检查资料完整度，自动触发收集任务 |

## Commands
- `identify` - 识别缺失资料
- `collect` - 收集资料
- `verify` - 验证资料准确性
- `archive` - 归档资料
