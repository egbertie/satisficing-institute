# 持续改进 Scripts

## 脚本说明

### continuous-improvement-runner.py
持续改进主执行脚本。

## 功能

- analyze
- suggest
- track

## 使用方法

```bash
# 执行功能
python3 continuous-improvement-runner.py run --feature analyze

# 查看状态
python3 continuous-improvement-runner.py status

# 生成报告
python3 continuous-improvement-runner.py report
```
