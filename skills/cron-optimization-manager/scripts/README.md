# Cron优化管理器 Scripts

## 脚本说明

### cron-optimization-manager-runner.py
Cron优化管理器主执行脚本。

## 功能

- merge
- analyze
- optimize

## 使用方法

```bash
# 执行功能
python3 cron-optimization-manager-runner.py run --feature merge

# 查看状态
python3 cron-optimization-manager-runner.py status

# 生成报告
python3 cron-optimization-manager-runner.py report
```
