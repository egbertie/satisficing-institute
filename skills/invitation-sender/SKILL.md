# Invitation Sender Skill

## Purpose
管理专家邀请函的发送与跟进

## 5-Standard Compliance

| Standard | Implementation |
|----------|----------------|
| 全局考虑 | 覆盖草稿→审核→发送→跟进全流程 |
| 系统考虑 | 模板选择→内容定制→发送→回复跟踪闭环 |
| 迭代机制 | 根据回复率优化模板和发送时机 |
| Skill化 | 标准化接口：draft/preview/send/followup |
| 自动化 | 获取联系方式后自动触发发送流程 |

## Commands
- `draft` - 生成邀请函草稿
- `preview` - 预览邀请函
- `send` - 发送邀请函
- `followup` - 跟进未回复
