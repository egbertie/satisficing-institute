# Unified Memory Suite

**统一记忆与上下文管理套件** - 整合记忆审计、承诺跟踪、自我优化。

> 🎯 替代: memory-auditor + promise-system-guardian + role33-task-manager + self-optimization-and-inspection

---

## 核心能力

| 功能模块 | 覆盖场景 |
|---------|---------|
| **记忆管理** | 上下文保持、记忆检索、遗忘策略 |
| **承诺跟踪** | 任务承诺、截止日期、履约监控 |
| **角色管理** | 角色切换、上下文隔离、状态恢复 |
| **自我优化** | 性能分析、行为改进、反馈循环 |
| **审计报告** | 记忆使用、承诺履行、优化效果 |

---

## 快速开始

```bash
# 记忆审计
memory-suite audit --scope session --report usage

# 承诺跟踪
memory-suite promise add --what "完成报告" --when 2024-04-01 --priority high
memory-suite promise list --status pending

# 角色切换
memory-suite role switch --to analyst --context preserve

# 自我优化
memory-suite optimize --analyze patterns --suggest improvements
```
