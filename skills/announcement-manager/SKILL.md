# Announcement Manager Skill

## Purpose
管理满意解研究所官宣内容的准备与发布

## 5-Standard Compliance

| Standard | Implementation |
|----------|----------------|
| 全局考虑 | 覆盖需求→草稿→审核→发布→复盘全流程 |
| 系统考虑 | 内容策划→文案撰写→视觉设计→发布执行闭环 |
| 迭代机制 | 发布后收集反馈，优化下次官宣 |
| Skill化 | 标准化接口：plan/draft/review/publish/analyze |
| 自动化 | 截止前3天自动预警，每日进度报告 |

## Commands
- `plan` - 制定官宣计划
- `draft` - 撰写文案
- `review` - 审核内容
- `publish` - 执行发布
- `analyze` - 效果分析
