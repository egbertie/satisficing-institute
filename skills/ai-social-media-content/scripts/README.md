# 社交媒体内容生成 Scripts

## 脚本说明

### ai-social-media-content-runner.py
社交媒体内容生成主执行脚本。

## 功能

- content
- calendar
- publish

## 使用方法

```bash
# 执行功能
python3 ai-social-media-content-runner.py run --feature content

# 查看状态
python3 ai-social-media-content-runner.py status

# 生成报告
python3 ai-social-media-content-runner.py report
```
