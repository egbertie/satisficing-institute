# xhs-note-creator Skill V5标准版本

## S1: 全局考虑

### 输入
- 产品/主题信息
- 目标受众画像
- 内容风格要求
- 图片需求（可选）

### 覆盖维度
| 维度 | 考虑内容 |
|------|----------|
| **人** | 小红书用户（18-35岁女性为主）、品牌方、创作者 |
| **事** | 笔记创作、标题优化、标签选择、封面建议 |
| **物** | 标题、正文、标签、封面图、互动引导 |
| **环境** | 小红书平台规则、流量机制、审核标准 |
| **外部集成** | 图片生成API（可选）、热点数据 |
| **边界情况** | 违规词、限流风险、同质化内容 |

---

## S2: 系统考虑

### 处理流程
```
需求分析 → 热点研究 → 标题创作 → 正文撰写 → 标签优化 → 封面建议 → 合规检查
```

### 故障处理
- **需求模糊**: 提供模板引导
- **违规风险**: 自动检测并提示修改
- **同质化**: 建议差异化角度
- **图片问题**: 提供替代方案

---

## S3: 输出规范

### 笔记输出格式
```
【标题选项】（3个）
1. [情绪化标题]
2. [干货标题]
3. [悬念标题]

【正文结构】
开头钩子（3行内）
核心内容（分段、emoji）
结尾引导（互动/关注）

【标签组合】
#大流量标签 #精准标签 #品牌标签

【封面建议】
- 风格：[建议]
- 配色：[建议]
- 文字：[建议]

【发布时间】
建议：[周X XX:XX]
理由：[流量高峰说明]

【风险提示】
- 需规避：[如有]
- 建议调整：[如有]
```

---

## S4: 自动化集成

### 内容类型
| 类型 | 说明 |
|------|------|
| 种草笔记 | 产品推荐 |
| 干货笔记 | 知识分享 |
| 测评笔记 | 产品对比 |
| 故事笔记 | 经历分享 |
| 探店笔记 | 地点推荐 |

---

## S5: 自我验证

### 质量指标
- 标题吸引力（点击率预估）
- 内容原创度
- 平台合规性
- 互动引导有效性

---

## S6: 认知谦逊

### 局限
- 无法保证爆款
- 不了解最新算法变化
- 需要人工最终审核
- 热点时效性有限

---

## S7: 对抗测试

| 场景 | 预期行为 |
|------|----------|
| 敏感品类（医美/金融） | 提示合规风险，建议调整 |
| 要求虚假宣传 | 拒绝并解释原因 |
| 竞品对比 | 客观描述，避免贬低 |
| 内容被限流 | 分析原因，建议修改 |

---

目标：25-35岁创业者
