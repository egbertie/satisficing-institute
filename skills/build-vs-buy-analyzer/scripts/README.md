# 自建vs采购分析器 Scripts

## 脚本说明

### build-vs-buy-analyzer-runner.py
自建vs采购分析器主执行脚本。

## 功能

- analyze
- compare
- recommend

## 使用方法

```bash
# 执行功能
python3 build-vs-buy-analyzer-runner.py run --feature analyze

# 查看状态
python3 build-vs-buy-analyzer-runner.py status

# 生成报告
python3 build-vs-buy-analyzer-runner.py report
```
