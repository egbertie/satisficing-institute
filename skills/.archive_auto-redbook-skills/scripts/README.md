# 小红书自动化 Scripts

## 脚本说明

### auto-redbook-skills-runner.py
小红书自动化主执行脚本。

## 功能

- generate
- post
- analyze

## 使用方法

```bash
# 执行功能
python3 auto-redbook-skills-runner.py run --feature generate

# 查看状态
python3 auto-redbook-skills-runner.py status

# 生成报告
python3 auto-redbook-skills-runner.py report
```
