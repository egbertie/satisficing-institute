# Vendor API Monitor Skill
> **厂商API能力监控器** - 追踪钉钉、企业微信等替代方案

## 功能概述

监控各大厂商（钉钉、企业微信、飞书、Notion）的API能力，特别关注：
1. **文件同步到API**（P0-最重要）
2. 自动记录
3. 到期前预警
4. 超期自动补救

## 核心需求（按优先级）

| 优先级 | 需求 | 说明 |
|--------|------|------|
| P0 | **文件同步到API** | 可编程上传文件到云端 |
| P1 | 自动记录 | 任务完成自动记录状态 |
| P2 | 到期前预警 | 截止日期前自动提醒 |
| P3 | 超期自动补救 | 超时自动触发补救流程 |
