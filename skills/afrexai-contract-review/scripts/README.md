# 合同分析工具 Scripts

## 脚本说明

### afrexai-contract-review-runner.py
合同分析工具主执行脚本。

## 功能

- analyze
- risk-check
- summary

## 使用方法

```bash
# 执行功能
python3 afrexai-contract-review-runner.py run --feature analyze

# 查看状态
python3 afrexai-contract-review-runner.py status

# 生成报告
python3 afrexai-contract-review-runner.py report
```
