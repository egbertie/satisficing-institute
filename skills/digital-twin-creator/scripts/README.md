# 数字孪生创建器 Scripts

## 脚本说明

### digital-twin-creator-runner.py
数字孪生创建器主执行脚本。

## 功能

- create
- sync
- simulate

## 使用方法

```bash
# 执行功能
python3 digital-twin-creator-runner.py run --feature create

# 查看状态
python3 digital-twin-creator-runner.py status

# 生成报告
python3 digital-twin-creator-runner.py report
```
