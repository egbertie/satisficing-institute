# 备份与灾难恢复 Scripts

## 脚本说明

### backup-disaster-recovery-runner.py
备份与灾难恢复主执行脚本。

## 功能

- backup
- restore
- verify

## 使用方法

```bash
# 执行功能
python3 backup-disaster-recovery-runner.py run --feature backup

# 查看状态
python3 backup-disaster-recovery-runner.py status

# 生成报告
python3 backup-disaster-recovery-runner.py report
```
