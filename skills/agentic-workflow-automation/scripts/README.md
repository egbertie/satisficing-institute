# 智能工作流自动化 Scripts

## 脚本说明

### agentic-workflow-automation-runner.py
智能工作流自动化主执行脚本。

## 功能

- workflow
- automation
- task

## 使用方法

```bash
# 执行功能
python3 agentic-workflow-automation-runner.py run --feature workflow

# 查看状态
python3 agentic-workflow-automation-runner.py status

# 生成报告
python3 agentic-workflow-automation-runner.py report
```
