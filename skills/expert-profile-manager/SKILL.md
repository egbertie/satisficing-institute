# 专家档案管理系统

满意解研究所核心工具 - 管理合伙人匹配决策专家网络。

## 核心功能

### 1. 专家档案管理
- 专家基础信息（姓名、职位、单位、联系方式）
- 专业领域标签（儒商哲学、技术转移、自贸政策等）
- 图腾关联（五行、角色定位）
- 数字替身设定（基于公开著作/论文）

### 2. 咨询记录追踪
- 咨询日期、主题、形式（线上/线下）
- 核心建议摘要
- 后续行动项

### 3. 专家网络可视化
- 五路图腾体系关联图
- 专家协作关系网络
- 咨询频率统计

## 使用方法

```bash
# 查看专家档案
./expert.sh list

# 查看特定专家
./expert.sh show 黎红雷

# 添加咨询记录
./expert.sh log 黎红雷 "2026-03-15" "合伙伦理框架讨论"

# 生成专家网络图
./expert.sh visualize
```

## 数据存储

- 档案文件：`experts/` 目录下 JSON 格式
- 咨询记录：`logs/` 目录下按日期归档
- 备份：自动同步到 GitHub

## 集成

- 飞书：自动推送咨询提醒
- Notion：双向同步专家资料
