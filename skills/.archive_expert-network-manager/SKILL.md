# Expert Network Manager Skill

## Purpose
管理6位专家数字替身的关系维护

## 5-Standard Compliance

| Standard | Implementation |
|----------|----------------|
| 全局考虑 | 覆盖发现→接触→邀请→维护全生命周期 |
| 系统考虑 | 资料管理→互动记录→关系维护闭环 |
| 迭代机制 | 每月更新专家档案，记录互动历史 |
| Skill化 | 标准化接口：add/update/engage/report |
| 自动化 | 每周扫描专家动态，生成关系报告 |

## Commands
- `add` - 添加新专家
- `update` - 更新专家资料
- `engage` - 记录互动
- `report` - 生成关系报告
