# 经济情报 Scripts

## 脚本说明

### economic-intelligence-runner.py
经济情报主执行脚本。

## 功能

- collect
- analyze
- forecast

## 使用方法

```bash
# 执行功能
python3 economic-intelligence-runner.py run --feature collect

# 查看状态
python3 economic-intelligence-runner.py status

# 生成报告
python3 economic-intelligence-runner.py report
```
