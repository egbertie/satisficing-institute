## 🎯 协调判断能力说明

### 风险分数计算
```python
过期任务 >4小时    → +10分/项
过期任务 <4小时    → +5分/项  
阻塞任务 >2天      → +3分/项
风险任务 <12小时   → +4分/项
```

### 模式选择逻辑
| 风险分数 | 模式 | 行为 |
|---------|------|------|
| ≥10 或有逾期 | sequential | 全力补救，暂停后台 |
| 只有阻塞 | notify_user | 批量确认，等待用户 |
| =0 无问题 | parallel | 并行推进，效率最大 |

### 学习迭代
- **每次决策**: 自动记录上下文和结果
- **每小时**: 分析历史决策效果
- **识别模式**: 如"飞书调试陷阱"
- **优化策略**: 自动调整阈值和权重
