# Task Coordinator Skill
> **任务协调管理Skill** - 确保正常计划和临时任务的统筹协调，永不掉链子

## 功能概述

本Skill负责任务的智能协调管理，核心能力：

1. **工作负载分析** - 实时分析当前任务状态（正常/临时/过期/阻塞）
2. **执行模式切换** - 自动切换顺序/并行/通知用户模式
3. **遗漏检测** - 自动检查日志，发现遗漏任务
4. **预警机制** - 任务到期前主动提醒

## 执行模式

### 模式1: Sequential（顺序执行）
**触发条件**: 有过期任务或高优先级临时任务
**行为**: 
- 暂停后台任务
- 全力处理过期/紧急任务
- 完成后恢复正常节奏

### 模式2: Parallel（并行执行）
**触发条件**: 无临时任务，无过期任务
**行为**:
- 正常计划任务并行推进
- 后台任务自主执行
- 效率最大化

### 模式3: Notify User（通知用户）
**触发条件**: 任务被用户阻塞
**行为**:
- 汇总所有阻塞项
- 一次性向用户确认
- 避免反复打扰
