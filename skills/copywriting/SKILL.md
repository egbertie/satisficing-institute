# copywriting Skill V5标准版本

## S1: 全局考虑

### 输入
- 文案类型（营销文案、广告标题、产品描述）
- 目标受众描述
- 品牌调性要求
- 字数限制

### 覆盖维度
| 维度 | 考虑内容 |
|------|----------|
| **人** | 营销人员、创业者、内容创作者 |
| **事** | 营销文案、广告创意、产品描述、社媒内容 |
| **物** | 标题、正文、CTA、标签 |
| **环境** | 平台特性（小红书/公众号/抖音）、行业语境 |
| **外部集成** | 无（纯内容生成） |
| **边界情况** | 敏感词、平台限制、品牌冲突 |

---

## S2: 系统考虑

### 处理流程
```
需求分析 → 受众画像 → 创意生成 → 多版本输出 → 优化建议
```

### 故障处理
- **需求模糊**: 反问澄清，提供模板
- **生成不符合**: 重新生成或人工调整
- **敏感内容**: 自动检测并提示

---

## S3: 输出规范

### 文案输出格式
```
【标题选项】
1. [标题1]
2. [标题2] 
3. [标题3]

【正文】
[正文内容]

【CTA】
[行动号召]

【标签建议】
#标签1 #标签2 #标签3

【使用建议】
- 适用平台: [平台]
- 最佳发布时间: [时间]
- A/B测试建议: [建议]
```

---

## S4: 自动化集成

### 支持文案类型
| 类型 | 说明 | 示例 |
|------|------|------|
| 营销文案 | 产品推广 | 新品上市文案 |
| 广告标题 | 吸引点击 | 信息流广告标题 |
| 产品描述 | 电商详情 | 商品详情页描述 |
| 社媒内容 | 平台适配 | 小红书/抖音文案 |
| 邮件主题 | 邮件营销 | 邮件标题+预览 |

---

## S5: 自我验证

### 质量指标
- 吸引力评分: 人工评估
- 平台适配度: 符合平台调性
- 原创性: 避免模板化

### 检查清单
- [ ] 标题吸引人
- [ ] 正文有逻辑
- [ ] CTA明确
- [ ] 无敏感词
- [ ] 符合平台调性

---

## S6: 认知谦逊

### 局限
- 无法替代人类创意
- 不了解具体品牌历史
- 可能生成通用内容
- 需要人工最终审核

---

## S7: 对抗测试

| 场景 | 预期行为 |
|------|----------|
| 需求过于宽泛 | 提供模板选项，引导细化 |
| 要求违法违规内容 | 拒绝并解释原因 |
| 品牌冲突 | 提示可能风险 |
| 多次不满意 | 建议人工创作 |

---

## 使用说明

```
我需要一篇小红书文案，推广一款AI决策工具。
目标用户：30-40岁创业者
调性：专业但有温度
字数：300字以内
```
