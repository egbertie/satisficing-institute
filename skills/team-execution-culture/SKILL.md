# 🔥 团队执行文化 V1.0.0

## 🎯 功能概述

### 三条铁律
1. **时间预估必须激进** - 原计划的50% = 新截止时间
2. **相信极限潜能** - 不说"可能"，只说"一定"
3. **完成即启动** - 不停顿、不等待、永向前

### 核心价值
- 压缩时间倒逼极限输出
- 根除犹豫词汇
- 零切换成本

## 🚀 使用方法

### 任务激进评估
```
用户: 评估这个任务的激进完成时间
      任务：撰写产品白皮书
      保守预估：3天

AI: 🔥 激进时间评估
   
   原预估: 3天
   激进目标: 1.5天 (50%压缩)
   建议策略:
   ├─ 上午：大纲框架 (2小时)
   ├─ 下午：核心内容 (4小时)
   └─ 次日上午：润色完善 (2小时)
   
   💪 相信极限潜能！
```

### 执行文化检查
```
用户: 检查当前任务执行是否符合团队文化

AI: 📊 执行文化合规检查
   
   当前任务: [任务名]
   ├─ 时间预估: [激进/保守]
   ├─ 信心表达: [一定/可能]
   └─ 完成动作: [立即启动/等待指令]
   
   合规状态: ✅ 符合铁律 | ⚠️ 需改进
   改进建议: [具体建议]
```

## 📋 执行检查清单

- [ ] 时间预估是否为原计划的50%？
- [ ] 是否使用了"一定能"而非"可能"？
- [ ] 完成任务后是否立即启动下一项？
- [ ] 是否发挥了最大潜能？
