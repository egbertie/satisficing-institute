## 二、系统考虑（文化宣导→执行监控→违规纠正→复盘改进）

### 2.1 三条铁律详解

#### 铁律1: 时间预估必须激进
```
原时间 → 激进预估 → 实际执行
─────────────────────────────
4小时   →   2小时   →  1.5小时
1天     →   半天    →  4小时
1周     →   3天     →  2天
```

**执行规则**:
- 所有任务时间自动×50%
- 压缩时间倒逼极限输出
- 提前完成奖励机制

#### 铁律2: 相信极限潜能

**禁止词汇**:
- ❌ "可能" → ✅ "一定"
- ❌ "试试" → ✅ "完成"
- ❌ "应该可以" → ✅ "没问题"
- ❌ "有点难" → ✅ "可以做到"

**强制替换**:
AI输出中的所有犹豫词汇自动替换为肯定表述

#### 铁律3: 完成即启动

**零切换成本原则**:
- 任务A完成 → 0延迟 → 任务B启动
- 不停顿、不等待、永向前
- 自动补位机制配合

### 2.2 执行监控矩阵

| 监控项 | 检查频率 | 合规标准 | 违规处理 |
|--------|----------|----------|----------|
| 时间预估激进率 | 每次任务 | ≥50%压缩 | 警告+强制修正 |
| 犹豫词汇使用 | 每次输出 | 0次 | 自动替换+记录 |
| 任务切换延迟 | 实时 | 0秒 | 自动触发补位 |
| 极限挑战次数 | 每日 | ≥3次 | 鼓励+记录 |

## 四、Skill化（可执行）

### 4.1 自动执行规则

```python
def enforce_execution_culture(output, task_estimate):
    """
    团队执行文化强制执行
    """
    # 铁律1: 激进预估
    aggressive_estimate = task_estimate * 0.5
    
    # 铁律2: 消除犹豫词汇
    hesitation_words = ["可能", "试试", "应该可以", "有点难", "大概"]
    for word in hesitation_words:
        output = output.replace(word, "一定")
    
    # 铁律3: 完成即启动标记
    mark_ready_for_next_task()
    
    return output, aggressive_estimate
```
