# 客户价值系统 Scripts

## 脚本说明

### client-value-system-runner.py
客户价值系统主执行脚本。

## 功能

- evaluate
- segment
- prioritize

## 使用方法

```bash
# 执行功能
python3 client-value-system-runner.py run --feature evaluate

# 查看状态
python3 client-value-system-runner.py status

# 生成报告
python3 client-value-system-runner.py report
```
