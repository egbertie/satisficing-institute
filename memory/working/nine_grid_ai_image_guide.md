# 九宫格朋友圈图片 · AI制作指引

> **用途**: 发送给AI设计工具（如Midjourney/SD/即梦等）生成图片  
> **风格**: 克制、专业、有学术感但不失温度  
> **格式**: 9张正方形图片（1080×1080px）

---

## 一、整体视觉风格指令

### 基础风格描述（用于所有图片）

```
Minimalist design, clean layout, generous white space,
professional business style with academic elegance,
muted color palette (deep navy blue #1a365d, warm gold #d4af37, pure white #ffffff),
modern sans-serif Chinese typography (Source Han Sans or PingFang),
subtle geometric elements, no photorealistic elements,
flat design with slight depth, zen-like calm atmosphere
```

### 中文指令版本

```
极简设计风格，干净的排版，大量留白，
专业商务风格带有学术优雅感，
柔和配色（深海蓝 #1a365d、暖金色 #d4af37、纯白 #ffffff），
现代无衬线中文字体（思源黑体或苹方），
 subtle几何元素，无写实照片元素，
扁平设计带轻微立体感，禅意宁静氛围
```

## 三、技术规格

### 输出要求

| 项目 | 规格 |
|------|------|
| 尺寸 | 1080 × 1080 px（正方形） |
| 格式 | JPG 或 PNG |
| 文件大小 | 单张 < 500KB |
| 色彩模式 | RGB |
| 分辨率 | 72 dpi（屏幕显示）|

### 字体建议

| 类型 | 推荐字体 |
|------|----------|
| 标题中文 | 思源黑体 Heavy / 苹方 Semibold |
| 正文中文 | 思源黑体 Regular / 苹方 Regular |
| 英文 | Avenir / Helvetica Neue / Futura |
| 装饰/书法 | 站酷文艺体 / 方正清刻本悦宋 |
