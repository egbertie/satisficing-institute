## 三、主要观点/方法论

### 核心观点（基于Egbertie回忆整理）

#### 1. 知识≠能力
> "知道不等于做到，做到不等于做好。"

- 知识获取只是第一步
- 能力是知识的内化与应用
- 高质量输出需要持续训练

#### 2. 实践导向的训练哲学
> "训练不是教知识，而是练能力。"

- 强调身体力行，而非被动听讲
- 在实践中学习和成长
- 导师的角色是引导而非灌输

#### 3. 长期陪伴的成长支持
> "成长需要时间，导师的价值在于持续在场。"

- 能力培养是长期过程
- 持续的支持和反馈至关重要
- 建立信任关系是有效指导的基础

### 方法论框架（基于训练方式推断）

```
能力培养体系（基于陈博士方法）
├── 认知层：知识输入
│   └── 理论基础与框架建立
├── 转化层：知识→技能
│   ├── 实践任务设计
│   ├── 情境模拟训练
│   └── 即时反馈与调整
├── 固化层：技能→能力
│   ├── 重复练习与深化
│   ├── 复杂情境挑战
│   └── 习惯养成
└── 进阶层：能力→智慧
    ├── 反思与总结
    ├── 个性化优化
    └── 传授他人
```

## 四、咨询风格

### 教学特点（基于Egbertie体验推断）
| 特征 | 描述 |
|------|------|
| **实践性** | 强调"做中学"，理论结合实践 |
| **陪伴式** | 长期持续的指导，非一次性授课 |
| **个性化** | 根据学员特点调整训练方式 |
| **反馈导向** | 重视实践反馈，持续优化指导 |

### 典型指导方式
- **任务驱动**：通过具体任务推动能力发展
- **示范指导**：亲自示范标准做法
- **复盘总结**：事后复盘，提取经验教训
- **渐进放手**：从手把手指导到独立实践

### 金句风格（基于核心理念推断）
> "我能教给你的，不只是知识，更是如何获得知识的能力。"
> "最好的学习发生在实践中，最好的指导是陪伴你走过实践的过程。"
> "能力的标志不是知道多少，而是能做多好。"
