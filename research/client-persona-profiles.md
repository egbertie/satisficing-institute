## 三、替身002：张建国（连续创业者画像）

### 3.1 基础信息

```yaml
基础信息:
  姓名: 张建国
  年龄: 38
  身份: 连续创业者，现智能硬件公司CEO
  背景: 曾创办2家公司（1家被收购，1家失败）
  教育: 本科计算机，MBA辍学

创业经历:
  第一次创业: 
    时间: 2012-2016
    领域: O2O平台
    结果: 被收购
    个人获利: 800万
    合伙人问题: 股权给太多，后期失去控制权
  
  第二次创业:
    时间: 2017-2020
    领域: 共享经济
    结果: 失败
    亏损: 300万
    合伙人问题: 股权给太少，合伙人不积极，最终分道扬镳
  
  第三次创业:
    时间: 2021至今
    领域: 智能硬件
    结果: 获A轮，团队15人

当前状态:
  融资: A轮2000万，估值1亿
  团队: 15人（技术8，运营4，商务2，行政1）
  补贴: 获政府补贴300万
  痛点: 前两次合伙人都有问题，第三次不敢轻易找合伙人
```

### 3.2 决策特征

```yaml
决策特征:
  风险偏好: 中高风险偏好，但对合伙人有创伤后应激
  损失厌恶: 极高（第二次失败损失300万）
  现状偏见: 倾向于维持现状，宁可团队弱一点也不愿冒险
  核心诉求: "帮我避开前两次的坑"
  
心理画像:
  - 吃过合伙人的亏，对"股权"极其敏感
  - 第一次给太多，第二次给太少，现在不知道给多少
  - 对直觉和感知力高度认同（"吃了太多亏才懂直觉的重要"）
  - 愿意为专业服务付费（愿意花钱买安心）
  - 对合伙人选择有PTSD，需要心理安全感
```

### 3.3 典型场景

1. **场景A：投资人介绍VP**
   - 有投资人介绍了一个VP背景的人
   - 犹豫要不要给合伙人位置
   - 担心重蹈覆辙

2. **场景B：股权分配纠结**
   - 担心股权分配重蹈覆辙
   - 第一次给太多了，第二次给太少了
   - 不知道这次给多少合适

3. **场景C：扩张焦虑**
   - 想快速扩张但怕合伙人踩雷
   - 业务机会窗口有限
   - 决策时间压力

4. **场景D：感知力共鸣**
   - 对"感知力决策"有强烈共鸣
   - "我吃了太多亏才懂直觉的重要"
   - 希望有系统化的直觉训练

### 3.4 对话示例

```
张建国: "Egbertie老师，我前两次创业都倒在合伙人上，
        第一次股权给太多失去控制，第二次给太少合伙人不积极。
        现在又有投资人介绍人来，我真不知道该不该信。"

关键回应策略:
1. 共情："吃过亏的人才知道合伙人的重要性"
2. 量化："我们用22年的数据和心理学模型来系统评估"
3. 安全："有决策框架保护，不是拍脑袋"
4. 案例："和你类似情况的客户，我们这样处理的..."
```

### 3.5 转化策略

| 策略 | 说明 |
|------|------|
| 案例共鸣 | 用类似案例建立信任 |
| 风险预警 | 提前指出潜在风险点 |
| 数据支持 | 用22年数据说话 |
| 心理安全 | 强调系统化的保护机制 |

```yaml
基础信息:
  姓名: 李晓雯
  年龄: 34
  身份: 前互联网大厂产品总监，现硬科技创业

当前状态:

  补贴: 申请中
  痛点: 完全不懂制造业，急需懂行的合伙人
```

### 4.2 决策特征

```yaml
决策特征:
  风险偏好: 低风险偏好，谨慎型
  禀赋效应: 过高估计自己的互联网经验
  锚定效应: 大厂薪资成为心理锚点（"我放弃了一年150万"）
  核心诉求: "帮我找到靠谱的制造业合伙人，但不能被忽悠"
  
心理画像:
  - 互联网经验是优势也是包袱
  - 不懂制造业，担心被合伙人蒙
  - 对方法论有强烈需求（"互联网那套不管用了"）
  - 资金有限，对价格敏感
  - 渴望学习，但需要系统化指导
```

### 4.3 典型场景

1. **场景A：行业信息差焦虑**
   - 不懂制造业，担心被合伙人蒙
   - 不知道怎么验证合伙人的行业背景
   - 担心技术判断失误

2. **场景B：投资陷阱识别**
   - 有制造业老板想投资并派人来
   - 不确定是不是陷阱
   - 担心失去控制权

3. **场景C：能力迁移困惑**
   - 想学习硬科技合伙人的特征
   - 但不知从哪学
   - 互联网经验如何迁移

4. **场景D：方法论渴求**
   - 对"方法论"有强烈需求
   - "互联网那套不管用了"
   - 希望有可复制的框架

### 4.4 对话示例

```
李晓雯: "Egbertie老师，我从互联网转型做工业AI，
        完全不懂制造业，有个老板想投资还派人来当合伙人，
        我不知道这算不算陷阱，也不知道怎么判断对方靠不靠谱。"

关键回应策略:
1. 认可："跨界转型需要勇气，你已经迈出了最难的一步"
2. 框架："我们有一套工业AI合伙人评估框架"
3. 教育："制造业合伙人主要看这几个维度..."
4. 安全："用决策框架保护，避免踩坑"
```

### 4.5 转化策略

| 策略 | 说明 |
|------|------|
| 方法论输出 | 提供系统化的评估框架 |
| 信任建立 | 逐步建立专业信任 |
| 教育引导 | 补充行业认知盲区 |
| 性价比 | 强调投入产出比 |
