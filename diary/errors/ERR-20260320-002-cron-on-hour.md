## 检查脚本

```bash
#!/bin/bash
# cron-time-validator.sh
# 验证cron时间是否为错峰

if [[ $1 =~ ^0+[0-9]*\  ]]; then
    echo "🚨 错误: 时间设为整点/准点"
    echo "请修改为错峰时间（如: 07, 13, 23, 37, 47, 53）"
    exit 1
else
    echo "✅ 时间已错峰"
    exit 0
fi
```
