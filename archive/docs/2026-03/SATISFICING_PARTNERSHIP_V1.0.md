## 一、合作原则

### 1.1 为什么合作

**OPC形态的必然选择**：
- Egbertie无法独自完成所有环节
- 合作可以弥补能力短板
- 合作可以保持轻资产运营

### 1.2 合作边界

**满意解在合作中的角色**：
- **不做执行方**：不负责具体执行工作
- **做连接方**：引荐资源、协调沟通
- **做监督方**：把控质量、确保交付
- **做背书方**：为合作方提供信任背书

### 1.3 合作 vs 自建决策树

```
新需求出现
    ↓
是否属于核心能力？（只有Egbertie能做）
    ↓
   是 → 自己做
    ↓
   否 → 能否找到合适合作方？
           ↓
       否 → 不做这个需求
           ↓
       是 → 合作做
```

## 三、合作方画像总表

### 3.1 优先级排序

| 优先级 | 合作领域 | 紧急度 | 原因 |
|-------|---------|-------|------|
| P0 | 专家引荐人（黎/罗/谢） | 立即 | 获客核心依赖 |
| P0 | 技术尽调专家 | 立即 | 服务交付需要 |
| P1 | 法律风控专家 | 1个月内 | 服务交付需要 |
| P1 | 视觉设计师 | 1个月内 | 品牌物料需要 |
| P2 | 行业研究员 | 2个月内 | 提效需要 |
| P2 | 后续服务转介绍 | 3个月内 | 增值服务需要 |
| P3 | 内容分发合作 | Q3后 | 品牌建设需要 |

### 3.2 理想合作方共性

**所有合作方应具备**：
1. **专业靠谱**：在其领域有口碑
2. **价值观契合**：认同满意解的极简理念
3. **响应及时**：不拖沓，不官僚主义
4. **长期思维**：愿意建立长期关系而非一锤子买卖

**所有合作方不应有**：
1. **过度承诺**：承诺做不到的事
2. **价格虚高**：超出市场合理价格
3. **流程复杂**：需要复杂的合同/审批
4. **独家绑定**：要求独家合作

## 四、合作模式详解

### 4.1 引荐费模式

**适用场景**：专家/服务机构

| 要素 | 说明 |
|-----|------|
| **收费方式** | 按项目收取引荐费 |
| **费率** | 服务费的10-20% |
| **结算** | 项目结束后结算 |
| **优势** | 简单清晰，激励对齐 |

### 4.2 转介绍费模式

**适用场景**：后续服务转介绍

| 要素 | 说明 |
|-----|------|
| **收费方式** | 成交后收取转介绍费 |
| **费率** | 成交额的10-20% |
| **结算** | 客户付款后结算 |
| **优势** | 无风险，成交后才付费 |

### 4.3 按件付费模式

**适用场景**：外包服务（设计/研究）

| 要素 | 说明 |
|-----|------|
| **收费方式** | 按交付件数付费 |
| **定价** | 事前协商确定 |
| **结算** | 交付验收后结算 |
| **优势** | 简单直接，无长期绑定 |

### 4.4 互惠模式

**适用场景**：专家引荐人

| 要素 | 说明 |
|-----|------|
| **收费方式** | 不收现金，互惠互利 |
| **互惠内容** | 优先服务其引荐客户、案例共享、背书 |
| **结算** | 长期关系 |
| **优势** | 建立深度信任关系 |

## 五、合作管理流程

### 5.1 合作方引入流程

```
需求识别 → 渠道寻找 → 初步沟通 → 小项目测试
                                          ↓
正式合作 ← 评估通过 ← 质量评估 ← 完成测试
```

### 5.2 合作方维护

**日常维护**：
- 定期同步案例进展（月度）
- 节日问候（适度）
- 优先响应其引荐的客户

**评估机制**：
- 每个项目后评估质量
- 年度复盘合作效果
- 优胜劣汰

### 5.3 合作退出

**退出条件**：
- 连续2次质量不达标
- 价格虚高且无改进
- 价值观冲突
- 响应速度过慢

**退出方式**：
- 友好告知，不续合作
- 保持关系，不撕破脸
- 寻找替代方案

## 七、合作风险与应对

| 风险 | 应对 |
|-----|------|
| 合作方质量不稳定 | 小项目测试，优胜劣汰 |
| 合作方临时不可用 | 每个领域储备2-3位 |
| 合作方要求独家 | 明确拒绝，保持灵活性 |
| 合作费用超预算 | 事前定价，控制数量 |
| 合作方泄露客户信息 | 签署保密协议，选择可信合作方 |
