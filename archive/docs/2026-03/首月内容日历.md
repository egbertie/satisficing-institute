## 内容生产SOP

### 文章写作流程
1. **选题**（周一）: 确定本周主题
2. **大纲**（周二）: 列出文章结构
3. **初稿**（周三）: 完成主要内容
4. **优化**（周四）: 润色+配图
5. **排版**（周五）: 公众号编辑器排版
6. **发布**（设定发布时间）

### 朋友圈发布模板
**早上 (07:30-08:30)**
- 主题: 能量开启 / 金句分享
- 风格: 正能量 / 启发性

**下午 (12:00-13:00)**
- 主题: 干货分享 / 案例片段
- 风格: 专业 / 实用

**晚上 (20:00-21:00)**
- 主题: 深度思考 / 互动提问
- 风格: 亲和 / 开放

### 短视频生产流程
1. **脚本**（提前3天）
2. **录制**（手机即可，自然光）
3. **剪辑**（剪映模板，30分钟内）
4. **发布**（视频号+抖音同步）
