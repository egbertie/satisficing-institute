## 三、内容创作模板

### 3.1 公众号文章结构模板

**类型A：痛点共鸣型**
```
标题：[数字]个选错合伙人的真实故事，看完我沉默了

结构：
1. 钩子开场（30秒抓住注意力）
   - 惊人数据/反直觉观点/悬念故事

2. 痛点展开（3个层次）
   - 表层：选合伙人难在哪
   - 中层：为什么大家都选错
   - 深层：错误的代价有多大

3. 案例故事（2-3个，脱敏处理）
   - 背景 → 困境 → 决策 → 结果

4. 方法论预告（给出希望）
   - 我们的解决方案
   - 核心价值点

5. 行动号召
   - 扫码预约/领取工具/转发福利
```

**类型B：方法论干货型**
```
标题：五路决策法 | 用中国智慧解决合伙人选择难题

结构：
1. 问题引入
   - 现有方法的局限

2. 方法论介绍
   - 来源（儒家智慧+现代管理）
   - 核心框架（五路详解）

3. 实操指南
   - 步骤1/2/3
   - 工具模板

4. 案例应用
   - 真实场景演示

5. 下载资源
   - 工具/模板/检查表

6. 互动引导
   - 留言讨论/预约咨询
```

**类型C：客户见证型**
```
标题：客户说 | "如果早点遇到你们，我能省下300万"

结构：
1. 客户金句（标题放大）

2. 客户背景
   - 行业/阶段/面临的选择

3. 困境描述
   - 当时的纠结和痛苦

4. 咨询过程
   - 我们如何帮助（简化版）

5. 决策结果
   - 选择了什么/为什么

6. 后续发展
   - 现状/成果

7. 客户原话
   - 真实反馈（脱敏后）

8. 服务价值总结
```

## 六、紧急预案

### 场景1：官宣文章阅读量低于预期
**应对**：
1. 立即在5个以上微信群分享
2. 请3-5位好友转发并评论
3. 调整标题后重新发布（如有必要）
4. 增加短视频导流

### 场景2：预约咨询人数不足
**应对**：
1. 扩大私聊范围（50人→100人）
2. 提高福利价值（如额外赠送工具包）
3. 延长首发福利时间
4. 增加直播/短视频曝光

### 场景3：客户满意度低
**应对**：
1. 立即跟进，了解具体问题
2. 免费提供补充咨询
3. 全额退款（如已付费）
4. 记录问题，迭代服务流程

### 场景4：时间不够，无法完成所有任务
**应对**：
优先级排序：
1. P0：公众号注册+官宣文章+海报
2. P1：种子客户私聊（前20人）
3. P2：朋友圈内容储备
4. P3：短视频/直播准备
