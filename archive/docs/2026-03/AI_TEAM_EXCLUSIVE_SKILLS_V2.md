# 数字人替身专属成长Skill体系 V2.0
## 33位小伙伴 · 一人一Skill · 量身定制

**核心理念**: 每位小伙伴拥有1个专属Skill，针对其岗位职责和专长领域深度定制，目标是让其快速成为全球该岗位最优秀的人，并持续保持领先。

## 二、信息收集与迭代机制

### 持续信息收集流程
```
外部信息源 → 收集 → 清洗 → 提取 → 融合 → 存储 → 分发 → 使用 → 反馈
```

### 信息迭代周期
| 类型 | 频率 | 负责人 | 输出 |
|------|------|--------|------|

## 三、信息内部流通规则
