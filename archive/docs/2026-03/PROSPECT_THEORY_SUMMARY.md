## 一、核心发现

### 1.1 本质洞察

**前景理论的本质不是"人类非理性"，而是：**

> **"人类决策在不确定性下的适应性算法"**

这些"偏差"在进化史上有重要的生存价值：
- **损失厌恶**：在资源匮乏环境中，避免损失比追求收益更重要
- **现状偏见**：改变需要能量，维持现状节省认知资源
- **禀赋效应**：保护已拥有的资源提高生存概率

**但在现代商业决策中，这些适应性算法需要被识别和管理。**

### 1.2 满意解的本质

满意解的本质不是"将就"，而是：

> **"在有限理性和认知偏差双重约束下，找到最优可行解"**

**核心公式**：
```
满意解决策质量 = f(有限理性, 认知偏差管理, 信息完备度)
```

我们的工作：
1. **承认有限理性**（西蒙）- 人不可能完全理性
2. **管理认知偏差**（卡尼曼）- 识别并纠正系统性偏差
3. **提升信息质量**（满意解方法）- 结构化决策流程

### 1.3 合伙人决策中的关键发现

| 偏差类型 | 在合伙人决策中的表现 | 量化影响 |
|----------|---------------------|----------|
| **损失厌恶** | 创始人对股权稀释的恐惧 > 合伙价值创造感知 | 65%案例因此推迟/拒绝优质合伙人 |
| **禀赋效应** | 自研技术估值溢价平均35% | 导致融资谈判破裂率+40% |
| **现状偏见** | "再等等看"的平均延迟6个月 | 窗口期错失率58% |
| **锚定效应** | 首轮估值成为后续谈判心理锚点 | 后续融资期望不切实际 |
| **反射效应** | 已有收益时过度保守，面临损失时过度冒险 | 过早退出或死撑亏损项目 |
