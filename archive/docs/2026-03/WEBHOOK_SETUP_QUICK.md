# 企业微信群机器人Webhook获取步骤

## 步骤1：进入目标群聊
1. 打开企业微信APP或PC客户端
2. 进入 `liuhuabin` 群聊窗口

## 步骤2：打开群设置
1. 点击右上角「···」（三个点）按钮
2. 在弹出的菜单中选择「群机器人」

## 步骤3：添加机器人
1. 点击「添加机器人」按钮
2. 选择「新创建一个机器人」

## 步骤4：配置机器人
1. **名称**：填写「满意解小助手」（或你喜欢的名字）
2. **头像**：可上传一张图片作为头像（可选）
3. 点击「确定」

## 步骤5：获取Webhook地址
1. 创建成功后，会显示Webhook地址
2. 格式如下：
   ```
   https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
   ```
3. **复制整个地址发给我**

## 安全提示
- Webhook地址相当于机器人的"密码"
- 不要截图发给别人（里面有关键字key=后面的字符串）
- 如果怀疑泄露，可在群设置中点击机器人旁边的「...」→「重置Webhook」

## 验证步骤（可选）
获取Webhook后，我可以立即发送一条测试消息验证是否成功。
