# 关键决策陪跑 - 第3轮迭代：知识沉淀系统

## 迭代目标
将每次决策的价值最大化，从"一次性服务"升级为"持续性资产"，建立个人决策智慧库和组织决策知识库。

### 3.1 相似决策匹配算法

**匹配维度**：

| 维度 | 权重 | 匹配方法 |
|-----|------|---------|
| 决策类型 | 25% | 标签匹配 |
| 行业属性 | 20% | 行业代码匹配 |
| 决策规模 | 15% | 数量级匹配 |
| 关键挑战 | 20% | 语义相似度 |
| 时间特征 | 10% | 时间窗口匹配 |
| 决策结果 | 10% | 成功/失败分类 |

**匹配算法**：
```python
def similarity_score(current_case, historical_case):
    score = 0
    
    # 决策类型匹配
    if current_case.decision_type == historical_case.decision_type:
        score += 25
    elif is_related_type(current_case.decision_type, historical_case.decision_type):
        score += 15
    
    # 行业匹配
    if current_case.industry == historical_case.industry:
        score += 20
    elif is_adjacent_industry(current_case.industry, historical_case.industry):
        score += 10
    
    # 规模匹配（数量级相近）
    scale_similarity = calculate_scale_similarity(
        current_case.amount, 
        historical_case.amount
    )
    score += 15 * scale_similarity
    
    # 关键挑战语义相似度
    challenge_similarity = semantic_similarity(
        current_case.challenges,
        historical_case.challenges
    )
    score += 20 * challenge_similarity
    
    # 时间特征
    if current_case.time_pressure == historical_case.time_pressure:
        score += 10
    
    # 结果权重（优先推荐成功案例）
    if historical_case.result == "success":
        score += 10
    elif historical_case.result == "partial":
        score += 5
    
    return score
```
