## 框架概述

本测评框架基于全球最新神经科学研究，整合生理、神经和行为多维度指标，旨在**科学、客观、可重复**地评估个体感知力水平。框架设计对标MIT Media Lab、Stanford H-STAR、Harvard Decision Science Laboratory等世界顶级实验室的研究标准。

**设计原则**：
1. **多模态整合**：生理(HRV/GSR) + 神经(EEG) + 行为(眼动/反应时)
2. **生态效度**：实验任务贴近真实决策场景
3. **文化适应性**：支持跨文化比较与本土化
4. **可扩展性**：支持不同精度等级的测评需求

### 1.2 维度定义与指标

#### 维度1：内感受力 (Interoceptive Awareness, IA)
**定义**：感知和解读身体内部信号的能力

| 子指标 | 测量方法 | 科学基础 |
|--------|----------|----------|
| **心脏感知准确度** | 心跳计数任务(HCT) | 内感受准确度的金标准(Garfinkel et al., 2015) |
| **HRV对内感受的响应** | 心脏内感受诱发电位(HEP) | Damasio躯体标记假说(2024验证) |
| **胃蠕动感知** | 水负荷感知任务 | 胃肠道内感受测量 |

**测量任务**：
```
心跳计数任务(HCT)流程：
1. 安静休息5分钟（基线）
2. 闭眼计数心跳（不触碰脉搏）
   - Trial 1: 30秒
   - Trial 2: 45秒  
   - Trial 3: 60秒
3. 实际心率通过ECG记录
4. 计算感知准确度分数：
   Accuracy = 1 - |实际心跳数 - 报告心跳数| / 实际心跳数
```

#### 维度2：直觉敏感度 (Intuitive Sensitivity, IS)
**定义**：在信息不完全或时间压力下做出快速、准确决策的能力

| 子指标 | 测量方法 | 科学基础 |
|--------|----------|----------|
| **前摄性生理反应** | 随机事件前GSR/HRV变化 | HeartMath研究所协议(McCraty et al., 2004) |
| **爱荷华赌博任务(IGT)表现** | 隐性学习速率 | Bechara等人(1994)躯体标记验证 |
| **眼动预测模式** | 决策前的注视级联 | Gaze Cascade Effect(Fiedler & Glockner, 2012) |

**前摄性直觉测量协议**：
```
时间线：
-10s  -8s   -6s   -4s   -2s    0s
  │    │     │     │     │     │
  ├────┴─────┴─────┴─────┴─────┤
  │       生理基线采集         │
  │  (HRV、GSR、瞳孔直径)      │
  │                            │
  │          刺激呈现          │
  │    (随机生成胜利/失败)     │
  │                            │
  │        参与者决策          │
  │       (选择左/右)          │
  │                            │
  │        结果反馈            │

关键分析：事件前6-7秒的生理信号区分度
```

#### 维度3：情绪调节力 (Emotion Regulation, ER)
**定义**：监测、评估和调节情绪反应以优化决策的能力

| 子指标 | 测量方法 | 科学基础 |
|--------|----------|----------|
| **杏仁核-前额叶连接** | EEG源连接/fMRI PPI | Berboth & Morawetz元分析(2021) |
| **认知重评成功率** | 情绪调节任务 + 面部肌电(EMG) | 皱眉肌(corrugator)活动作为客观指标 |
| **情绪Stroop任务** | 情绪词干扰效应 | 情绪对认知控制的影响 |

**认知重评任务流程**：
```
阶段1：基线情绪图片观看（负面）
  ↓ 记录GSR、瞳孔、EEG
阶段2：自然观看(Look条件)
  ↓ 记录主观情绪评分(1-9)
阶段3：认知重评(Decrease条件)
  ↓ 指导语："从旁观者视角看待，降低情绪反应"
阶段4：情绪增强(Increase条件)
  ↓ 指导语："代入情境，增强情绪体验"
  
成功指标：
- 主观：Decrease评分 - Look评分
- 客观：GSR变化率、皱眉肌EMG变化
- 神经：前额叶-杏仁核连接强度
```

#### 维度4：认知灵活性 (Cognitive Flexibility, CF)
**定义**：在不同思维模式、规则集或任务需求间灵活切换的能力

| 子指标 | 测量方法 | 科学基础 |
|--------|----------|----------|
| **任务切换成本** | 切换任务范式 | 转换成本测量 |
| **威斯康辛卡片分类** | WCST持续性错误 | 前额叶功能评估 |
| **EEG θ/γ比率** | 静息态EEG频谱分析 | 认知控制与直觉加工的平衡 |

#### 维度5：跨模态整合力 (Intermodal Integration, IMI)
**定义**：整合来自不同感觉通道和认知系统信息的能力

| 子指标 | 测量方法 | 科学基础 |
|--------|----------|----------|
| **多感觉整合效率** | 视听时间判断任务 | 时间绑定窗口测量 |
| **躯体-视觉整合** |  rubber hand illusion | 身体所有权感知 |
| **情绪-认知整合** | Iowa Gambling Task + 生理同步 | 躯体标记整合 |

## 二、生理指标采集规范

### 2.1 心率变异性(HRV)采集

**设备要求**：
- 推荐：Biopac MP160, Empatica E4/E5, Polar H10
- 采样率：≥250Hz（用于波形分析）
- 精度：±1ms（R波检测）

**采集协议**：
| 阶段 | 时长 | 条件 | 目的 |
|------|------|------|------|
| 基线1 | 5分钟 | 仰卧休息 | 基础自主神经张力 |
| 任务期 | 按任务 | 决策任务中 | 任务相关变化 |
| 恢复 | 3分钟 | 任务后休息 | 自主恢复能力 |

**分析参数**：
```python
# HRV特征提取
时域特征：
- RMSSD (ms): 连续RR间期差值的均方根
- pNN50 (%): 相邻RR间期差>50ms的百分比
- SDNN (ms): 所有正常RR间期的标准差

频域特征：
- LF (0.04-0.15Hz): 低频功率 (ms²)
- HF (0.15-0.4Hz): 高频功率 (ms²)
- LF/HF ratio: 交感-副交感平衡指数

非线性特征：
- SD1/SD2: Poincaré plot指数
- DFA α1/α2: 去趋势波动分析
```

### 2.2 皮肤电反应(GSR/EDA)采集

**设备要求**：
- 推荐：Biopac EDA100C, Empatica E4, MindWare EDA
- 采样率：≥100Hz
- 电极：Ag/AgCl电极，10mm直径

**电极放置**：
```
推荐位置（按优先级）：
1. 食指和中指远端指骨掌侧（最可靠）
2. 手掌小鱼际区域
3. 足底（如手部不可用）

注意事项：
- 皮肤清洁（酒精擦拭后干燥）
- 导电胶涂抹均匀
- 避免电极间距过近
```

**分析参数**：
```python
GSR特征提取：

相位性成分(SCR)：
- SCR振幅 (μS): 皮肤电导峰值变化
- SCR潜伏期 (s): 刺激到反应 onset 的时间
- SCR上升时间 (s): 达到峰值的时间
- SCR恢复时间 (s): 回到基线的时间

基线性成分(SCL)：
- 平均SCL (μS): 时段内的平均电导水平
- SCL变异性: 电导波动的标准差

事件相关分析：
- 刺激前基线(1-2s)
- 刺激后窗口(0-5s)
- 峰值检测阈值：0.05μS
```

### 2.3 眼动追踪采集

**设备要求**：
- 实验室级：Tobii Pro Spectrum (1200Hz), SR Research EyeLink 1000 (2000Hz)
- 便携级：Tobii Pro Fusion, Pupil Labs Core
- 屏幕眼动：Tobii Pro X3-120

**校准协议**：
```
标准9点校准：
• 屏幕中心
• 四角（左上、右上、左下、右下）

接受标准：
• 平均误差 < 0.5° 视角
```

**关键指标**：
| 指标 | 定义 | 认知意义 |

### 3.3 分析参数

#### 频谱功率分析
```python
频段定义：
- Delta (δ): 0.5-4 Hz  (深度睡眠、无意识加工)
- Theta (θ): 4-8 Hz   (认知控制、工作记忆、内感受)
- Alpha (α): 8-13 Hz  (抑制、放松、注意)
  - 低频α (8-10Hz): 与注意相关
  - 高频α (10-13Hz): 与语义加工相关
- Beta (β): 13-30 Hz  (警觉、主动思维、运动准备)
  - 低频β (13-20Hz)
  - 高频β (20-30Hz)
- Gamma (γ): 30-100 Hz (特征绑定、意识、快速加工)

关键比率：
- Theta/Gamma比率：认知负荷与直觉加工的平衡
- Alpha不对称性(右-左)：情绪效价(右侧α↓=负性情绪)
- Frontal Alpha Asymmetry (FAA): (log(F4α) - log(F3α))
```

#### 事件相关电位(ERP)
```
决策相关ERP成分：

P300 (300-600ms):
- 源：顶叶和颞顶联合区
- 意义：工作记忆更新、决策信心

ERN/Ne (错误相关负波, 0-100ms after error):
- 源：前扣带皮层(ACC)
- 意义：错误检测、冲突监控

Pe (错误正波, 200-400ms after error):
- 源：PFC
- 意义：错误意识、情绪反应

LRP (偏侧化准备电位):
- 意义：运动准备和反应选择

CNV (关联性负变, 准备电位):
- 意义：预期和准备
```

#### 连接性分析
```python
功能连接指标：
- 相干性(Coherence): 两通道间频谱相关
- 相位锁定值(PLV): 相位同步性
- 格兰杰因果(Granger Causality): 方向性影响

网络分析：
- 前额叶-杏仁核有效连接
- DMN与SN的网络动态
```

### 4.2 置信度校准

**测量方法**：
```
试次结构：
刺激呈现 → 决策 → 置信度评分(1-7) → 反馈

校准分析：
- 过度自信指标：主观置信度 - 实际准确率
- 校准曲线：分箱(bin)后的置信度-准确率关系
- 分辨率：高置信度试次的准确率 - 低置信度试次的准确率

元认知敏感性(meta-d')：
- 信号检测论框架下的元认知能力

```

### 4.3 漂移扩散模型(DDM)参数

**模型参数**：
```
漂移率(v): 证据积累的速率
- 高v = 高质量证据提取
- 与感知清晰度、注意力相关

边界分离(a): 决策的谨慎程度
- 高a = 更保守，更多证据才做决策
- 与速度-准确性权衡相关

非决策时间(t0): 感知和运动过程的时间
- 反映基本感知和运动速度

起始点(z): 先验偏向
- z > a/2 = 偏向"是"反应
- z < a/2 = 偏向"否"反应
```

### 5.1 内感受任务包

**心跳计数任务(HCT)**
```python
def heartbeat_counting_task():
    """
    基于Garfinkel et al. (2015)的标准协议
    """
    phases = [
        {"duration": 30, "instruction": "闭眼，默数心跳次数"},
        {"duration": 45, "instruction": "闭眼，默数心跳次数"},
        {"duration": 60, "instruction": "闭眼，默数心跳次数"}
    ]
    
    accuracy_scores = []
    for phase in phases:
        actual_beats = get_ecg_heartbeat_count(phase["duration"])
        reported_beats = get_subjective_report()
        
        accuracy = 1 - abs(actual_beats - reported_beats) / actual_beats
        accuracy_scores.append(accuracy)
    
    ia_score = mean(accuracy_scores)  # 内感受力分数
    return ia_score
```

### 5.3 情绪调节任务包

**认知重评任务**
```
刺激材料：
- 国际情绪图片系统(IAPS)负面图片
- 标准化唤醒度和效价评分

条件：
1. 观看(Look)：自然观看，不调节情绪

- EEG(前额叶-杏仁核连接)
```

### 5.4 认知灵活性任务包

**任务切换范式**
```
任务规则：
- 颜色判断：红色/蓝色 → 左手/右手
- 形状判断：圆形/方形 → 左手/右手

试次类型：
- 重复试次：与前一试次相同任务
- 切换试次：与前一试次不同任务
```

## 六、数据分析流程

### 6.1 数据预处理

**生理数据**：
```python
# HRV预处理
1. ECG信号滤波：0.05-35Hz带通
2. R波检测：Pan-Tompkins算法
3. 异常值检测与插值：RR间期异常值(>20%变化)
4. 去除非窦性心律

# GSR预处理
1. 低通滤波：5Hz（去除高频噪声）
2. 信号分解：SCL（基线）+ SCR（相位性）
3. SCR检测：0.05μS阈值，1-5秒窗口
4. 标准化：z-score或min-max

# 眼动预处理
1. 眨眼检测与插值
2. 扫视检测（速度阈值法）
3. 注视聚类（离差法或速度法）
4. 瞳孔直径滤波与基线校正
```

**EEG数据**：
```python
# 标准预处理流程
1. 滤波：
   - 高通：0.5Hz（去除漂移）
   - 低通：40-80Hz（根据分析需求）
   - 陷波：50/60Hz（去除工频干扰）

2. 重参考：
   - 离线重参考到平均乳突或全脑平均

3. 伪迹处理：
   - ICA（独立成分分析）去除眼动、肌电、心电
   - 或基于回归的眼动校正

4. 分段与基线校正：
   - 刺激前-200ms至刺激后1000ms
   - 基线：刺激前200ms

5. 伪迹剔除：
   - 振幅阈值：±100μV
   - 独立成分标记为伪迹
```

### 6.2 特征提取

```python
# 多模态特征向量
features = {
    # HRV特征
    "hrv_rmssd": calculate_rmssd(rr_intervals),
    "hrv_lf_hf_ratio": calculate_lf_hf(rr_intervals),
    "hrv_pnn50": calculate_pnn50(rr_intervals),
    
    # GSR特征
    "gsr_mean_scl": mean(skin_conductance),
    "gsr_scr_rate": scr_count / duration,
    "gsr_scr_amplitude": mean(scr_amplitudes),
    
    # 眼动特征
    "eye_fixation_count": len(fixations),
    "eye_mean_fixation_duration": mean(fixation_durations),
    "eye_pupil_peak": max(pupil_diameter),
    "eye_pupil_mean": mean(pupil_diameter),
    
    # EEG特征
    "eeg_frontal_theta": power(Fz, 4-8Hz),
    "eeg_frontal_gamma": power(Fz, 30-50Hz),
    "eeg_theta_gamma_ratio": theta_power / gamma_power,
    "eeg_faa": log(F4_alpha) - log(F3_alpha),  # 前额α不对称
    
    # 连接特征
    "connectivity_pfc_amygdala": estimate_connectivity(PFC, amygdala),
    "network_efficiency": calculate_network_efficiency(eeg_channels)
}
```
