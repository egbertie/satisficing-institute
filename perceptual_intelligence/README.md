# 感知力神经科学基础研究
## Perceptual Intelligence Neuroscience Research

---

## 项目概述

本项目旨在建立**感知力决策的科学基础**，与全球顶级实验室（MIT Media Lab、Stanford H-STAR、Harvard Decision Science Lab等）对标。基于第一性原理：

1. **直觉不是玄学，是可测量的神经活动**
2. **躯体标记假说（Damasio）需要深化和扩展**
3. **跨文化感知力存在差异，但底层机制共通**

### 🧪 3. 训练验证方案
**文件**：`03_training_validation_protocol.md`

**研究设计**：随机对照试验(RCT) + 12个月纵向追踪

```
           随机分配 (N=160)
                │
    ┌───────────┼───────────┐
    ▼           ▼           ▼
┌────────┐  ┌────────┐  ┌────────┐
│感知力  │  │标准正念│  │等待名单│
│综合训练│  │(MBSR)  │  │对照组  │
│(n=40)  │  │(n=40)  │  │(n=40)  │
└────────┘  └────────┘  └────────┘
    │           │           │
    └───────────┴───────────┘
                │
        8周干预 + 追踪
```

**训练模块**（8周，每周3次，每次45-60分钟）：

| 模块 | 内容 | 占比 | 科学依据 |
|------|------|------|----------|
| **内感受训练** | 心跳觉知冥想、身体扫描、呼吸觉察 | 30% | Garfinkel et al. (2015) |
| **直觉决策训练** | 快速决策练习、躯体信号识别、决策日志 | 25% | HeartMath, Damasio |
| **情绪调节训练** | 认知重评、情绪标签、暂停反应 | 25% | Gross, Davidson |
| **认知灵活性训练** | 六顶思考帽、规则切换、悖论冥想 | 20% | 执行功能训练 |

**评估时间点**：
- T1（基线）→ T2-4（中期检查）→ T5（后测）→ T6（3月追踪）→ T7（6月追踪）→ T8（12月追踪）

**主要结局指标**：
- 内感受准确度（HCT）
- 直觉决策表现（改良IGT）
- 情绪调节成功率

**预期效应量**：d = 0.6（中等效应），统计功效80%

## 科学贡献与创新点

### 理论创新
1. **整合框架**：首次整合躯体标记假说、双系统理论、正念神经科学
2. **跨文化视角**：引入基因-文化交互作用视角
3. **生态效度**：从实验室走向日常决策场景

### 方法创新
1. **多模态整合**：HRV+GSR+EEG+眼动的同步采集与分析协议
2. **前摄性直觉测量**：基于HeartMath协议的前瞻性直觉评估
3. **闭环训练**：神经反馈整合的感知力训练

### 应用价值
1. **个人发展**：科学化的感知力评估与提升方案
2. **组织决策**：高管决策能力培训
3. **临床转化**：情绪障碍辅助干预
