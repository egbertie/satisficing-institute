# 满意解研究所 · 学术根基
## 核心理论来源：赫伯特·西蒙（Herbert A. Simon）

### 核心著作
| 著作 | 年份 | 核心贡献 |
|------|------|----------|
| 《管理行为》 | 1947 | 有限理性、满意解、行政人概念 |
| 《组织》（与马奇合著） | 1958 | 组织协调、冲突理论 |
| 《管理决策新科学》 | 1960 | 决策四阶段论 |
| 《人类问题解决》 | 1972 | 问题解决过程、搜索策略 |
| 《有限理性模型》 | 1982-1997 | 毕生理论集大成 |

### 关键概念应用

#### 1. 有限理性（Bounded Rationality）
> "企业决策者并不拥有'完全理性'，而只有'有限理性'；他们不再追求'最优'，而只追求'满意'。"

**业务应用**：
- 客户面临信息不全、时间压力、认知局限
- 满意解不是妥协，而是对认知现实的尊重
- 缓解客户"追求完美决策"的焦虑

#### 2. 满意解原则（Satisficing）
> "设定一个可调整的渴望水平，一旦遇到超过该水平的备选方案就结束搜索。"

**业务应用**：
- 岗位成功画像中的"满意标准"设计
- 引导客户明确：及格线、良好线、优秀线
- 避免过度搜索导致的时机延误

#### 3. 决策四阶段论
1. **情报活动**：搜寻需要决策的情境 → 需求澄清访谈
2. **设计活动**：开发分析行动方案 → 候选人深度访谈
3. **抉择活动**：从备选方案中选择 → 最终决策会
4. **审查活动**：评估过去决策 → 90天陪跑

#### 4. 程序化 vs 非程序化决策
- 合伙人引入属于**非程序化决策**
- 服务价值：将非程序化决策转化为可复用的评估框架
