```
当前替代率 = 自建Skill数 / (自建Skill数 + 可替代外部Skill数)
          = 19 / (19 + 71)
          = 19 / 90
          = 21.1%

目标替代率（6个月后）= 40 / (40 + 50)
                    = 40 / 90
                    = 44.4%

理论最大替代率 = 60 / (60 + 30)
              = 60 / 90
              = 66.7%
```

- 当前替代率：**21.1%**
- 目标替代率（6个月）：**44.4%**（翻倍）
- 理论最大替代率：**66.7%**
