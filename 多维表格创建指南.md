# 飞书多维表格创建与协作指南（方案B）

**目标**：创建应用空间的多维表格，实现双向协作
**验证时间**：2026-03-09

---

## 第一步：创建多维表格（应用空间）

### 1.1 进入API调试台
- 登录：https://open.feishu.cn/app/cli_a9208a79e2b99cc4
- 找到「API调试台」

### 1.2 选择接口
- **分类**：`base`（多维表格）
- **接口**：`POST /open-apis/base/v1/apps`
- **名称**：「创建多维表格」/「Create App」

### 1.3 填写请求体
```json
{
  "name": "满意解研究所_案例库V2",
  "description": "应用空间创建的案例管理表格"
}
```

### 1.4 发送请求
- 点击「发送」
- **预期成功响应**：
```json
{
  "code": 0,
  "data": {
    "app": {
      "app_token": "xxxxxxxxxx",  // 重要！保存这个
      "name": "满意解研究所_案例库V2",
      "url": "https://feishu.cn/base/xxxxxxxx"
    }
  }
}
```

**保存**：`app_token`（后续步骤需要）

---

## 第二步：创建数据表（Table）

### 2.1 选择接口
- **接口**：`POST /open-apis/base/v1/apps/{app_token}/tables`

### 2.2 填写路径参数
- `app_token`：上一步获取的token

### 2.3 填写请求体
```json
{
  "table": {
    "name": "合伙人决策案例"
  }
}
```

### 2.4 发送请求
- **预期响应**：返回 `table_id`
- **保存**：`table_id`

---

## 第三步：添加字段（可选）

### 3.1 选择接口
- **接口**：`POST /open-apis/base/v1/apps/{app_token}/tables/{table_id}/fields`

### 3.2 示例字段
```json
{
  "field_name": "客户名称",
  "type": 1  // 1=文本
}
```

字段类型参考：
- `1` = 文本
- `2` = 数字
- `3` = 单选
- `4` = 多选
- `5` = 日期
- `18` = 人员

---

## 第四步：添加你为协作者

### 4.1 选择接口
- **分类**：`base`（多维表格）
- **接口**：`POST /open-apis/base/v1/apps/{app_token}/roles/{role_id}/members`
- 或搜索：「添加协作者」/「Create Collaborator」

### 4.2 填写参数
- `app_token`：第一步获取的
- `role_id`：`admin` 或 `editor`

### 4.3 填写请求体
```json
{
  "member_type": "openid",
  "member_id": "ou_aea2e859a6cee1f3a11e8c018eb88ec9",
  "perm": "edit"
}
```

### 4.4 发送请求
- **预期成功**：你成为表格协作者
