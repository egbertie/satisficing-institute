## 六、常见决策陷阱与对策

```
┌─────────────────────────────────────────────────────────────────────┐
│                  常见决策陷阱与对策速查表                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│ 陷阱1：分析瘫痪（Analysis Paralysis）                                │
│ ┌───────────────────────────────────────────────────────────────┐  │
│ │ 症状：过度收集信息，无法做出决定                                 │  │
│ │ 对策：                                                          │  │
│ │ • 设定"满意"标准，应用满意即止原则                             │  │
│ │ • 设定硬性截止时间                                              │  │
│ │ • 区分"需要知道"vs"想要知道"                                  │  │
│ │ • 启动右脑：直觉什么时候说过"够了"？                           │  │
│ └───────────────────────────────────────────────────────────────┘  │
│                                                                     │
│ 陷阱2：确认偏误（Confirmation Bias）                                 │
│ ┌───────────────────────────────────────────────────────────────┐  │
│ │ 症状：只寻找支持自己观点的证据                                   │  │
│ │ 对策：                                                          │  │
│ │ • 主动寻找反面证据                                              │  │
│ │ • 指定"魔鬼代言人"挑战现有观点                                 │  │
│ │ • 使用双引擎：直觉是否对某个选项特别"偏爱"？                   │  │
│ │ • 应用儒商伦理：是否有见利忘义的倾向？                          │  │
│ └───────────────────────────────────────────────────────────────┘  │
│                                                                     │
│ 陷阱3：过度自信（Overconfidence）                                    │
│ ┌───────────────────────────────────────────────────────────────┐  │
│ │ 症状：高估自己的判断准确性                                       │  │
│ │ 对策：                                                          │  │
│ │ • 进行敏感性分析，测试假设变化的影响                            │  │
│ │ • 参考类似案例的实际结果                                        │  │
│ │ • 听取外部专家意见                                              │  │
│ │ • 降低期望水平，增加安全边际                                    │  │
│ └───────────────────────────────────────────────────────────────┘  │
│                                                                     │
│ 陷阱4：群体思维（Groupthink）                                        │
│ ┌───────────────────────────────────────────────────────────────┐  │
│ │ 症状：团队追求一致，压制不同意见                                 │  │
│ │ 对策：                                                          │  │
│ │ • 鼓励异见，匿名收集观点                                        │  │
│ │ • 邀请外部视角                                                  │  │
│ │ • 领导者最后发言                                                │  │
│ │ • 应用"先义后利"：真理比和谐更重要                             │  │
│ └───────────────────────────────────────────────────────────────┘  │
│                                                                     │
│ 陷阱5：沉没成本谬误（Sunk Cost Fallacy）                             │
│ ┌───────────────────────────────────────────────────────────────┐  │
│ │ 症状：因为已经投入太多，不愿放弃失败项目                         │  │
│ │ 对策：                                                          │  │
│ │ • 聚焦未来收益，忽略沉没成本                                    │  │
│ │ • 设定清晰的止损点                                              │  │
│ │ • 引入外部评估者                                                │  │
│ │ • 儒商智慧：知止不殆，该放手时放手                              │  │
│ └───────────────────────────────────────────────────────────────┘  │
│                                                                     │
│ 陷阱6：直觉陷阱（Intuition Traps）                                   │
│ ┌───────────────────────────────────────────────────────────────┐  │
│ │ 症状：直觉基于偏见而非经验                                       │  │
│ │ 对策：                                                          │  │
│ │ • 记录直觉并追踪准确率                                          │  │
│ │ • 区分"专家直觉"vs"认知偏见"                                   │  │
│ │ • 用数据进行验证                                                │  │
│ │ • 检查直觉的情绪基础（恐惧？贪婪？）                            │  │
│ └───────────────────────────────────────────────────────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
