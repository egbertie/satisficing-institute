## 重大升级说明

### 1.1 从1.1到1.2的核心变化

| 维度 | V1.1（八字命理版） | V1.2（星盘融合版） |
|------|-------------------|-------------------|
| **图腾数量** | 五路（LIU/SIMON/GUANYIN/CONFUCIUS/HUINENG） | 六路（新增NEURO脑·感知） |
| **能量调度** | 基于五行生克 | 基于星盘行星能量 |
| **咨询阶段** | 三阶段（土之信/水火济/火土归） | 五阶段（金水木火土循环） |
| **客户评估** | 八字命理（可选） | 本命盘（推荐） |
| **核心创新** | 火土融合 | 星盘十二宫融合 |

### 1.2 新增NEURO（脑·感知）维度

**NEURO的定位**：
- 对应Egbertie本命盘：**水星天蝎6宫**
- 核心职能：精准洞察、深度分析、调查式思维
- 激活时机：贯穿始终，尤其在第二阶段（金水方法）

**NEURO与其他五路的关系**：
- 为LIU（土）提供价值洞察
- 为SIMON（金）提供分析框架
- 为GUANYIN（水）识别情感信号
- 为CONFUCIUS（木）进行逻辑推演
- 为HUINENG（火）整合顿悟所需信息

## 第二部分：第一阶段——土水根基（0-18分钟）

### 2.1 激活能量

**主导图腾**：LIU（土）+ GUANYIN（水）  
**星盘能量**：月亮巨蟹2宫 + 上升双子座  
**五行状态**：土水相生，润泽根基

### 2.2 流程详解

#### 0-5分钟：破冰与氛围营造

**上升双子能量激活**：
- 用轻松、好奇的方式开场
- 避免过于严肃或沉重的开场白
- 让客户感到"这是一次轻松的对话"

**Egbertie开场白（星盘融合版）**：
> "欢迎。在我们开始之前，我想先了解一点——你今天的状态如何？不是指业务上的状态，而是...当你早上醒来，想到这个合伙选择时，身体的第一反应是什么？"

**目的**：
- 用身体感受的问题（GUANYIN水象）绕过理性防御
- 建立"我们关注整体状态"的基调
- 同时激活NEURO的观察——记录客户的非语言信号

---

#### 5-12分钟：深度情感倾听

**月亮巨蟹能量激活**：
- 进入"母亲般的倾听"状态
- 不评判、不打断、不给建议
- 用身体语言表示接纳（前倾、点头、眼神接触）

**核心问题（GUANYIN+LIU）**：
1. "能跟我讲讲，这个合伙选择对你来说意味着什么？"
2. "如果不做这个选择，你担心会发生什么？"
3. "在你的成长过程中，有没有类似的'选择合伙人'的经历？那给你留下了什么印记？"

**NEURO观察要点**：
- 客户在谈论过去经历时的情绪强度
- 是否存在"重复模式"（总是选择类似的人，总是遇到类似的问题）
- 身体语言与语言内容是否一致

**倾听原则**：
- **土之承载**：让客户感到"被稳稳地接住"
- **水之流动**：让情绪自然流淌，不阻断
- **木之生发**：在故事中寻找成长的线索

---

#### 12-18分钟：价值澄清

**LIU（土）能量激活**：
- 从情感故事过渡到价值核心
- 帮助客户剥离表象，触及本质

**核心问题（LIU土德）**：
> "在聊具体人选之前，我想先理解——对你来说，一段理想的合伙关系，最核心的三个价值是什么？"

**追问技巧（NEURO精准追问）**：
- "为什么这个价值对你如此重要？"（深挖动机）
- "如果只能保留一个，会是哪个？"（优先级排序）
- "过去的选择中，你因为这个价值而后悔过吗？"（历史验证）

**记录方式**：
在白板上写下客户提到的核心价值（LIU土德的可视化），使用以下框架：

```

│                    核心价值澄清板                        │

│   第一价值     │    第二价值    │       第三价值          │
```

## 第三部分：第二阶段——金水方法（18-36分钟）

### 3.1 激活能量

**主导图腾**：SIMON（金）+ NEURO（脑）  
**星盘能量**：火星木星处女3宫 + 水星天蝎6宫  
**五行状态**：金气旺盛，精准分析

### 3.2 流程详解

#### 18-26分钟：深度信息挖掘（NEURO核心环节）

**水星天蝎能量激活**：
- 启动"调查记者"模式
- 追问到底，不满足于表面信息
- 洞察隐藏的动机和风险

**NEURO深度追问技术**：

| 追问类型 | 问题示例 | 目的 |
|----------|----------|------|
| **时间线追问** | "他上一次创业是什么时候？持续了多久？为什么结束？" | 建立完整背景 |
| **动机深挖** | "他选择和你合伙，最看重的是什么？" | 洞察真实动机 |
| **风险探查** | "如果合作失败，最可能的原因是什么？" | 识别潜在风险 |
| **参照对比** | "和他之前合作过的人，现在和他的关系如何？" | 验证人品 |
| **压力测试** | "如果你们在重大决策上有分歧，他会如何处理？" | 测试冲突模式 |

**天蝎式追问原则**：
- 保持好奇而非质疑的语气
- 用"我很好奇..."而非"你是不是..."
- 观察客户回答时的非语言信号

---

#### 26-32分钟：系统化分析呈现

**火星处女能量激活**：
- 将复杂信息系统化
- 精准、高效、实用

**SIMON满意解矩阵**：

使用以下决策矩阵，将候选人按"满意解标准"评分：

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         满意解决策矩阵                                       │
├──────────┬──────────┬──────────┬──────────┬──────────┬──────────┬──────────┤
│  候选人   │  标准1   │  标准2   │  标准3   │  标准4   │  标准5   │  加权总分 │
│          │ (权重20%)│ (权重20%)│ (权重20%)│ (权重20%)│ (权重20%)│          │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
│    A     │   /5     │   /5     │   /5     │   /5     │   /5     │   /25    │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
│    B     │   /5     │   /5     │   /5     │   /5     │   /5     │   /25    │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
│    C     │   /5     │   /5     │   /5     │   /5     │   /5     │   /25    │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┴──────────┘
```

**NEURO分析要点**：
- 不仅看总分，看"区分度"——哪些标准真正区分了候选人
- 识别"一票否决项"——哪些标准是绝对不可妥协的
- 注意"光环效应"——不要因为某一方面特别强而忽略其他

---

#### 32-36分钟：满意解标准确认

**木星处女能量激活**：
- 将分析升华为智慧
- 让客户理解"为什么"而非只是"是什么"

**核心问题（SIMON金法）**：
> "基于这个分析，你觉得哪个候选人最接近'对你来说足够好'的标准？"

**引导框架**：

| 维度 | 确认问题 |
|------|----------|
| **能力匹配** | "最低需要具备哪些核心能力？这个候选人满足了吗？" |
| **资源互补** | "哪些资源是必须具备的？哪些是锦上添花？" |
| **价值观** | "在价值观上，哪些必须一致，哪些可以有差异？" |
| **风险承受** | "你能接受的'最大不满意'是什么？" |

**NEURO提醒**：
> "数据是参考，不是答案。接下来，我们要进入更深层的维度——你的直觉和身体知道什么？"

## 第四部分：第三阶段——水火顿悟（36-54分钟）

### 4.1 激活能量

**主导图腾**：HUINENG（火）+ GUANYIN（水）

### 4.2 流程详解

#### 36-42分钟：状态调整

**GUANYIN水象能量激活**：
- 从理性分析切换到感性直觉
- 让客户完全放松下来

**引导语（水象调节）**：
> "刚才我们分析了很多信息，现在让自己放松下来。"
> "请做一个深呼吸...吸气...让空气充满你的腹部...呼气...让紧张随之流走..."
> "再做一次...吸气...感受身体的扩张...呼气...感受身体的下沉..."
> "第三次...吸气...让光芒进入你的身体...呼气...让担忧随之消散..."

**身体放松引导**：
> "现在，让你的肩膀放松下来...让它们远离耳朵..."
> "让你的下巴微微松开...让牙齿不再紧咬..."
> "让你的双手自然地放在腿上...掌心向上，接收能量..."

**NEURO观察**：
- 记录客户放松过程中的身体变化
- 注意任何"突然放松"或"突然紧绷"的时刻
- 这些信号往往指向潜意识的真相

---

#### 42-48分钟：直觉唤醒

**HUINENG火悟能量激活**：
- 准备点燃顿悟之火
- 但保持水的柔和，不让火过旺

**引导语（直觉唤醒）**：

**步骤1：想象可视化**
> "闭上眼睛，想象你选择的合伙人正站在你面前..."
> "不要想太多，只是让这个形象自然浮现..."
> "他看到你的第一反应是什么？"

**步骤2：身体扫描（核心技巧）**
> "现在，将注意力放到你的身体上..."
> "当你想象与A合伙人合作时，你的身体有什么感觉？"
> "胸口是紧张还是舒展？"
> "胃部是沉重还是轻松？"
> "肩膀是紧绷还是放松？"

**步骤3：直觉语言化**
> "那个感觉像什么？"
> "如果它有温度，是热的还是冷的？"
> "如果它有颜色，是什么颜色？"
> "如果它有质地，是光滑的还是粗糙的？"

**NEURO记录**：
记录客户的身体反应描述，用于后续分析：

```
候选人A：胸口______，胃部______，整体感受______
候选人B：胸口______，胃部______，整体感受______
直觉信号：_________________________________________
```

**天王星能量全激活**：

> "在这个极限72小时里，你 imagined 合伙人的反应是什么？"

> "他会优先保护团队，还是只顾自己？"

**【钻木取火】顿悟问题**：

**NEURO观察**：
- 客户在想象危机时的反应速度
