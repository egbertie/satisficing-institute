# 案例：技术合伙人中途退出

## 基本信息
| 项目 | 内容 |
|------|------|
| **案例ID** | CASE-003 |
| **行业** | AI芯片 |
| **团队阶段** | Pre-A轮 |
| **案例来源** | 行业观察 |

## 背景
两位创始人：
- A（CEO，商业背景，持股60%）
- B（CTO，技术背景，持股40%）

产品进入流片阶段，B突然提出退出，要求公司回购股权或带走技术另起炉灶。

## 问题
1. **技术依赖过高**: 核心技术都在B脑中，未文档化
2. **股权回购争议**: B要求按公司估值回购，A认为应按净资产
3. **竞业条款模糊**: 合伙协议中竞业条款表述不清

## 决策失误点
1. **技术资产化不足**: 未要求B将核心技术文档化
2. **退出机制缺失**: 未约定合伙人退出的处理方式
3. **竞业条款不严**: 未明确技术合伙人退出后的竞业限制

## 后果
- 产品流片延期9个月
- 为回购股权支付高昂对价
- B带走部分技术，创办竞品公司
- Pre-A轮融资失败

## 满意解分析
**LIU（价值纯度）**: 核心技术未转化为公司资产
**SIMON（理性框架）**: 退出机制设计缺失
**CONFUCIUS（合伙伦理）**: 只谈合作，未谈分手

## 教训
1. 核心技术必须文档化、资产化
2. 合伙协议必须包含退出条款
3. 竞业限制必须具体可执行
