# 案例：联合创始人价值观冲突

## 基本信息
| 项目 | 内容 |
|------|------|
| **案例ID** | CASE-004 |
| **行业** | 生物医药 |
| **团队阶段** | A轮 |
| **案例来源** | 媒体报道 |

## 背景
三位创始人：
- A（CEO，连续创业者，追求快速上市）
- B（CSO，科学家背景，追求学术声誉）
- C（COO，产业背景，追求稳健盈利）

## 问题
1. **发展方向分歧**: A想快速推进临床，B坚持多发论文
2. **融资节奏冲突**: A希望高估值多融资，C担心稀释过多
3. **团队文化对立**: 科学家vs商人思维冲突

## 决策失误点
1. **价值观未对齐**: 合伙前未深入探讨长期目标
2. **决策机制僵化**: 重大事项需要三人一致同意
3. **角色边界模糊**: 科学家与经营者权责不清

## 后果
- 临床实验进度拖延
- 错过最佳融资窗口
- 核心研发团队分裂
- A轮估值被迫下调40%

## 满意解分析
**CONFUCIUS（合伙伦理）**: 合伙前未探讨"义"与"利"的平衡
**SIMON（理性框架）**: 决策机制无法应对价值观分歧
**HUINENG（知行合一）**: 理念不一，难以形成合力

## 教训
1. 合伙前必须进行价值观对齐
2. 决策机制要考虑僵局处理
3. 科学家与经营者角色必须明确
