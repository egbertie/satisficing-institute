# 满意解决策方法论 1.1 - 实践速查表

## 一、快速启动清单（5分钟版）

```
□ Step 1: 识别决策类型
   ├── 程序化决策 → 左脑主导(80%)
   ├── 半结构化 → 双引擎均衡(50/50)
   └── 非程序化 → 右脑主导(70%)

□ Step 2: 设定期望水平
   ├── 最低可接受标准：______
   ├── 情境调整因子：______
   └── 伦理底线：______

□ Step 3: 启动双引擎
   ├── 左脑：收集数据、量化分析
   └── 右脑：感知情境、形成直觉

□ Step 4: 交叉验证
   ├── 数据支持直觉吗？
   └── 直觉补充了什么盲点？

□ Step 5: 义利检查
   ├── 见利思义？
   ├── 先义后利？
   └── 以义制利？

□ Step 6: 选择满意解
   └── 第一个满足全部条件的方案
```

## 二、决策类型速判

| 特征 | 程序化决策 | 半结构化决策 | 非结构化决策 |
|------|-----------|-------------|-------------|
| **典型场景** | 预算审批、常规采购 | 战略调整、产品迭代 | 危机处理、创新突破 |
| **信息完备度** | 高 | 中等 | 低 |
| **时间压力** | 低 | 中等 | 高/不确定 |
| **先例可用** | 多 | 部分 | 无 |
| **推荐模式** | 左脑80%+右脑20% | 左脑50%+右脑50% | 右脑70%+左脑30% |
| **关键工具** | SOP、决策树 | 决策矩阵、情景规划 | 直觉验证、快速原型 |

## 三、双引擎协作模式选择

### 模式A：串联模式（右脑→左脑）
**适用**：创新探索、战略突破
```
直觉洞察 → 可行性验证 → 决策
```

### 模式B：并联模式（同时启动）
**适用**：重大投资、复杂战略决策
```
      ┌→ 数据分析 ─┐
情境 ─┤            ├──► 综合判断
      └→ 直觉感知 ─┘
```

### 模式C：融合模式（实时互动）
**适用**：危机处理、快速决策
```
数据 + 直觉 → 实时校正 → 涌现决策
```

## 四、期望水平设定公式

```
期望水平 = 基础阈值 × 情境系数 + 伦理约束

基础阈值：最低可接受标准（如ROI≥15%）
情境系数：市场环境调整（0.8-1.2）
伦理约束：儒商伦理底线（绝对不可突破）
```

## 五、右脑感知力训练（每日5分钟）

### 晨间：3分钟静默
1. 闭目深呼吸3次
2. 清空思维，专注呼吸
3. 感受身体状态
4. 记录第一反应/直觉

### 日间：观察练习
- 每天至少一次"深度观察"（5分钟无目的观察）
- 注意被忽略的细节
- 练习"第三层倾听"（听意图而非内容）

### 晚间：直觉日记
记录当天：
- 直觉信号：________
- 后续验证：________
- 模式总结：________

## 六、义利权衡检查清单

```
□ 义利相生检查
   └── 该方案是否创造了义利共赢的可能？

□ 先义后利检查
   └── 当义利冲突时，是否选择义优先？

□ 义以生利检查
   └── 长期利益是否建立在道义基础之上？

□ 以义制利检查
   └── 是否存在见利忘义的风险？

□ 利益相关者检查
   └── 各利益相关者是否会认同该方案的正当性？

□ 公开测试
   └── 如果决策被公开，是否能坦然面对？
```

## 七、常见问题快速诊断

### Q1：直觉和分析冲突怎么办？
**诊断流程**：
```
直觉 vs 数据
    │
    ├─ 数据可靠？→ 重新检查数据来源
    │
    ├─ 直觉有经验支撑？→ 深入挖掘直觉依据
    │
    └─ 仍冲突？→ 小步实验/延迟决策/寻求第三方
```

### Q2：信息太多无法决策？
**应对策略**：
1. 应用"满意即止"原则，停止信息收集
2. 使用80/20法则，聚焦关键20%信息
3. 设决策截止时间，强制选择

### Q3：时间压力下的快速决策？
**快速流程**：
```
30秒：直觉判断（选A/B/C？）
  ↓
60秒：快速验证（有明显陷阱吗？）
  ↓
30秒：伦理检查（符合义吗？）
  ↓
执行 → 边做边调整
```

### Q4：团队决策意见分歧？
**协调策略**：
1. 明确各自的数据依据和直觉来源
2. 检查是否存在信息差异
3. 识别分歧是事实分歧还是价值分歧
4. 事实分歧→补充信息；价值分歧→升级决策
