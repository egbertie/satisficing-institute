## 一、模块概述

### 1.1 测试目的

感知力测试模块帮助创始人：
- **了解**自己的感知力水平
- **识别**感知力优势和短板
- **获得**针对性的提升建议
- **建立**个人感知力基线数据

### 1.2 测试结构

```

│                   感知力测试模块 V1.1                        │

│  • 时长: 10-15分钟                                            │
│  • 形式: 在线问卷                                             │
│  • 产出: 直觉力/洞察力/共情力三个维度分数                     │
```

### 1.3 测试环境要求

- **安静的空间**：避免打扰，能专注自我觉察
- **舒适的状态**：不饥饿、不疲劳、情绪平稳
- **诚实的心态**：测试无好坏之分，诚实回答才能真实了解自己
- **计时器**：部分测试需要计时

## 二、感知力自评问卷（PSI-Self）

### 2.1 问卷说明

- **题目数量**: 30题（每个维度10题）
- **评分方式**: 1-10分（1=完全不符合，10=完全符合）
- **填写时间**: 10-15分钟
- **填写原则**: 根据第一反应回答，不要过度思考

### 2.2 直觉力维度 (Intuition)

**I1. 躯体信号敏感度**

| 题号 | 题目内容 | 1-10分 |
|------|----------|--------|
| I1-1 | 我能清晰感知自己心跳的变化 | ___ |
| I1-2 | 当环境有危险时，我能在意识到之前就有身体反应 | ___ |
| I1-3 | 我能区分胃部不适是因为饥饿还是紧张 | ___ |
| I1-4 | 我注意到自己在不同人面前身体紧张度不同 | ___ |
| I1-5 | 我能感知自己呼吸深浅的变化 | ___ |

**I2. 快速判断准确性**

| 题号 | 题目内容 | 1-10分 |
|------|----------|--------|
| I2-1 | 我的第一印象往往很准确 | ___ |
| I2-2 | 我经常在认识一个人5分钟内就有"喜欢"或"不喜欢"的感觉 | ___ |
| I2-3 | 回头看，我忽略第一印象的决策后来往往被证明是错的 | ___ |
| I2-4 | 朋友经常来问我"你觉得这个人怎么样" | ___ |
| I2-5 | 我信任自己对人的直觉判断 | ___ |

**I3. 危险预警能力**

| 题号 | 题目内容 | 1-10分 |
|------|----------|--------|
| I3-1 | 我经常在事情出问题前就有不祥的预感 | ___ |
| I3-2 | 我能感知到一个人是否在隐瞒什么 | ___ |
| I3-3 | 当有人在说谎时，即使我无法指出证据，也能感觉到 | ___ |
| I3-4 | 我避免过很多麻烦，因为我"觉得"某件事不对劲 | ___ |
| I3-5 | 我能感知一个商业机会的潜在风险 | ___ |

**直觉力维度计分**：
```
直觉力 = (I1-1到I3-5所有15题得分) / 15
```

---

### 2.3 洞察力维度 (Insight)

**IN1. 模式识别能力**

| 题号 | 题目内容 | 1-10分 |
|------|----------|--------|
| IN1-1 | 我能快速识别一个人的行为模式 | ___ |
| IN1-2 | 我善于从多次互动中发现规律 | ___ |
| IN1-3 | 我能预测一个人在不同情境下的反应 | ___ |
| IN1-4 | 我善于识别"这种人我以前见过" | ___ |
| IN1-5 | 我能快速理解一个组织的运作逻辑 | ___ |

**IN2. 矛盾发现能力**

| 题号 | 题目内容 | 1-10分 |
|------|----------|--------|
| IN2-1 | 我善于发现人们言行之间的不一致 | ___ |
| IN2-2 | 我能注意到一个人今天和昨天的表述有矛盾 | ___ |
| IN2-3 | 我能看穿一个人的"表演" | ___ |
| IN2-4 | 我注意到人们经常在无意中暴露真实想法 | ___ |
| IN2-5 | 我能发现PPT/计划书中隐藏的逻辑漏洞 | ___ |

**洞察力维度计分**：
```
洞察力 = (IN1-1到IN2-5所有10题得分) / 10
```

---

### 2.4 共情力维度 (Empathy)

**E1. 情绪识别能力**

| 题号 | 题目内容 | 1-10分 |
|------|----------|--------|
| E1-1 | 我能准确感知他人的情绪状态 | ___ |
| E1-2 | 我能从语气和表情中读出对方的真实感受 | ___ |
| E1-3 | 我能感知一个人是否在假装开心/平静 | ___ |
| E1-4 | 我能理解一个人为什么会产生某种情绪 | ___ |
| E1-5 | 我注意到人们的情绪会影响我的情绪 | ___ |

**E2. 动机理解能力**

| 题号 | 题目内容 | 1-10分 |
|------|----------|--------|
| E2-1 | 我理解人们行为背后的真实动机 | ___ |
| E2-2 | 我能感知一个人说"没关系"时是否真的没关系 | ___ |
| E2-3 | 我能理解不同文化背景的人的行为逻辑 | ___ |
| E2-4 | 我善于站在对方角度思考问题 | ___ |
| E2-5 | 人们觉得我能理解他们 | ___ |

**共情力维度计分**：
```
共情力 = (E1-1到E2-5所有10题得分) / 10
```

---

### 2.5 PSI综合评分

**计算公式**：
```
PSI = 直觉力 × 0.35 + 洞察力 × 0.35 + 共情力 × 0.30
```

**PSI评分等级**：

| PSI分数 | 等级 | 解读 |
|---------|------|------|
| 9.0-10 | 卓越 | 感知力极其敏锐，是非常可靠的决策者 |
| 8.0-8.9 | 优秀 | 感知力很强，可以信任直觉，辅助以逻辑验证 |
| 7.0-7.9 | 良好 | 感知力较好，直觉+验证的混合决策模式 |
| 5.0-6.9 | 一般 | 感知力中等，需要刻意训练，重大决策以逻辑为主 |
| 3.0-4.9 | 较弱 | 感知力较弱，容易误判，建议依赖结构化评估工具 |
| 1.0-2.9 | 薄弱 | 感知力薄弱，必须依赖外部意见和数据分析 |

## 三、情境模拟测试

### 3.1 测试说明

- **形式**: 阅读情境描述 + 回答问题
- **时长**: 10分钟
- **评分**: 每题0-10分，总分50分
- **目的**: 测试模式识别、矛盾发现和本质洞察能力

### 3.2 情境题

**情境1：合伙人面试**

```
情境描述：
你正在面试一位潜在CTO候选人李明。面试进行了45分钟，整体非常顺利。

李明表现：
• 技术背景优秀，曾在知名科技公司担任架构师
• 对你们的产品方向表现出极大热情
• 表示愿意降薪加入，只要股权比例合适
• 多次提到"我一直想加入早期团队做一番事业"
• 当被问及什么时候可以全职加入时，他说："我这边交接需要1-2个月，
  但我可以立即开始兼职投入，保证每周至少20小时"

然而，你注意到：
• 每次提到"全职"时，他会下意识地看一眼手表
• 谈到股权比例时，他详细询问了如果公司被收购的股权处理方式
• 他提到"1-2个月交接"时语调略微升高

问题1：你发现了什么潜在矛盾或风险信号？（列出3点，每点3分）

1. _____________________________________________
2. _____________________________________________
3. _____________________________________________

问题2：基于这些信号，你会采取什么行动？（4分）

_____________________________________________
```

**参考答案与评分**：

| 要点 | 分值 | 参考答案 |
|------|------|----------|
| 1-1 对"全职"话题的身体语言回避 | 3分 | 看手表=焦虑/不确定的信号 |
| 1-2 对退出机制的过度关注 | 3分 | 频繁询问收购时的股权=对长期承诺的保留 |
| 1-3 语调变化显示对时间线的不确定 | 3分 | "1-2个月"的模糊性可能是拖延策略 |
| 1-4 兼职承诺vs全职期望的矛盾 | 3分 | 20小时/周的兼职与创业全职要求存在落差 |
| 行动建议的合理性 | 4分 | 建议设置明确的里程碑测试；要求更具体的入职时间表；观察后续行动一致性 |

**情境2：股东会议**

```
情境描述：
你的合伙人张华提议引入一位新的战略投资人。在股东会议上，他详细介绍了这位投资人。

张华的介绍：

其他股东的反应：
• 两位股东表示"听起来不错，可以推进"
• 一位股东问："他的退出预期是什么？"
• 张华回答："他说看好长期，至少5-8年"

然而，你注意到张华介绍时：
• 语速比平时快约20%
• 避免与那位提问股东的眼神接触
• 当被问及尽调时间时，说"越快越好，他还有其他项目在谈"

问题1：你发现了什么需要警惕的信号？（列出3点，每点3分）

1. _____________________________________________
2. _____________________________________________
3. _____________________________________________

问题2：你会如何验证这些疑虑？（4分）

_____________________________________________
```

**参考答案与评分**：

| 要点 | 分值 | 参考答案 |
|------|------|----------|
| 2-1 语速加快+避免眼神接触 | 3分 | 可能是紧张或隐瞒某些信息的信号 |
| 2-2 "越快越好"的压力策略 | 3分 | 人为制造紧迫感，减少尽调时间 |
| 2-3 过度美好的承诺组合 | 3分 | 高估值+不干预+资源对接的"完美"组合需要验证 |
| 2-4 对退出话题的回避 | 3分 | "至少5-8年"可能不是完整的退出预期 |
| 验证方案的合理性 | 4分 | 要求进行完整尽调；联系该投资人过往被投企业；寻求独立估值意见；明确对赌条款 |

## 四、微表情识别测试

### 4.1 测试说明

- **形式**: 观察图片/视频中的面部表情，识别情绪
- **时长**: 5-8分钟
- **题目**: 10题，每题1分
- **目的**: 测试对微表情的识别能力

### 4.2 测试题目

**说明**：以下描述基于微表情特征，请判断对应的情绪状态。

**题1**：
描述：嘴角轻微上扬，但眼睛没有笑纹，眼神略微向下看。
这可能是：
- A. 真诚的开心
- B. 礼貌性假笑
- C. 尴尬
- D. 思考中

**答案**：B（礼貌性假笑的关键特征是眼轮匝肌未收缩，即眼睛不参与"笑"）

---

**题2**：
描述：眉毛内侧上扬，眼睛微微睁大，嘴角略微下垂。
这可能是：
- A. 惊讶
- B. 愤怒
- C. 悲伤/担忧
- D. 厌恶

**答案**：C（悲伤的经典表情特征）

---

**题3**：
描述：单眉上扬，嘴角一侧轻微上扬，头部微微倾斜。
这可能是：
- A. 困惑
- B. 怀疑/不信任
- C. 感兴趣
- D. 疲劳

**答案**：B（怀疑的典型表情）

---

**题4**：
描述：眉毛压低并靠拢，眼睛瞪大，嘴唇紧闭或微微张开。
这可能是：
- A. 恐惧
- B. 愤怒
- C. 惊讶
- D. 集中注意力

**答案**：B（愤怒的表情特征）

---

**题5**：
描述：上唇提升，鼻子皱起，眼睛微微眯起。
这可能是：
- A. 轻蔑
- B. 厌恶
- C. 愤怒
- D. 困惑

**答案**：B（厌恶的经典表情）

---

**题6-10**：
（实际测试中可加入真实微表情图片或视频片段，此处提供描述版）

| 题号 | 描述 | 答案 |
|------|------|------|
| 6 | 嘴巴张开呈"O"型，眉毛上扬，额头出现横纹 | 惊讶 |
| 7 | 单侧嘴角上扬，眼神略带俯视感 | 轻蔑 |
| 8 | 眉毛上扬并靠拢，嘴巴横向拉开 | 恐惧 |
| 9 | 眼睛看向一侧，嘴角轻微上扬但快速收回 | 隐瞒/掩饰 |
| 10 | 面部肌肉整体放松，但手指轻微敲击桌面 | 不耐烦（身体信号） |

**微表情测试得分**：
```
微表情得分 = 答对题数 / 10 × 10
```

## 五、躯体信号觉察测试

### 5.1 测试说明

- **形式**: 引导式身体扫描 + 情境反应记录
- **时长**: 5-10分钟
- **评分**: 根据觉察的细致程度评分
- **目的**: 测试对自己身体信号的觉察敏感度

### 5.2 测试流程

**第一部分：基线身体扫描（3分钟）**

请找一个舒适的姿势坐下，闭上眼睛，按照引导进行身体扫描：

```
1. 将注意力放在头顶...感受头皮的状态...
2. 移动到额头...觉察是否有紧绷...
3. 到眼睛...眼周肌肉的状态...
4. 到下巴...牙齿是否咬紧...
5. 到脖子...是否有僵硬感...
6. 到肩膀...是下沉还是耸起...
7. 到胸部...呼吸的深度...
8. 到胃部...是否有紧绷或不适...
9. 到双手...温度、湿度...
10. 到双腿...是放松还是紧绷...
```

**第二部分：情境反应测试（5分钟）**

请阅读以下情境，并记录你的身体反应：

**情境A**：
"想象你正在和一个潜在合伙人吃饭，对方说：'其实我已经在和另一家公司谈合作了，
但我更看好你们的方向。如果你们能在一周内给我答复，我就推掉那边。'"

**记录你的身体反应**（选择所有符合的）：
- [ ] 心跳加速
- [ ] 胃部紧绷
- [ ] 肩膀耸起
- [ ] 呼吸变浅
- [ ] 手心出汗
- [ ] 身体前倾
- [ ] 没有特别感觉
- [ ] 其他：__________

**你对这个信号的解释**：__________

---

**情境B**：
"想象你走进会议室，看到你的合伙人和另一位投资人单独谈话，看到你进来时，
他们迅速结束了话题。"

**记录你的身体反应**（选择所有符合的）：
- [ ] 心跳加速
- [ ] 胃部下沉感
- [ ] 喉咙发紧
- [ ] 肌肉紧张
- [ ] 想要离开的冲动
- [ ] 好奇感
- [ ] 没有特别感觉
- [ ] 其他：__________

**你对这个信号的解释**：__________

---

**情境C**：
"想象你和一位潜在合伙人第一次正式见面，交流了30分钟后，你感到身体放松，
嘴角不自觉上扬，有一种'终于找到对的人了'的感觉。"

### 6.3 PSI等级详细说明

**等级1：卓越感知者（PSI 9.0-10）**

特征：
- 躯体信号极其敏锐，几乎能立即感知环境中的微妙变化
- 直觉准确率极高，很少被表象欺骗
- 能穿透复杂情境，直接看到本质

应用建议：
- 可以高度信任直觉，作为决策的主要依据
- 在合伙人选择中，"感觉不对"就是足够强的否决理由
- 帮助他人提升感知力

注意事项：
- 避免过度自信，重大决策仍需逻辑验证
- 注意感知疲劳，定期休息

---

**等级2：优秀感知者（PSI 8.0-8.9）**

特征：
- 感知力强，能捕捉大部分重要信号
- 直觉较可靠，偶尔需要验证
- 能快速识别模式和矛盾

应用建议：
- 采用"直觉+验证"的决策模式
- 信任直觉，但保留理性检查的空间
- 在关键决策中，将感知作为重要权重

---

**等级3：良好感知者（PSI 7.0-7.9）**

特征：
- 感知力较好，能识别明显的信号
- 在熟悉领域直觉较准
- 需要刻意练习才能提升到更高水平

应用建议：
- 使用结构化工具（如V1.1工具包）辅助决策
- 记录直觉并追踪准确率，建立个人数据库
- 在高压情境下，降低直觉权重

---

**等级4：一般感知者（PSI 5.0-6.9）**

特征：
- 感知力中等，容易忽略 subtle 信号
- 直觉有时准确，有时误导
- 需要大量训练才能提升

应用建议：
- 主要依赖逻辑分析和结构化评估
- 感知作为辅助参考，不单独作为决策依据
- 必须进行完整的V1.1评估流程
- 寻求高感知力人士的第二意见

---

**等级5：较弱感知者（PSI 3.0-4.9）**

特征：
- 感知力较弱，容易被表象迷惑
- 经常忽略重要的危险信号
- 倾向于过度理性化

应用建议：
- 几乎完全依赖结构化评估工具
- 必须引入第三方评估（顾问、导师）
- 延长观察期，增加尽职调查时间
- 避免在压力情境下做重大决策

---

**等级6：薄弱感知者（PSI 1.0-2.9）**

特征：
- 感知力薄弱，无法信任直觉
- 经常误判人际情境
- 需要系统性训练才能改善

应用建议：
- 完全依赖逻辑和数据
- 所有重大决策必须有外部意见
- 考虑聘请专业顾问进行合伙人评估
- 从基础感知训练开始（冥想、身体扫描等）

## 七、个性化提升方案

### 7.1 基于短板的定制训练

**如果直觉力最低**：

```
Week 1-2: 基础躯体觉察
• 每日身体扫描冥想 10分钟
• 建立"身体-情绪"对应日记
• 练习在交流中随机检查身体状态

Week 3-4: 直觉验证训练
• 记录每次直觉判断
• 1周后回顾准确率
• 识别个人直觉模式

Week 5-8: 情境应用
• 在低风险情境中练习依赖直觉
• 逐步提升到中等风险情境
• 建立个人"直觉词典"
```

**如果洞察力最低**：

```
Week 1-2: 观察力训练
• 每日观察1个人的行为模式
• 练习识别"不一致"信号
• 学习基本微表情知识

Week 3-4: 案例分析
• 分析5个成功/失败的合伙人案例
• 练习"穿透表象看本质"
• 建立"矛盾信号检查清单"

Week 5-8: 实战演练
• 在每次交流中刻意寻找3个洞察
• 记录并验证
• 与导师讨论洞察准确性
```

**如果共情力最低**：

```
Week 1-2: 倾听训练
• 练习3分钟不打断的倾听
• 学习反射式回应
• 记录对方的情绪词汇

Week 3-4: 情绪识别
• 每日识别并命名5种情绪
• 练习从非语言信号读情绪
• 学习情绪词汇扩展

Week 5-8: 关系实践
• 主动进行深度对话
• 练习"站在对方角度"
• 寻求情绪反馈
```
