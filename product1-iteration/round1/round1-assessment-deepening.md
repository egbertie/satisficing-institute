# 第1轮迭代：评估科学深化

## 一、迭代说明

### 1.1 改进概述

本轮迭代将"决策基因体检"从基础问卷工具升级为**科学化、标准化的心理测量产品**，核心改进包括：

| 改进维度 | 原状态 | 迭代后 | 价值提升 |
|---------|-------|-------|---------|
| 测量信度 | 未验证 | Cronbach α > 0.85 | 结果可靠，可重复验证 |
| 测量效度 | 主观判断 | 多维度效度验证 | 真正测量决策能力 |
| 数据采集 | 静态问答 | 反应时+行为数据 | 捕捉直觉vs理性决策模式 |
| 防作弊 | 无 | 陷阱题+一致性检验 | 排除社会期望偏差 |
| 参照标准 | 无 | 行业/阶段/地域常模 | 精准定位个体位置 |
| 认知风格 | 无 | MBTI+认知偏好整合 | 深度理解思维特质 |

### 1.2 为什么需要这些改进

**科学依据：**
- 心理测量学标准要求：任何用于商业决策的测评工具，信度系数应≥0.8
- 创业者决策具有高风险特征，测量误差可能导致重大损失
- 反应时是决策研究的核心变量（Kahneman双系统理论）
- 社会期望偏差在自我报告中普遍存在（效应量d=0.3-0.5）

**商业价值：**
- 科学背书提升产品可信度，支撑高客单价（¥2,980→¥4,980）
- 常模数据库构建竞争壁垒，形成数据资产
- 精准测量提高客户满意度和转介绍率

#### 2.2.2 经典决策题示例（24题）

**题目结构：情境描述 + 选项 + 反应时记录**

```
【风险倾向 - 题目1】
情境：你的创业公司已连续亏损6个月，现金流仅能支撑2个月。
此时有一个机会：投入剩余全部资金到一个新项目，成功率40%，
成功后回报5倍；失败则公司破产。

你会：
A. 立即投入全部资金（高风险偏好）
B. 投入部分资金测试（中高风险）  
C. 寻求外部融资后再决定（中低风险）
D. 放弃机会，专注于现有业务优化（低风险偏好）

[记录：选择 + 思考时间 + 鼠标移动轨迹]
```

#### 2.2.3 认知反应题（12题）

**原理：基于Cognitive Reflection Test (CRT) 改编**
- 直觉答案（错误但快速）vs 理性答案（正确但慢速）
- 反应时分布揭示决策模式

```
【认知反应 - 题目1】
球拍和球一共卖11元。球拍比球贵10元，球多少钱？

直觉答案：1元（错误）→ 反应时<3秒
理性答案：0.5元（正确）→ 反应时>5秒
```

**评分维度：**
- 正确率（认知能力）
- 平均反应时（理性/直觉倾向）
- 反应时变异系数（决策稳定性）

#### 2.2.4 陷阱题设计（8题）

**目的：检测"装好"倾向，识别不真实回答**

```
【陷阱题 - 示例1】
在做重大决策时，我从不依赖直觉，总是进行彻底的数据分析。

□ 完全同意  □ 同意  □ 中立  □ 不同意  □ 完全不同意

[解析：完全理性在现实中不存在，选"完全同意"标记为可疑]
```

```
【陷阱题 - 示例2】
我从不担心决策的后果，因为我相信一切都会向好的方向发展。

□ 完全同意  □ 同意  □ 中立  □ 不同意  □ 完全不同意

[解析：零担忧不符合心理学现实，选"完全同意"标记为可疑]
```

**陷阱题评分规则：**
- 每题设定"极端正向"选项为陷阱
- 选中陷阱选项计+1分
- 总分≥5分：社会期望偏差高，报告需标注
- 总分≥6分：建议重新测试

#### 2.2.5 一致性检验题（6题）

```
【一致性检验 - 题目对】

第3题：面对不确定性，我倾向于等待更多信息再做决定。
第28题：在信息不足的情况下，我也能快速做出决策。

[期望：两题答案应相反或至少不一致]
[如果都选"同意"或"完全同意"：答题不认真或矛盾]
```

### 2.3 反应时测量系统

#### 2.3.1 采集指标

| 指标 | 定义 | 技术实现 | 心理学意义 |
|-----|------|---------|-----------|
| 首因时间 | 看到题目到首次操作 | JavaScript事件监听 | 初始加工速度 |
| 决策时间 | 看到题目到最终提交 | 页面计时器 | 深思熟虑程度 |
| 选项悬停 | 鼠标在各选项停留时长 | 鼠标追踪 | 选项间犹豫 |
| 修改次数 | 选择后修改的次数 | 状态记录 | 决策坚定性 |

#### 2.3.2 反应时分析模型

```python
# 直觉-理性决策风格判定算法
def classify_decision_style(reaction_time, accuracy, crt_score):
    """
    基于反应时和正确率的决策风格分类
    """
    # 标准化处理
    rt_z = (reaction_time - population_mean_rt) / population_std_rt
    
    if rt_z < -0.5 and accuracy > 0.7:
        return "直觉高效型"  # 快速且准确
    elif rt_z < -0.5 and accuracy < 0.5:
        return "直觉冲动型"  # 快速但错误
    elif rt_z > 0.5 and accuracy > 0.8:
        return "理性深思熟虑型"  # 慢速且准确
    elif rt_z > 0.5 and accuracy < 0.6:
        return "理性过度分析型"  # 慢速仍错
    else:
        return "平衡型"
```

# 个体分数在常模中的位置
def calculate_percentile(score, norm_group, dimension):
    """
    计算个体在某维度上的百分位排名

    norm_data = load_norm_data(norm_group, dimension)

### 2.6 信效度验证方案

#### 2.6.1 信度验证

| 信度类型 | 方法 | 目标值 | 验证计划 |
|---------|------|-------|---------|
| 内部一致性信度 | Cronbach's α | α > 0.80 | 预测试N=200计算 |
| 重测信度 | 间隔2周再测 | r > 0.75 | 追踪样本N=50 |
| 分半信度 | 奇偶分半 | r > 0.80 | 预测试数据 |
| 评分者信度 | 专家独立评分 | ICC > 0.85 | 开放题评分 |

#### 2.6.2 效度验证

| 效度类型 | 方法 | 目标 | 验证计划 |
|---------|------|------|---------|
| 内容效度 | 专家评议 | 专家一致率>80% | 邀请5位决策心理学专家 |
| 结构效度 | 探索性+验证性因子分析 | 因子载荷>0.50 | 预测试N=500 |
| 效标效度 | 与真实决策结果相关 | r > 0.40 | 追踪6个月创业决策 |
| 区分效度 | 已知群体差异检验 | d > 0.50 | 对比成功/失败创业者 |
