## 🎯 各级别学习框架

### L1 萌新级（第1-3个月）

**学习目标：**
- 理解AI、ML、DL基本概念
- 掌握Python基础编程
- 熟悉常用工具和框架
- 完成3-5个入门级项目

**学习资源：**
- 📚 书籍：《Python机器学习基础教程》《深度学习入门》
- 🎓 课程：Coursera机器学习专项、吴恩达深度学习课程
- 📄 论文：LeNet、AlexNet经典论文（泛读）
- 🔧 工具：Python、NumPy、Pandas、Scikit-learn

**每日时间表：**
| 时间段 | 内容 | 时长 |
|--------|------|------|
| 08:00-09:00 | 晨间阅读（概念/论文） | 1h |
| 14:00-16:00 | 课程学习 | 2h |
| 19:00-21:00 | 编程实践/项目 | 2h |
| 21:00-21:30 | 日总结笔记 | 0.5h |

---

### L2 进阶级（第4-8个月）

**学习目标：**
- 深入理解主流算法原理
- 掌握深度学习框架（PyTorch/TensorFlow）
- 完成端到端项目开发
- 开始阅读顶会论文

**学习资源：**
- 📚 书籍：《深度学习》（花书）、《动手学深度学习》
- 🎓 课程：Stanford CS231n/CS224n、李宏毅机器学习
- 📄 论文：ResNet、Transformer、BERT等核心论文精读
- 🔧 工具：PyTorch、TensorFlow、Git、Linux

**每日时间表：**
| 时间段 | 内容 | 时长 |
|--------|------|------|
| 08:00-09:00 | 论文阅读+笔记 | 1h |
| 14:00-16:30 | 课程/算法深入学习 | 2.5h |
| 19:30-22:00 | 项目实践/实验 | 2.5h |
| 22:00-22:30 | 代码Review/总结 | 0.5h |

**学习目标：**
- 建立专业领域深度（CV/NLP/推荐等）
- 复现顶会SOTA模型
- 独立设计和优化算法

- 📄 论文：顶会论文（CVPR/ICML/NeurIPS/ACL）精读+复现
- 🔧 工具：完整ML工程栈、CUDA优化、模型部署

## 📚 共享资源库

### 论文推荐清单

**必读经典：**
1. AlexNet - ImageNet Classification (NIPS 2012)
2. ResNet - Deep Residual Learning (CVPR 2016)
3. Transformer - Attention Is All You Need (NIPS 2017)
4. BERT - Pre-training of Deep Bidirectional Transformers (NAACL 2019)
5. GPT系列 - Generative Pre-trained Transformer

**前沿追踪：**
- arXiv每日更新
- Papers with Code
- 各顶会官网

### 课程推荐

**入门：**
- 吴恩达机器学习/深度学习
- fast.ai Practical Deep Learning

**进阶：**
- Stanford CS231n (CV)
- Stanford CS224n (NLP)
- Berkeley CS285 (RL)

**高级：**
- MIT 6.824 (分布式系统)
- Stanford CS329S (ML系统设计)

### 书籍推荐

1. 《深度学习》(花书) - 理论基石
2. 《动手学深度学习》 - 实践入门
3. 《Python机器学习》 - 工程实践
4. 《机器学习实战》 - 项目导向
5. 《人工智能：一种现代方法》 - 全面视角

## 📈 评估体系

### 月度评估维度

| 维度 | L1 | L2 | L3 | L4 | L5 |
|------|----|----|----|----|----|
| 知识掌握 | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 项目完成 | 2个/月 | 1个/月 | 1个/月 | 持续迭代 | 战略规划 |
| 论文阅读 | 5篇/月 | 10篇/月 | 15篇/月 | 20篇/月 | 精选精读 |
| 代码产出 | 500行/周 | 800行/周 | 1000行/周 | 架构设计 | 指导为主 |
| 社区贡献 | 提问学习 | 回答问题 | PR贡献 | 技术分享 | 行业影响 |

### 晋级标准

- **L1→L2**：完成基础课程 + 3个项目 + 基础论文理解
- **L2→L3**：独立完成端到端项目 + 复现顶会模型
- **L3→L4**：专业领域深度 + 原创研究/专利
- **L4→L5**：行业影响力 + 人才培养成果
