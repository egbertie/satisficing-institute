## 一、核心理念

### 1.1 双系统决策模型

基于卡尼曼双系统理论，合伙人选择必须同时调动：

- **左脑（系统2 - 慢思考）**：逻辑分析、数据验证、风险排查
- **右脑（系统1 - 快思考）**：直觉感知、化学反应、价值观共鸣

**决策公式：**

```
合伙人匹配度 = Σ(左脑评估维度 × 权重) × 右脑直觉系数
```

### 1.2 硬件创业的特殊性

硬件创业 vs 软件创业的合伙人选择差异：

| 维度 | 硬件创业 | 软件创业 |
|------|---------|---------|
| **资金需求** | 高（设备、模具、库存） | 低（主要是人力） |
| **迭代周期** | 长（12-18个月） | 短（周级） |
| **失败成本** | 极高（库存积压） | 较低（代码可修改） |
| **核心能力** | 供应链 + 硬件工程 + 工业设计 | 产品 + 技术 + 增长 |
| **合伙人风险** | 中途退出 = 项目可能死亡 | 中途退出 = 可快速补位 |

**结论：** 硬件创业合伙人选择更需要谨慎，一旦选择错误，纠正成本极高。

## 二、左脑评估体系

### 2.1 六大评估维度

#### 维度1：能力互补性（权重25%）

评估双方能力矩阵的重叠与互补：

```
能力矩阵表（示例）：

                技术    产品    市场    运营    融资    供应链
合伙人A          ●●●     ●●      ●       ●       ●       ○
合伙人B          ○       ●       ●●●     ●●      ●●      ●●●
────────────────────────────────────────────────────────────────
重叠度           低      低      低      低      中      低
互补度           高      中      高      中      中      高
```

**关键问题：**
- 双方核心能力是否有足够的互补空间？
- 是否存在双方都弱的"能力盲区"？
- 谁负责供应链？（硬件创业生死线）

#### 维度2：价值观对齐度（权重20%）

评估维度：
- 创业动机（赚钱 vs 理想 vs 证明自我）
- 风险偏好（激进 vs 保守）
- 成功定义（上市 vs 被收购 vs 小而美）
- 道德底线（是否接受灰色地带）

**评估工具：** 价值观对齐问卷（待开发）

#### 维度3：承诺度与投入度（权重20%）

硬件创业需要长期承诺：
- 全职 vs 兼职？
- 个人资金投入比例？
- 家庭支持程度？
- 是否有退路（其他offer、家庭资产）？

**红线指标：**
- ❌ 兼职创业（除非有明确过渡计划）
- ❌ 无个人资金投入
- ❌ 家庭强烈反对

#### 维度4：过往经历与信用记录（权重15%）

- 过往创业项目经历（成功/失败/退出方式）
- 前合伙人/同事评价（背调）
- 是否有"黑历史"（法律纠纷、信用问题）
- 行业资源与人脉积累

#### 维度5：决策风格兼容性（权重10%）

基于哈蒙德认知连续体理论：

```
决策风格光谱：

分析型 ◀────────────────────────▶ 直觉型
        合伙人A            合伙人B
        
理想状态：双方位于光谱不同位置但有重叠区
危险状态：两端极端且无任何重叠
```

#### 维度6：股权与利益分配共识（权重10%）

- 股权比例是否达成初步共识？
- 对 vesting（股权成熟期）的理解是否一致？
- 未来融资稀释的预期是否一致？
- 退出机制是否有基本共识？

---

### 2.2 风险信号清单（Red Flags）

出现以下信号，需要高度警惕：

| 类别 | 危险信号 | 风险等级 |
|------|---------|----------|
| **能力** | 双方都是技术背景，无商业/供应链能力 | 🔴 极高 |
| **价值观** | 对"成功"的定义完全不同 | 🔴 极高 |
| **承诺** | 一方想兼职、另一方全职 | 🔴 极高 |
| **信用** | 有前合伙人强烈负面评价 | 🟡 高 |
| **决策** | 一方极度控制狂、另一方完全随意 | 🟡 高 |
| **利益** | 股权分配完全谈不拢 | 🟡 高 |
| **直觉** | 见第一面就觉得"不对劲"（见第三部分） | 🟠 中高 |

## 三、右直觉知体系

### 3.1 躯体标记的应用

基于达马西奥躯体标记假设，训练客户觉察以下身体信号：

**积极信号（绿灯）：**
- 和对方聊天时身体自然放松
- 谈到困难时依然有能量（而非焦虑）
- 能产生"一起打仗"的兴奋感
- 直觉上觉得"这个人靠谱"

**消极信号（红灯）：**
- 每次沟通后感到疲惫
- 对方面前不敢说真话/需要表演
- 想到未来一起工作感到压抑
- 直觉上觉得"哪里不对，但说不清"

### 3.2 直觉校准练习

**练习1：快速画像**
在见面后24小时内，不看笔记，凭印象写下：
- 这个人的三个关键词
- 如果一起创业，最可能爆发冲突的场景
- 三年后，你们关系的三种可能走向

**练习2：压力测试想象**
想象以下场景，记录身体反应：
- 公司账上只剩3个月钱，对方主张继续投入研发，你主张裁员求生
- 你发现对方有一个重大判断失误，可能导致公司损失
- 有巨头想收购，对方想卖你想继续

**身体反应解读：**
- 紧张但可控 → 正常
- 极度焦虑/愤怒 → 价值观或决策风格可能不兼容
- 麻木/无所谓 → 承诺度可能不足

### 3.3 直觉与数据的整合

**直觉不是用来替代理性的，而是：**
- 在数据不足时提供初步方向
- 在数据矛盾时帮助判断优先级
- 在数据完备但仍有犹豫时作为"最终决定者"

**整合公式：**

```
如果 左脑评分 ≥ 80分 且 右脑直觉 = 积极
    → 建议推进，深入尽职调查

如果 左脑评分 ≥ 80分 但 右脑直觉 = 消极
    → 暂停，找出直觉信号的来源（可能有未意识到的风险点）

如果 左脑评分 < 60分 但 右脑直觉 = 积极
    → 谨慎，可能是一方在"表演"或你过度理想化

如果 左脑评分 < 60分 且 右脑直觉 = 消极
    → 明确拒绝
```

## 四、决策流程

### 4.1 三阶段模型

```
第一阶段：初筛（1-2周）
    ↓
左脑：基础信息收集（履历、背景、资源）
右脑：首次见面直觉记录
    ↓
决策：是否进入深度评估？

第二阶段：深度评估（2-4周）
    ↓
左脑：六维度详细打分 + 风险信号排查
右脑：多次互动直觉校准 + 压力测试想象
    ↓
决策：是否进入"试合伙期"？

第三阶段：试合伙（1-3个月）
    ↓
左脑：小项目合作数据复盘
右脑：日常协作中的直觉感受
    ↓
决策：正式合伙 or 友好分手
```

### 4.2 关键决策节点

| 节点 | 决策问题 | 决策依据 | 输出物 |
|------|---------|----------|--------|
| 初筛通过 | 值得继续投入时间吗？ | 基础匹配度 + 直觉初判 | 初筛评估表 |
| 深度评估通过 | 值得试合伙吗？ | 六维度打分 + 风险清单 | 深度评估报告 |
| 试合伙通过 | 值得正式合伙吗？ | 协作数据 + 感受记录 | 最终决策备忘录 |
