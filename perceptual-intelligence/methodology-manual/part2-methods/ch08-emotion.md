### 第8章: 情绪识别与调节

#### 8.1 情绪的基础

##### 8.1.1 什么是情绪

**定义**: 情绪是对内部或外部事件的复杂反应，包括生理唤醒、主观体验和行为表达。

**情绪的组成**:
```
         ┌─────────────┐
         │  刺激事件    │
         └──────┬──────┘
                │
    ┌───────────┼───────────┐
    ▼           ▼           ▼
┌───────┐  ┌────────┐  ┌────────┐
│生理唤醒│  │主观体验│  │行为表达│
│(身体) │  │(感受)  │  │(表情动作)│
└───────┘  └────────┘  └────────┘
    │           │           │
    └───────────┴───────────┘
                │
                ▼
         ┌─────────────┐
         │    情绪      │
         └─────────────┘
```

##### 8.1.2 基本情绪 vs 复杂情绪

**基本情绪 (Ekman, 1992)**:
- 快乐 (Joy)
- 悲伤 (Sadness)
- 愤怒 (Anger)
- 恐惧 (Fear)
- 惊讶 (Surprise)
- 厌恶 (Disgust)

**复杂情绪**: 基本情绪的混合 + 认知评估
- 羞耻 (Shame)
- 嫉妒 (Envy)
- 骄傲 (Pride)
- 感激 (Gratitude)
- 希望 (Hope)

##### 8.1.3 情绪的功能

**情绪的进化功能**:
- **恐惧**: 保护，远离危险
- **愤怒**: 边界，维护权益
- **悲伤**: 连接，寻求支持
- **快乐**: 重复，强化有益行为

**在决策中的作用**:
- 提供快速评估信息
- 标记重要事项
- 影响风险偏好
- 驱动行动

#### 8.2 情绪识别

##### 8.2.1 情绪粒度 (Emotional Granularity)

**定义**: 区分和命名细微情绪差异的能力。

**高情绪粒度的人**:
- 能准确描述"我感觉不好"是悲伤、沮丧、失望还是无助
- 更好的情绪调节
- 更好的决策

**情绪词汇扩展**:
```
基础: 生气
    ↓
进阶: 愤怒、恼火、愤慨、愤懑、暴怒、愠怒、激愤
    ↓
精微: 被冒犯的、被轻视的、被背叛的、被误解的
```

```
        信任
         │
  恐惧◄──┼──►愤怒
         │
    惊讶─┼─预期
         │
  悲伤◄──┼──►快乐
         │
        厌恶
```

##### 8.2.2 身体-情绪联结识别

**练习**: 情绪身体地图

```
步骤:
1. 回忆一个强烈情绪体验
2. 觉察身体各部位的感受
3. 绘制在身体上
4. 重复不同情绪
5. 比较模式
```

**常见身体-情绪模式**:

| 情绪 | 常见身体表现 |
|-----|-------------|
| 愤怒 | 面部发热、拳头握紧、下巴紧绷 |
| 恐惧 | 心跳加速、手心出汗、胃部紧缩 |
| 悲伤 | 胸口压迫、流泪、能量低落 |
| 快乐 | 面部放松、身体轻盈、能量上升 |
| 焦虑 | 肩膀紧张、呼吸浅快、坐立不安 |
| 平静 | 呼吸深长、肌肉放松、重心稳定 |

##### 8.2.3 微表情识别

**微表情**: 短暂 (1/25秒) 的面部表情，泄露真实情绪。

**练习方法**:
1. 观看微表情训练视频 (如Micro Expressions Training Tool)
2. 暂停视频，猜测情绪
3. 对比正确答案
4. 重复练习

**在决策中的应用**:
- 谈判中读取对方真实态度
- 团队决策中感知未说出的顾虑
- 识别自己的隐藏情绪

#### 8.3 情绪调节

##### 8.3.1 情绪调节策略分类

**Gross的过程模型**:

```
情境选择 → 情境修正 → 注意分配 → 认知改变 → 反应调节
(选场景)    (改场景)    (转注意)    (重评估)    (抑反应)
   │           │           │           │           │
   └───────────┴───────────┴───────────┴───────────┘
                    情绪反应
```

**策略效果比较**:

| 策略 | 效果 | 能量消耗 | 长期使用 |
|-----|------|---------|---------|
| 情境选择 | 最佳 | 低 | 推荐 |
| 认知重评 | 很好 | 中 | 推荐 |
| 注意转移 | 好 | 低 | 可以 |
| 表达抑制 | 差 | 高 | 不推荐 |

##### 8.3.2 认知重评技术

**原理**: 改变对事件的解读，改变情绪反应。

**步骤**:
```
1. 识别自动思维
   "这个决策会失败"

2. 质疑自动思维
   "证据是什么？"
   "还有其他可能吗？"
   "最坏的情况是什么？"

3. 生成替代思维
   "这是一个挑战，我有能力应对"

4. 观察情绪变化
```

**重评框架**:

| 原始解读 | 替代解读 |
|---------|---------|
| "这是威胁" | "这是挑战" |
| "我必须成功" | "我尽力就好" |
| "这是坏事" | "这是学习机会" |
| "我不够好" | "我正在成长" |

##### 8.3.3 身体调节技术

**原理**: 通过改变身体状态调节情绪。

**技术清单**:

1. **呼吸调节**
   ```
   焦虑/愤怒时: 延长呼气 (吸4-呼8)
   低落/疲劳时: 延长吸气 (吸8-呼4)
   ```

2. **姿势调整**
   ```
   力量姿势: 扩张姿态2分钟，提升自信
   开放姿势: 打开肩膀，减少防御
   ```

3. **温度调节**
   ```
   冷水洗脸: 激活潜水反射，平复情绪
   热敷: 放松肌肉，缓解紧张
   ```

4. **运动释放**
   ```
   快速运动: 释放紧张能量
   缓慢运动: 激活副交感神经
   ```

#### 8.4 情绪在决策中的作用

##### 8.4.1 情绪作为信息

**躯体标记假说 (Damasio)**:
- 情绪是身体状态的信号
- 这些信号帮助快速决策
- 岛叶受损者无法做出有利决策

**应用**:
```
情绪出现 → 询问:"这个情绪在告诉我什么？"
              ↓
         分析情绪信息
              ↓
         纳入决策考量
```

##### 8.4.2 区分有用情绪与干扰情绪

| 有用情绪 | 干扰情绪 |
|---------|---------|
| 基于当前情境 | 基于过去创伤 |
| 提供相关信息 | 过度放大 |
| 随情境变化 | 持续存在 |
| 可以命名和表达 | 混乱难以表达 |

**处理干扰情绪**:
1. 识别:"这不是关于当下"
2. 暂停决策
3. 进行情绪调节
4. 必要时寻求专业支持

##### 8.4.3 情绪调节与决策质量

**研究证据**:
- 情绪稳定时决策更理性
- 适度情绪激活有助于创造性
- 过度情绪激活损害判断

**决策前情绪检查**:
```
1. 我现在的情绪强度 (1-10)?
2. 这个情绪与决策相关吗？
3. 如果强度>7，是否应该推迟决策？
4. 需要先做情绪调节吗？
```
