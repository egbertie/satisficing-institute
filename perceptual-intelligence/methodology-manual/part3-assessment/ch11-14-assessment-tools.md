## 第三部分: 测评工具篇

### 第11章: 感知力综合评估体系

#### 11.1 评估体系设计原则

##### 11.1.1 评估目标

1. **诊断**: 了解当前感知力水平
2. **发展**: 识别需要提升的领域
3. **认证**: 评估是否达到认证标准
4. **研究**: 收集数据，优化方法论

##### 11.1.2 评估设计原则

**科学级测评**:
- 标准化
- 信效度验证
- 可重复
- 客观指标为主

**日常版测评**:
- 简便易行
- 自我评估
- 高频使用
- 主观体验为主

#### 11.2 感知力维度模型

##### 11.2.1 四维模型

```
            ┌─────────────┐
            │  身体觉察力  │
            │  (Body)     │
            └──────┬──────┘
                   │
    ┌──────────────┼──────────────┐
    │              │              │
    ▼              │              ▼
┌─────────┐        │        ┌─────────┐
│ 情绪识别力│◄───────┴───────►│ 直觉敏锐力│
│(Emotion)│                     │(Intuition)│
└─────────┘                     └─────────┘
    │                              │
    └──────────────┬───────────────┘
                   │
            ┌──────┴──────┐
            │  整合决策力  │
            │(Integration)│
            └─────────────┘
```

##### 11.2.2 各维度定义

**身体觉察力 (B)**:
- 内感受敏感度
- 身体信号识别
- 身心联结

**情绪识别力 (E)**:
- 情绪粒度
- 情绪-身体联结
- 他人情绪识别

**直觉敏锐力 (I)**:
- 快速模式识别
- 真假直觉区分
- 直觉准确度

**整合决策力 (Int)**:
- 理性-直觉整合
- 决策质量
- 执行一致性

#### 11.3 感知力综合评估 (PIA)

##### 11.3.1 评估结构

**时长**: 90分钟
**形式**: 在线 + 线下结合
**总分**: 100分

**模块分配**:

| 模块 | 时长 | 分值 | 形式 |
|-----|------|-----|------|
| 理论基础 | 20分钟 | 20分 | 选择题 |
| 身体觉察 | 20分钟 | 25分 | 实践测试 |
| 情绪识别 | 15分钟 | 20分 | 视频测试 |
| 直觉测试 | 20分钟 | 20分 | 情境任务 |
| 整合应用 | 15分钟 | 15分 | 案例分析 |

##### 11.3.2 评分标准

**认证分数线**:

| 等级 | 分数 | 说明 |
|-----|------|-----|
| 初级觉察师 | 60-69分 | 具备基础感知力 |
| 中级觉察师 | 70-79分 | 感知力良好 |
| 高级觉察师 | 80-89分 | 感知力优秀 |
| 专家级 | 90-100分 | 感知力卓越 |

**维度分数线**:
- 每个维度不低于该等级下限的70%
- 避免"偏科"

### 第12章: 科学级测评工具

#### 12.1 身体觉察测评

##### 12.1.1 心跳感知测试 (Heartbeat Perception Task)

**原理**: 测试内感受敏感度

**流程**:
```
1. 安静休息5分钟
2. 闭眼，不摸脉搏
3. 计数30秒内的心跳次数
4. 对比实际心率 (心率监测器)
5. 计算准确度
```

**评分**:
```
准确度 = 1 - |报告值 - 实际值| / 实际值

≥90%: 优秀
70-89%: 良好
50-69%: 中等
<50%: 需要训练
```

##### 12.1.2 身体扫描准确度测试

**流程**:
```
1. 播放引导身体扫描音频
2. 在特定部位设置轻微触觉刺激
3. 被试报告感受到刺激的部位
4. 计算准确度
```

##### 12.1.3 身体信号解读测试

**情境模拟**:
```
呈现决策情境 → 记录身体反应 → 解读含义 → 对比标准答案
```

#### 12.2 情绪识别测评

##### 12.2.1 微表情识别测试

**工具**: 标准化微表情视频库

**流程**:
```
1. 播放微表情视频 (1/25秒-3秒)
2. 被试选择情绪类别
3. 记录正确率
```

**情绪类别**:
- 快乐、悲伤、愤怒、恐惧、惊讶、厌恶
- 混合情绪

##### 12.2.2 情绪粒度测试

**流程**:
```
1. 呈现情绪情境
2. 被试用词汇描述感受
3. 评估词汇的精确度和丰富度
4. 对比情绪轮的标准描述
```

**评分维度**:
- 词汇数量
- 区分度
- 准确度

##### 12.2.3 情绪-身体联结测试

**流程**:
```
1. 唤起特定情绪 (回忆或视频)
2. 被试绘制身体感受图
3. 对比标准情绪身体地图
4. 评估匹配度
```

#### 12.3 直觉测评

##### 12.3.1 模式识别测试

**工具**: 抽象模式识别任务

**流程**:
```
1. 呈现复杂信息序列
2. 被试预测下一项
3. 记录准确度和反应时
```

**示例**:
- 序列预测
- 视觉模式识别
- 社交情境判断

##### 12.3.2 直觉-分析冲突测试

**流程**:
```
1. 呈现两难情境
2. 测量直觉反应时
3. 测量分析后的决策
4. 评估整合能力
```

##### 12.3.3 预测准确度追踪

**长期评估**:
```
1. 被试对未来事件做预测
2. 记录置信度
3. 追踪实际结果
4. 计算校准度
```

#### 12.4 整合决策测评

##### 12.4.1 决策情境模拟

**工具**: 沉浸式决策场景

**类型**:
- 商业决策模拟
- 人际决策模拟
- 伦理决策模拟

**评估维度**:
- 信息使用
- 直觉运用
- 风险评估
- 决策质量
- 执行承诺

##### 12.4.2 决策复盘分析

**流程**:
```
1. 回顾过去的真实决策
2. 应用PIDM框架分析
3. 评估各环节表现
4. 识别模式
```

#### 13.1 5分钟感知力自检

##### 13.1.1 快速版本

```
1. 身体扫描 (1分钟)
   - 从头到脚快速扫描
   - 标注主导感受位置

2. 情绪标注 (1分钟)
   - 当前主导情绪是什么？
   - 强度1-10?

3. 能量评估 (1分钟)
   - 当前能量水平1-10?
   - 与平时相比如何？

   - 对当前决策的第一感觉?
```

#### 13.3 月度感知力评估

##### 13.3.1 自评量表

**身体觉察力** (1-10分)
```
□ 1-3: 很少注意到身体信号
□ 4-6: 能注意到明显的身体信号
□ 7-8: 能识别细微的身体信号
□ 9-10: 对身体状态有持续觉察
```

**情绪识别力** (1-10分)
```
□ 1-3: 情绪混乱，难以命名
□ 4-6: 能识别基本情绪
□ 7-8: 能区分复杂情绪
□ 9-10: 高情绪粒度，精准命名
```

**直觉敏锐力** (1-10分)
```
□ 1-3: 很少意识到直觉
□ 4-6: 能注意到直觉信号
□ 7-8: 能区分真假直觉
□ 9-10: 直觉准确，信任并运用
```

**整合决策力** (1-10分)
```
□ 1-3: 决策时理性直觉冲突
□ 4-6: 尝试整合，但不流畅
□ 7-8: 能有效整合
□ 9-10: 整合流畅，系统3运作
```
