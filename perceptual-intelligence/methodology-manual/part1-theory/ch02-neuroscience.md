### 第2章: 神经科学基础

#### 2.1 大脑的三层结构

根据Paul MacLean的三元脑理论，人类大脑由三个进化层次组成：

```
┌─────────────────────────────────────────────┐
│              新皮质 (Neocortex)              │
│            (理性脑 / 人类脑)                  │
│  • 语言 • 逻辑 • 抽象思维 • 计划              │
│  • 厚度: 2-4mm • 占大脑体积80%               │
├─────────────────────────────────────────────┤
│            边缘系统 (Limbic System)          │
│           (情绪脑 / 哺乳动物脑)               │
│  • 情绪 • 记忆 • 动机 • 社会行为              │
│  • 核心结构: 杏仁核、海马体、下丘脑           │
├─────────────────────────────────────────────┤
│           脑干与小脑 (Brainstem/Cerebellum)   │
│            (本能脑 / 爬虫脑)                 │
│  • 生存本能 • 呼吸心跳 • 战斗-逃跑-僵住       │
│  • 自动化反应 • 身体节律                      │
└─────────────────────────────────────────────┘
```

**对PIDM的意义**:
- 理性脑太慢 (40-60 bits/秒)
- 情绪脑和本能脑才是快速处理的主力 (1100万 bits/秒)
- 高质量决策需要三层脑的协同

#### 2.2 双重神经系统详解

##### 2.2.1 认知神经系统 (CNS)

**主要结构**:
- 前额叶皮层 (PFC): 执行功能、工作记忆
- 顶叶: 空间处理
- 颞叶: 语言处理
- 枕叶: 视觉处理

**功能特点**:
- 串行处理: 一次只能专注一件事
- 高能耗: 占大脑2%体重，消耗20%能量
- 易疲劳: 葡萄糖耗尽后判断力下降

**决策应用**:
- 适合: 复杂计算、逻辑推理、规则应用
- 不适合: 快速反应、模式识别、复杂情境评估

##### 2.2.2 边缘-直觉神经系统 (LINS)

**主要结构**:
- 杏仁核: 恐惧与威胁检测
- 岛叶: 内感受 (身体状态感知)
- 前扣带回: 冲突监测与错误检测
- 基底神经节: 习惯与技能学习

**功能特点**:
- 并行处理: 同时处理海量信息
- 快速: 毫秒级响应
- 模式识别: 识别熟悉模式
- 自动化: 熟练后无需意识参与

**决策应用**:
- 适合: 快速判断、模式识别、风险评估
- 风险: 可能受偏见和情绪扭曲

#### 2.3 岛叶: 身体与意识的桥梁

##### 2.3.1 岛叶的功能

岛叶 (Insula) 是位于大脑两侧皮层层叠结构之间的一个区域，被认为是"身体-大脑整合"的核心。

**功能分区**:
- 后部岛叶: 接收原始身体信号 (心跳、内脏感觉)
- 前部岛叶: 将身体状态转化为意识体验 (感受)

**关键作用**:
1. **内感受**: 感知身体内部状态
2. **情绪觉知**: 将身体变化标记为情绪
3. **风险感知**: "直觉上觉得不对劲"
4. **时间感知**: 主观时间体验

##### 2.3.2 岛叶与决策的关系

**Damásio的躯体标记假说 (Somatic Marker Hypothesis)**:

```
决策情境 → 身体反应 → 岛叶感知 → 情绪标记 → 决策倾向
    │                                       │
    └───────────── 直觉形成 ────────────────┘
```

**经典研究**:
- 岛叶受损的赌博任务研究
- 患者能描述规则，但无法做出有利决策
- 原因: 无法通过身体信号感知"好"与"坏"

**对PIDM的意义**: 
- 岛叶是"直觉"的神经基础
- 岛叶功能可以通过训练增强
- 感知力训练 = 岛叶功能开发

#### 2.4 神经可塑性: 大脑可以被训练

##### 2.4.1 什么是神经可塑性

神经可塑性 (Neuroplasticity) 指大脑结构和功能能够随经验改变的能力。

**类型**:
1. **结构可塑性**: 神经元连接的改变
2. **功能可塑性**: 现有连接效率的改变

##### 2.4.2 经验依赖的可塑性

**Hebb法则**: "一起激发的神经元连在一起。"
```
Neurons that fire together, wire together.
```

**关键发现**:
- 伦敦出租车司机: 海马体后部的空间记忆区增大
- 音乐家: 运动皮层和听觉皮层连接增强
- 冥想者: 前额叶皮层增厚，杏仁核活动减弱

##### 2.4.3 感知力训练的可塑性证据

**研究证据**:

1. **心跳感知训练**
   - 被试通过训练提高心跳计数准确度
   - 伴随岛叶灰质密度增加
   
2. **正念冥想**
   - 8周正念训练改变大脑结构
   - 前额叶-杏仁核连接增强
   - 情绪调节能力提升

3. **专家直觉**
   - 消防员、急诊医生的神经模式
   - 特定情境下的自动化反应

**对PIDM的意义**: 
- 感知力不是天赋，是可训练的技能
- 训练需要时间 (21天形成习惯，90天巩固)
- 不同层次需要不同训练强度

#### 2.5 默认模式网络与决策

##### 2.5.1 默认模式网络 (DMN)

当人不专注于外部任务时，一组脑区会活跃起来，称为"默认模式网络"。

**核心脑区**:
- 内侧前额叶皮层 (自我相关思考)
- 后扣带回 (自传体记忆)
- 楔前叶 (自我表征)
- 角回 (心智理论)

##### 2.5.2 DMN与直觉

**研究发现**:
- 创造性顿悟前，DMN活跃度增加
- "啊哈时刻"常发生在走神时
- DMN允许远距离概念联结

**对PIDM的意义**:
- 持续专注不是最佳决策状态
- 需要"有意的走神"让潜意识工作
- P-E-R-C-E-I-V-E模型中的"Pause"阶段激活DMN

#### 2.6 压力与决策的神经机制

##### 2.6.1 应激反应的两条通路

**急性应激反应**:
```
威胁感知 → 下丘脑 → 交感神经 → 肾上腺素释放
     │
     └──► 战斗-逃跑-僵住反应 (秒级)
```

**慢性应激反应**:
```
持续压力 → HPA轴激活 → 皮质醇释放
     │
     └──► 影响海马体、前额叶功能 (分钟-小时级)
```

##### 2.6.2 压力对决策的影响

| 压力水平 | 神经影响 | 决策表现 |
|---------|---------|---------|
| 低 | 前额叶正常运作 | 理性但可能过度分析 |
| 中等 | 适度激活，警觉提高 | 最佳状态，直觉与分析平衡 |
| 高 | 前额叶功能抑制 | 冲动、短视、自动化反应 |
| 慢性 | 海马体萎缩，前额叶变薄 | 记忆力下降，判断力受损 |

**对PIDM的意义**:
- 学会识别自己的压力状态
- 在高压力下需要特别的技术调节
- 长期训练提高压力下的决策质量

#### 2.7 镜像神经元系统与社交决策

##### 2.7.1 镜像神经元的发现

1990年代，意大利研究者Rizzolatti发现：
- 猴子做动作时，特定神经元激活
- 猴子看别人做同样动作时，相同神经元也激活
- 命名为"镜像神经元"

##### 2.7.2 人类的镜像系统

人类有更复杂的镜像系统，涉及：
- 动作模仿
- 情绪共鸣 (看到他人痛苦时自己感到痛苦)
- 意图理解
- 社会学习

##### 2.7.3 镜像系统与决策

**应用**:
- **同理心决策**: 理解他人立场
- **团队决策**: 感知团队情绪氛围
- **谈判**: 读取对方真实意图

**对PIDM的意义**:
- 身体觉察包括对他人身体信号的觉察
- "神经共鸣"是合伙人匹配的科学基础
- 社交直觉可以被训练和校准
