### 第3章: 建构主义心理学

#### 3.1 什么是建构主义

**核心命题**: 知识不是被发现的，而是被建构的。

**与客观主义对比**:

| 客观主义 | 建构主义 |
|---------|---------|
| 现实独立于观察者 | 现实是主动建构的 |
| 知识是真理的反映 | 知识是可行的假设 |
| 学习是被动接收 | 学习是主动建构 |
| 目标是找到正确答案 | 目标是发展有效理解 |

**对PIDM的意义**:
- 每个人的"直觉"都是基于其独特的建构系统
- 感知力 = 对自己建构系统的觉察能力
- 可以通过觉察和重构来优化决策

#### 3.2 George Kelly的个人构念理论

##### 3.2.1 人是科学家

Kelly (1955) 在《个人构念心理学》中提出：

> "人人都是科学家，每个人都试图预测和控制周围事件。"

**基本假设**: 人通过"构念" (Constructs) 来解释世界并预测未来。

##### 3.2.2 什么是构念

**构念的定义**: 一种区分方式，一种看待事物的角度。

**例子**:
- "好-坏"
- "成功-失败"
- "安全-危险"
- "真诚-虚伪"

**构念的特性**:
1. **两极性**: 每个构念都有两极 (如好-坏)
2. **个人性**: 每个人的构念系统独特
3. **层级性**: 有些构念比其他构念更重要
4. **动态性**: 构念系统可以变化和更新

##### 3.2.3 核心构念 vs 边缘构念

```
                    ┌─────────────────┐
                    │    核心构念      │  ← 定义"我是谁"
                    │  (Superordinate) │
                    └────────┬────────┘
                             │
                    ┌────────┴────────┐
                    │    次级构念      │
                    │ (Subordinate)   │
                    └────────┬────────┘
                             │
                    ┌────────┴────────┐
                    │    边缘构念      │  ← 容易改变
                    │  (Peripheral)   │
                    └─────────────────┘
```

**核心构念的特点**:
- 与身份认同紧密相关
- 改变会引发存在焦虑
- 通常是早期形成的

**边缘构念的特点**:
- 与具体情境相关
- 相对容易改变
- 不涉及核心身份

**对PIDM的意义**:
- 重大决策往往触及核心构念
- 感知力包括觉察自己的核心构念
- 变革需要重构核心构念系统

##### 3.2.4 构念系统的维度

**构念系统可以从以下维度分析**:

1. **范围 (Range)**
   - 宽范围: 构念适用于多种情境
   - 窄范围: 构念仅适用于特定情境

2. **渗透性 (Permeability)**
   - 高渗透: 愿意纳入新元素
   - 低渗透: 固守已有分类

3. **弹性 (Flexibility)**
   - 弹性系统: 能适应新情境
   - 僵化系统: 用旧构念套用新情况

**对PIDM的意义**:
- 感知力包括觉察自己构念系统的弹性和渗透性
- 僵化构念系统是直觉偏差的重要来源
- 训练目标是提高构念系统的弹性

#### 3.3 构念系统与直觉

##### 3.3.1 直觉的构念基础

**命题**: 直觉是构念系统快速运作的产物。

**过程**:
```
新情境 ──► 快速匹配构念 ──► 预测结果 ──► 直觉反应
             │
             └──► 自动化的模式识别
```

**例子**:
- 经验丰富的医生看病人"觉得不对劲"
- 背后的构念: "正常病人 vs 异常病人"
- 快速匹配到"异常"类别，触发警觉

##### 3.3.2 偏见作为僵化的构念

**认知偏见的构念解释**:

| 偏见 | 构念解释 |
|-----|---------|
| 确认偏见 | 只使用支持已有构念的信息 |
| 锚定效应 | 过度依赖初始构念 |
| 可得性启发 | 最近激活的构念更容易被调用 |
| 刻板印象 | 过度泛化的社会构念 |

**对PIDM的意义**:
- 区分"真正的直觉"与"偏见驱动的反应"
- 真正的直觉基于丰富、弹性的构念系统
- 偏见源于僵化、狭窄的构念系统

#### 3.4 意义的建构与重建

##### 3.4.1 意义的层次

**Claire Petitmengin的意义层次模型**:

```
Level 4: 反思性意义 (Reflection on experience)
         "我觉察到我有这样的反应"

Level 3: 情绪意义 (Emotional meaning)
         "这让我感到焦虑"

Level 2: 身体意义 (Somatic meaning)
         "我胸口发紧"

Level 1: 前反思意义 (Pre-reflective meaning)
         纯粹的原始体验
```

**对PIDM的意义**:
- 感知力训练是"向下"移动的过程
- 从Level 4走向Level 1
- 越深的层次，信息越丰富，越接近直觉

##### 3.4.2 重建构念的技术

**固定角色疗法 (Fixed Role Therapy)**:

Kelly开发的技术，帮助人们尝试不同的"自我"。

**步骤**:
1. 描述想要成为的角色
2. 在特定情境中"扮演"这个角色
3. 反思体验
4. 整合新构念

**对PIDM的意义**:
- 重大决策常常需要"成为不同的人"
- 通过"试演"来测试不同的决策路径
- 身体层面的试演比纯想象更有效

#### 3.5 叙事与决策

##### 3.5.1 人是讲故事的动物

**叙事心理学**: 人通过故事来理解自己的生活。

**关键概念**:
- **叙事身份**: 我们讲述的关于自己的故事
- **情节线**: 过去-现在-未来的连贯性
- **转折点**: 改变叙事方向的关键事件

##### 3.5.2 决策作为叙事的选择

**视角**: 每个决策都在"继续旧故事"或"开启新故事"之间选择。

**例子**:
- 接受新工作 = "冒险者"叙事 vs "稳定优先"叙事
- 结束一段关系 = "自我成长"叙事 vs "承诺"叙事

**对PIDM的意义**:
- 感知力包括觉察自己正在讲述的故事
- 重大决策需要审视根本叙事
- 改变叙事 = 改变直觉基础

#### 3.6 社会建构主义

##### 3.6.1 现实的协商本质

**Gergen的社会建构主义**:
- 现实不是在个人头脑中建构的
- 现实是在社会互动中"协商"的
- 语言不是反映现实，而是创造现实

##### 3.6.2 决策的社会维度

**命题**: 任何决策都嵌入在特定的社会关系和文化话语中。

**应用**:
- 合伙人决策: 不仅是两个人的事，是两个世界的碰撞
- 组织决策: 权力关系、政治博弈的影响
- 跨文化决策: 不同文化有不同"默认"的决策逻辑

**对PIDM的意义**:
- 感知力包括对社会情境的觉察
- 区分"我的直觉"与"他人的期待内化的声音"
- 团队决策需要协商共同的感知框架

#### 3.7 建构主义与PIDM的整合

```
┌─────────────────────────────────────────────────────────┐
│              建构主义 → PIDM 应用                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  构念系统理论 ───────► 直觉的结构分析                    │
│  核心构念概念 ───────► 重大决策的深度理解                │
│  构念弹性 ──────────► 偏见识别与矫正                     │
│  意义层次 ──────────► 深度觉察训练                       │
│  叙事心理学 ────────► 决策的生命史视角                   │
│  社会建构主义 ──────► 决策的社会情境分析                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**核心洞见**:
- 直觉不是神秘的"第六感"
- 直觉是个人构念系统快速运作的产物
- 感知力 = 对自己构念系统的元认知
- 优化决策 = 优化构念系统

---

*本章要点*:
- 建构主义强调知识是主动建构而非被动接收
- 个人构念系统是直觉运作的基础结构
- 感知力包括对建构系统的觉察能力
- 改变直觉需要重构构念系统
