### 3.1 P-E-R-C-E-I-V-E流程卡

```
┌─────────────────────────────────────────────────────────┐
│              P-E-R-C-E-I-V-E 决策流程                   │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  P - Pause (暂停)                                       │
│     □ 深呼吸3次                                         │
│     □ 问自己:"我有选择吗？"                             │
│     □ 创造空间                                          │
│                                                         │
│  E - Embody (具身)                                      │
│     □ 快速身体扫描                                      │
│     □ 注意主导感受位置                                  │
│     □ 从思考切换到感知                                  │
│                                                         │
│  R - Recognize (识别)                                   │
│     □ 这是什么情境？                                    │
│     □ 之前遇到过吗？                                    │
│     □ 我的默认反应是什么？                              │
│                                                         │
│  C - Connect (联结)                                     │
│     □ 这与我的核心价值相关吗？                          │
│     □ 利益相关者会怎么看？                              │
│                                                         │
│  E - Evaluate (评估)                                    │
│     □ 列出选项                                          │
│     □ 对每个选项身体测试                                │
│     □ 理性分析                                          │
│                                                         │
│  I - Intuit (直觉)                                      │
│     □ 倾听深层直觉                                      │
│     □ 检查真假直觉                                      │
│                                                         │
│  V - Verify (验证)                                      │
│     □ 设计小规模实验                                    │
│     □ 咨询信任的人                                      │
│                                                         │
│  E - Execute (执行)                                     │
│     □ 全身心投入                                        │
│     □ 持续觉察                                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 4.2 学员手册要点

#### 学习建议

```
如何最大化学习效果:
1. 每日练习优先于理论学习
2. 记录比记忆更重要
3. 分享比独自练习更有效
4. 耐心比速度更重要

常见困难及对策:
- 感觉不到身体 → 从强烈感受开始
- 情绪混乱 → 使用情绪词汇卡
- 怀疑直觉 → 记录并验证
- 整合困难 → 寻求督导
```
