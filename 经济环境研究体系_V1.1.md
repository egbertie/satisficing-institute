# 经济环境研究体系 V1.1

## 第一性原理声明
> **深圳南山的硬科技企业家，其决策受特定时空约束。**
> 宏观政策周期 + 中观产业生态 + 微观企业节奏 = 合伙人决策的时空背景

---

## 一、研究体系架构

### 1.1 三层监测模型

```
┌─────────────────────────────────────────────────────────────┐
│                    宏观环境监测层                            │
│  （国家政策/资本市场/国际形势）                              │
├─────────────────────────────────────────────────────────────┤
│                    中观环境监测层                            │
│  （产业集群/区域生态/供应链）                                │
├─────────────────────────────────────────────────────────────┤
│                    微观环境监测层                            │
│  （企业动态/人才流动/融资节奏）                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 研究目标

| 层级 | 研究目标 | 输出价值 |
|------|----------|----------|
| 宏观 | 把握大周期，识别系统性风险 | 为客户提供战略级合伙人建议 |
| 中观 | 跟踪区域生态，发现机会窗口 | 为客户提供产业级合伙人建议 |
| 微观 | 监测企业动态，预警风险信号 | 为客户提供战术级合伙人建议 |

## 二、宏观环境监测体系

### 2.1 监测维度

#### 维度1：硬科技政策监测

**监测范围**
| 层级 | 关注重点 | 监测渠道 |
|------|----------|----------|
| 国家层 | 科技战略规划、产业政策导向 | 国务院/科技部/工信部官网 |
| 省级层 | 广东省科技政策、产业扶持 | 广东省科技厅/工信厅 |
| 市级层 | 深圳市创新政策、人才计划 | 深圳市科创委/人社局 |
| 区级层 | 南山区专项扶持、园区政策 | 南山区政府/高新区管委会 |

**关键指标**
- 新出台政策数量及方向
- 补贴资金规模变化
- 重点支持领域调整
- 监管政策变化

**输出形式**
- 政策速报（即时）
- 政策周报（每周一）
- 政策解读（重大政策24h内）

**监测范围**
| 市场 | 关注重点 | 数据来源 |

### 2.3 宏观环境-合伙人策略映射

#### 繁荣期策略（Expansion Strategy）

**环境特征**
- 资本市场活跃，估值高企
- 融资窗口开放，资金充裕
- 人才市场竞争激烈
- 扩张机会多

**合伙人策略建议**
| 方面 | 建议 |
|------|------|
| 人才策略 | 抢人阶段，适当放宽标准，快速补齐能力短板 |
| 股权策略 | 期权池预留充足，吸引顶级人才 |
| 扩张节奏 | 激进扩张，快速占领市场 |
| 风险提示 | 警惕估值泡沫，避免过度招聘 |

**话术模板**
> "当前是硬科技的黄金时代，融资窗口开放。建议快速组建核心团队，用股权锁定关键人才。但要注意估值合理性，避免寒冬来临时被动。"

---

#### 调整期策略（Consolidation Strategy）

**环境特征**
- 融资难度增加，估值回调
- 市场回归理性
- 人才市场松动
- 需要证明PMF

**合伙人策略建议**
| 方面 | 建议 |
|------|------|
| 人才策略 | 精兵简政，保留核心，优化边缘 |
| 股权策略 | 重新评估期权分配，向核心倾斜 |
| 扩张节奏 | 聚焦核心业务，暂缓多元化 |
| 风险提示 | 关注现金流，避免核心人员流失 |

**话术模板**
> "市场进入调整期，融资更注重基本面。建议优化团队结构，把资源集中在最核心的人才上。这是一个去泡沫、练内功的好时机。"

---

#### 寒冬期策略（Survival Strategy）

**环境特征**
- 资本寒冬，融资困难
- 裁员潮起，人才供给增加
- 生存优先，削减成本
- 整合加速

**合伙人策略建议**
| 方面 | 建议 |
|------|------|
| 人才策略 | 降本增效，只保留最核心人才 |
| 股权策略 | 用股权替代现金，绑定核心团队 |
| 扩张节奏 | 收缩战线，确保 runway |
| 风险提示 | 警惕合伙人冲突加剧，加强沟通机制 |

**话术模板**
> "寒冬也是机会。优秀人才更容易吸引，竞争对手收缩给你空间。关键是守住核心团队，共度难关。建议加强合伙人之间的沟通机制，避免危机下的内耗。"

## 四、微观环境监测体系

### 4.1 目标客户动态监测

#### 监测维度

| 维度 | 指标 | 来源 |
|------|------|------|
| 融资动态 | 融资轮次/金额/投资方 | 公开披露/行业数据库 |
| 团队变化 | 高管加入/离职 | LinkedIn/脉脉 |
| 业务进展 | 产品发布/客户签约 | 新闻/官网 |
| 风险信号 | 诉讼/仲裁/失信 | 企查查/天眼查 |

#### 预警机制

**红色预警（立即关注）**
- 核心高管集体离职
- 重大诉讼/仲裁
- 融资失败传闻
- 监管调查

**黄色预警（持续关注）**
- 单个核心高管离职
- 业绩大幅波动
- 战略方向调整
- 股权变动

**绿色提示（日常关注）**
- 新产品发布
- 新客户签约
- 获得新荣誉/认证
- 媒体报道

---

### 4.2 客户情境匹配表

根据客户所处阶段和环境，提供定制化建议：

| 企业阶段 | 繁荣期建议 | 调整期建议 | 寒冬期建议 |
|----------|------------|------------|------------|
| **种子期** | 快速组建MVP团队 | 证明PMF后再扩张 | 最小团队验证 |
| **天使期** | 补齐核心能力短板 | 聚焦最核心岗位 | 只保留CTO级别 |
| **A轮** | 规模化招聘 | 优化人效 | 削减非核心岗位 |
| **B轮+** | 国际化/多元化团队 | 组织架构优化 | 降本增效 |
