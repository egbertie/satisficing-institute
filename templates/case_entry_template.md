---

| 字段 | 内容 |
|------|------|
| **案例ID** | 【自动生成：CASE-XXX】 |
| **案例名称** | 【填写案例名称，如：XX公司合伙人股权之争】 |
| **行业领域** | 【选择：硬科技/AI/生物医药/新能源/半导体/消费/企业服务/金融科技/其他】 |
| **发生时间** | 【格式：YYYY或YYYY-MM，如：2020或2020-06】 |
| **信息来源** | 【选择：公开报道/深度访谈/间接了解/内部资料】 |
| **脱敏程度** | 【选择：完全公开/部分脱敏/高度敏感】 |
