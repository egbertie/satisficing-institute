闭上眼睛，深呼吸...

1. **感受脚下的大地（土/LIU）**——我如大地般稳重可靠
2. **感受胸中的火焰（火/HUINENG）**——我如火焰般充满洞见  
3. **感受流动的智慧（水/GUANYIN）**——我如流水般自在从容
4. **感受清晰的边界（金/SIMON）**——我如金属般精准有力
5. **感受生长的仁心（木/CONFUCIUS）**——我如树木般生生不息

**默念**：火土融合，五路齐备。我已准备好。

## 二、第一阶段：土之信——根基建立（30分钟）

### 2.1 破冰与信任建立（5分钟）

**目标**：让客户放松，建立自在氛围

**Egbertie开场白**：
> "欢迎来到满意解研究所。在我们开始之前，我想先说明一点——这里不是考试，没有标准答案。我们的目标是帮你找到属于你自己的满意解。"

**动作**：
- 微笑，眼神接触
- 递上温水
- 调整自己的坐姿，展现自在状态（GUANYIN水象）

---

### 2.2 深度倾听（10分钟）

**目标**：全面理解客户的处境和需求

**核心问题**：
1. "能跟我讲讲你目前的合伙选择困境吗？"
2. "这个决策对你来说意味着什么？"
3. "如果不做这个选择，会怎样？"

**倾听原则（LIU土象）**：
- 不打断
- 不评判
- 用身体语言表示关注（点头、前倾）
- 适时复述确认："我听到你说...是这样吗？"

**命理适配**：
- 土主承载，让客户感受到"被稳稳地接住"
- 火之热情通过真诚的眼神和专注的姿态传递

---

### 2.3 价值澄清（10分钟）

**目标**：剥离表象，直击核心价值

**核心问题（LIU土象）**：
> "在聊具体人选之前，我想先理解——对你来说，一段理想的合伙关系，最核心的三个价值是什么？"

**追问技巧**：
- "为什么这个价值对你如此重要？"
- "如果只能保留一个，会是哪个？"
- "过去的选择中，你因为这个价值而后悔过吗？"

**记录方式**：
在白板上写下客户提到的核心价值（土之信的可视化）

---

### 2.4 满意解标准设定（5分钟）

**目标**：设定"足够好"的标准，避免追求完美

**核心问题（SIMON金象）**：
> "完美合伙人不存在，那什么样的合伙人算'对你来说足够好'？"

**引导框架**：
| 维度 | 问题 |
|------|------|
| 能力 | "最低需要具备哪些核心能力？" |
| 资源 | "哪些资源是必须具备的，哪些是锦上添花？" |
| 价值观 | "在价值观上，哪些是必须一致的，哪些可以有差异？" |
| 风险承受 | "你能接受的'最大不满意'是什么？" |

**输出**：
与客户共同确定3-5条"满意解标准"，写入工作表

---

## 三、第二阶段：水火济——辩证洞察（30分钟）

### 3.1 信息结构化呈现（10分钟）

**目标**：将复杂信息系统化呈现

**方法（SIMON金象）**：

如果客户有具体的候选人，使用决策矩阵：

| 候选人 | 标准1 | 标准2 | 标准3 | 标准4 | 标准5 | 总分 |
|--------|-------|-------|-------|-------|-------|------|
| A | 4/5 | 3/5 | 5/5 | 2/5 | 4/5 | 18/25 |
| B | 3/5 | 5/5 | 3/5 | 4/5 | 3/5 | 18/25 |
| C | ... | ... | ... | ... | ... | ... |

**讲解重点**：
- 数据是参考，不是答案
- 注意"区分度"——哪些标准真正区分了候选人

---

### 3.2 状态调整（10分钟）

**目标**：从理性分析切换到感性直觉，让客户放松

**引导语（GUANYIN水象）**：
> "刚才我们分析了很多信息，现在让自己放松下来。"
> "请做一个深呼吸...吸气...呼气..."
> "再做一次...吸气...呼气..."
> "第三次...吸气...让空气充满你的腹部...呼气...让紧张随之流走..."

**身体放松引导**：
> "现在，让你的肩膀放松下来..."
> "让你的下巴微微松开..."
> "让你的双手自然地放在腿上..."

**命理适配**：
- 水克火，平衡客户的焦虑（火过旺）
- 让客户进入"观自在"的状态

---

### 3.3 直觉唤醒（10分钟）

**目标**：唤醒身体直觉，获取深层洞见

**引导语（HUINENG火象）**：

**步骤1：想象可视化**
> "闭上眼睛，想象你选择的合伙人正站在你面前..."
> "不要想太多，只是让这个形象自然浮现..."

**步骤2：身体扫描**
> "现在，将注意力放到你的身体上..."
> "当你想象与A合伙人合作时，你的身体有什么感觉？"
> "胸口是紧张还是舒展？"
> "胃部是沉重还是轻松？"
> "肩膀是紧绷还是放松？"

**步骤3：直觉语言化**
> "那个感觉像什么？"
> "如果它有温度，是热的还是冷的？"
> "如果它有颜色，是什么颜色？"
> "如果它有质地，是光滑的还是粗糙的？"

**记录**：
记录客户的身体反应描述（火之直觉的可视化）

---

## 四、第三阶段：火土归——决策定格（30分钟）

### 4.1 伦理审视（10分钟）

**目标**：从长期关系角度审视决策

**核心问题（CONFUCIUS木象）**：

**仁心检验**：
> "你觉得这位候选人是否有'己欲立而立人'的心态？"
> "从你们的交往中，他/她是否表现出愿意成就他人的特质？"

**长期视角**：
> "如果我们选择A，五年后回头看，这段关系是否符合'义利合一'？"
> "当出现分歧时，你们能否'和而不同'？"

**共赢检验**：
> "这个合伙结构，是否让双方都有成长空间？"
> "还是一方只是另一方的工具？"

**命理适配**：
- 土克木，以命主价值观（土）审视伦理（木）
- 确保决策符合Egbertie的"德馨"标准

---

### 4.2 五路综合校准（10分钟）

**目标**：用五路图腾全面检视决策

**校准仪式**：

将五行罗盘转向客户，逐一校准：

| 图腾 | 校准问题 | 客户回答 |
|------|----------|----------|
| **LIU（土）** | "价值上契合吗？这个选择是否符合你最初的核心价值？" | |
| **SIMON（金）** | "方法上行得通吗？这是'足够好'的选择吗？" | |
| **GUANYIN（水）** | "你感觉自在吗？想象合作后，你的状态是放松的吗？" | |
| **CONFUCIUS（木）** | "对方有共赢之心吗？这段关系能长期生长吗？" | |
| **HUINENG（火）** | "你的直觉确认吗？身体的感觉是舒展的吗？" | |

**通过标准**：
- 5项全部通过 → 决策确认
- 4项通过 → 讨论未通过项，调整后确认
- ≤3项通过 → 建议暂缓决策，补充信息后再议

---

### 4.3 决策定格与行动规划（10分钟）

**目标**：明确决策，制定行动计划

**决策确认**：
> "综合五路图腾的校准，你的决策是什么？"
> "是选择A，还是B，还是暂时不选择？"

**行动规划**：
| 时间 | 行动 | 负责人 |
|------|------|--------|
| 本周 | | |
| 本月 | | |
| 三个月内 | | |

**风险预案**：
> "如果合作中出现X情况，你会怎么处理？"
> "你的退出机制是什么？"

## 六、特殊情况处理

### 6.1 客户过度焦虑（火旺）

**表现**：语速快、身体紧绷、反复询问"万一错了怎么办"

**处理**：
1. 延长第二阶段的水象调节时间（增加呼吸练习）
2. 强调"满意解"而非"完美解"（SIMON金象平衡）
3. 分享类似案例，降低焦虑

---

### 6.2 客户过度理性（金旺）

**表现**：要求更多数据、难以进入直觉环节、质疑身体感受的可靠性

**处理**：
1. 先用LIU（土）建立信任，让其放下戒备
2. 用科学语言解释直觉（"躯体标记假说"）
3. 缩短分析时间，强制进入直觉环节

---

### 6.3 客户无法决策（水旺/木旺）

**表现**：优柔寡断、不断追加新问题、难以聚焦

**处理**：
1. 强化SIMON（金）的满意解边界设定
2. 设定限时决策机制
3. 必要时建议"小规模试错"而非"一次性决策"

---

### 6.4 客户命理偏向调整（如客户愿意提供）

| 客户命理 | 调整策略 |
|----------|----------|
| 火旺 | 增加水象调节，强调冷静期 |
| 土旺 | 增加火象直觉唤醒，避免固执 |
| 金旺 | 增加土象价值对话，缩短分析 |
| 水旺 | 增加金象边界设定，限时决策 |
| 木旺 | 强化伦理审视，增加现实约束 |
