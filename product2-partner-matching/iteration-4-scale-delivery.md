### 2.3 标准化要素

| 要素 | 标准化内容 | 可调整空间 |
|-----|-----------|-----------|
| **师资** | 认证讲师(40小时培训+考核) | 行业背景适配 |
| **场地** | 合作酒店/会议中心标准配置 | 城市级别差异 |
| **材料** | 统一课件+工作手册+工具包 | 行业案例替换 |
| **测评** | 标准化测评系统 | 深度版可选升级 |
| **报告** | 模板化报告(20页) | 关键发现个性化 |

```

│  ├── 屏幕共享(协作任务)                                 │
│  ├── 实时问卷系统                                       │
│  └── 视频录制(分析用，加密存储)                         │
│                                                          │
│  服务端                                                  │
│  ├── AI视频分析(微表情识别)                             │
│  ├── 语音转文字(NLP分析)                                │
│  ├── 实时数据同步                                       │
│  └── 报告自动生成                                       │
│                                                          │
│  安全与合规                                              │
│  ├── 数据加密传输                                       │
│  ├── 隐私协议签署                                       │
│  └── 视频存储期限约定(30天后删除)                       │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 3.3 评估流程

```
Pre-session (提前3天)
├── 在线问卷填写 (PFI完整版)
├── 资料收集 (背景/业务/期望)
├── 技术测试 (视频/音频)
└── 准备指南发送

Session 1 (90分钟) - 个人深度访谈
├── 甲方访谈 (45分钟)
└── 乙方访谈 (45分钟)

Session 2 (90分钟) - 双人互动
├── 结构化任务 (30分钟)
├── 冲突场景模拟 (30分钟)
└── 愿景讨论 (30分钟)

Post-session (3个工作日内)
├── AI辅助分析
├── 专家解读
├── 报告生成
└── 视频解读会(30分钟)
```

|-----|-----------|-------|
| **价格** | ¥88,000 | ¥4,800 |

| **神经科学** | 全量(眼动/脑波/HRV) | 视频AI分析(微表情) |
