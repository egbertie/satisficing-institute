# 第1篇：产品理念与定位

## 1.1 市场背景

中国每年新增创业公司超过300万家，其中约70%采用合伙制。然而：
- **65%** 的合伙关系在3年内出现严重冲突
- **40%** 的创业公司分手与合伙关系破裂直接相关
- **90%** 的合伙人在合伙前未进行系统匹配评估

**痛点：** 合伙人选择是当前创业领域最大的"非理性决策"之一，依赖直觉、缺乏科学工具。

## 1.2 产品使命

> **让每一次合伙，都建立在科学匹配的基础之上。**

通过多学科融合的诊断体系，帮助合伙人：
1. **事前：** 科学评估匹配度，降低选错人风险
2. **事中：** 预警潜在冲突，提供干预方案
3. **事后：** 友好分手机制，降低分手损失

```
AI对齐专家 → Egbertie的共情能力（对齐模式）
```

**应用原则**：
- 创业者高自信区 → 对齐模式（建立信任）
- 创业者低自信区 → 互补模式（精准纠错）
- 路由依据：22年经验训练的直觉感知

## 2.2 神经科学测量详解

### 2.2.1 眼动追踪 (EyeSync模块)

**设备：** Tobii Pro Glasses 3 / Pupil Labs

**核心指标：**
| 指标 | 含义 | 合伙匹配意义 |
|-----|------|-------------|
| 注视热点重叠度 | 双方关注点的重合程度 | 价值观/优先级对齐度 |
| 扫视同步率 | 视线移动的时间相关性 | 默契程度/思维同步 |
| 瞳孔变化协方差 | 情绪唤醒的同步程度 | 情感共鸣能力 |
| 回避模式 | 特定话题的视线回避 | 冲突敏感点识别 |

### 2.2.2 心率变异性 (HRVSync模块)

**关键发现：**
- **高同步性：** 压力下HRV变化趋势一致，表明情绪共振能力强
- **恢复速度差：** 一方恢复快、一方恢复慢，可能产生长期张力
- **基线差异：** 长期压力水平差异大，生活节奏可能不协调

### 2.2.3 脑波分析 (BrainWave模块)

**频带解读：**
| 频带 | 频率 | 状态 | 合伙意义 |
|-----|------|-----|---------|
| Delta | 0.5-4Hz | 深度放松/睡眠 | 恢复力指标 |
| Theta | 4-8Hz | 创造力/直觉 | 创新风格 |
| Alpha | 8-12Hz | 放松专注 | 压力调节能力 |
| Beta | 12-30Hz | 活跃思考 | 认知风格 |
| Gamma | 30-100Hz | 高级认知 | 复杂问题解决 |

## 2.3 潜意识测试 (IAT)

**测试维度：**
1. **权力态度：** 对领导/服从的内隐偏好
2. **风险偏好：** 冒险/保守的潜意识倾向
3. **信任模式：** 信任/怀疑的自动联想
4. **冲突态度：** 对冲突的内隐评价
5. **成功归因：** 能力/运气的内隐归因

**显隐一致性分析：**
- **高一致：** 内外如一，预期行为稳定
- **低一致：** 内外冲突，可能出现意外行为

## 2.4 语言与微表情分析

### 语言DNA分析维度
- **权力语言：** "应该"/"必须"/"建议"的使用比例
- **确定性：** "绝对"/"可能"/"也许"的分布
- **参照系：** "我"/"我们"/"你"的频率
- **时间导向：** 过去/现在/未来聚焦程度

### 微表情识别
**重点关注：**
- 压抑的负面情绪（表面同意，内心抗拒）
- 真实兴趣点（瞳孔扩张+微表情愉悦）
- 压力信号（频繁眨眼+面部僵硬）

### 3.2.4 最优股权建议

```

- 贡献评估 = 资本 + 行业经验 + 网络 + 时间 + 风险承担 + 执行能力

- 博弈均衡 = 纳什均衡检验(双方都能接受的方案)
```

| 反馈艺术 | 建设性反馈 | SBI-I模型训练 |

### 4.3.2 SBI-I反馈模型
