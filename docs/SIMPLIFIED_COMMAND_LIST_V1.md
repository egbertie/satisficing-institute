## 五、指令执行标准

### 5.1 「启动第一性原则」执行标准

**触发条件**: 用户说出「启动第一性原则」+ 任何任务描述

**执行流程**:
```
1. 解析任务描述，确定复盘对象
2. 执行五步复盘法：
   - 观察：收集相关数据
   - 复盘：分析根因
   - 思考：理论最优
   - 优化：制定方案
   - 实施：写入Skill代码
3. 生成可执行Skill文件
4. 部署并启用强制执行
5. 报告执行结果和后续验证计划
```

**输出要求**:
- 必须生成可执行的Skill代码
- 必须设置检查/复盘机制
- 必须说明验证方式
- 必须在24小时内首次验证

### 5.2 其他指令执行标准

所有简化指令执行后需报告：
1. 指令识别结果
2. 执行动作摘要
3. 产出物位置
4. 后续验证计划（如有）
