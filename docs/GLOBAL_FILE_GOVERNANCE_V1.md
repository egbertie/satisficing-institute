**检查脚本**:

### 维度2: 内容时效性（每日09:00 + 周六深度）

**过时内容识别规则**:
| 规则 | 触发条件 | 动作 |
|------|----------|------|
| 日期引用 | 倒计时超过3天未更新 | 标记过时 |
| 版本声明 | "最后更新"超过7天 | 提示检查 |
| 任务状态 | "进行中"超过截止日 | 标记逾期 |
| 链接失效 | 引用路径不存在 | 标记失效 |

**检查脚本**:
```bash
# 检查过时日期引用
grep -r "倒计时.*天" --include="*.md" | grep -v "$(date +%Y-%m-%d)"

# 检查任务逾期
grep -r "进行中" --include="*.md" | grep "截止.*2026-03"
```

### 维度3: 引用一致性（每周六10:00）

**必检引用类型**:
| 类型 | 示例 | 检查方式 |
|------|------|----------|
| 内部引用 | `A满意哥专属文件夹/XX/XX.md` | 验证路径存在 |
| 跨目录引用 | `skills/XXX/SKILL.md` | 验证版本一致 |

| 记忆引用 | `MEMORY.md#章节` | 验证章节存在 |
