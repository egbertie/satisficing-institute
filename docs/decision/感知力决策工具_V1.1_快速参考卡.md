## 📊 核心指标速查

### PFI - 合伙人匹配度指数

| 分数 | 等级 | 建议 |
|------|------|------|
| ≥7.0 | 优秀 | 推荐合作 |
| 5.1-6.9 | 良好 | 条件推荐 |
| 3.6-5.0 | 一般 | 谨慎合作 |
| <3.6 | 较差 | 不建议 |

### PSI - 感知力指数

| 分数 | 等级 | 决策模式 |
|------|------|----------|
| ≥7.0 | 高 | 信任直觉 |
| 5.0-6.9 | 中 | 直觉+验证 |
| <5.0 | 低 | 依赖逻辑 |

### 置信度 - 综合决策指标

| 分数 | 等级 | 行动 |
|------|------|------|
| ≥7.5 | 极高 | 全力推荐 |
| 6.0-7.4 | 高 | 标准推进 |
| 4.5-5.9 | 中 | 风险缓释 |
| <4.5 | 低 | 深度调查 |

## ⚠️ 冲突检测速查

### 类型A：逻辑高分 + 感知红灯
**信号**：
- PFI≥7.0 但身体持续紧张
- 7维度优秀但感觉"不对劲"
- 对方回答完美但缺乏真实情感

**行动**：
1. 延长观察期2-4周
2. 进行压力情境测试
3. 引入第三方背景调查

### 类型B：逻辑低分 + 感知绿灯
**信号**：
- PFI<5.0 但相处舒适
- 有明显短板但感觉"就是这个人"
- 理性不建议但感性想推进

**行动**：
1. 重新检查7维度评分
2. 评估是否过度依赖"熟悉感"
3. 设置试用期+退出机制

## 🚦 躯体信号红绿灯

### 🟢 绿灯信号（正向）
- 身体放松，肩膀下沉
- 呼吸平稳深沉
- 眼神交流自然
- 面带微笑，肌肉放松
- 内在"对了"的确认感

### 🔴 红灯信号（警示）
- 肩膀紧绷，身体后仰
- 呼吸变浅或屏息
- 眼神躲闪或过度凝视
- 面部肌肉僵硬
- 胃部不适或紧绷感
- 想尽快结束对话的冲动

## 📈 常用公式

```
PFI = Σ(维度得分 × 维度权重)

PSI = 直觉力×0.35 + 洞察力×0.35 + 共情力×0.30

置信度 = PFI×0.6 + min(PSI,躯体信号)×0.4 - 冲突惩罚

冲突惩罚：
• 类型A冲突：-1.5
• 类型B冲突：-0.5
• 无冲突：0
```
