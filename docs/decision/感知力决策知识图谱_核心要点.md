# 感知力决策知识图谱核心要点

## 核心理论框架（已整合）

### 1. 证据累积理论家族
| 模型 | 核心机制 | 关键参数 | 满意解应用 |
|------|----------|----------|------------|
| **漂移-扩散模型(DDM)** | 单一证据累积器，随机游走 | 漂移率v、边界阈值a、非决策时间Ter | 合伙人评估的量化框架 |
| **线性弹道累积器(LBA)** | 并行线性累积器，竞争机制 | 漂移率分布、阈值A | 多选项合伙人比较 |
| **泄漏累积器(LCA)** | 证据衰减+选项间抑制 | 泄漏率λ、抑制强度β | 情境效应分析 |
| **紧急门控模型** | 动态阈值降低 | 紧急信号增益γ | 限时决策（72小时） |

### 2. 双重系统理论（卡尼曼）
| 维度 | 系统1（直觉） | 系统2（分析） | 满意解融合 |
|------|---------------|---------------|------------|
| 速度 | 快（毫秒至秒级） | 慢（秒至分钟级） | 左脑+右脑协同 |
| 意识 | 无意识/前意识 | 意识清晰 | 躯体信号+理性分析 |
| 努力程度 | 低 | 高 | 感知力训练降低系统2负担 |
| 典型情境 | 专家直觉、危机反应 | 复杂分析、陌生问题 | 合伙人匹配决策 |

**满意解创新**：不是二选一，而是"感知力桥接"——用系统1的快速模式识别，引导系统2的深入分析

### 3. 神经科学基础（已验证）

**关键发现**：
- **早期视觉皮层(V1)**存在内源性选择预测性活动（*Nature* MEG研究）
- **眶额皮层(OFC)**编码主观价值，价格信息可改变真实体验（葡萄酒实验）
- **躯体标记假说**(Damasio)：情感在复杂决策中的功能性作用
- **心理理论网络**：颞顶联合区(TPJ)、内侧前额叶(mPFC)参与合伙人意图推断
- **人际神经同步**：合作中的脑-脑耦合预测合作绩效

**满意解应用**：
- 合伙人评估 = 信号检测理论(SDT) + 贝叶斯决策
- 直觉训练 = 知觉学习的皮层重组
- 冲突管理 = 社会感知决策的神经机制

### 4. 西蒙"满意解"理论的科学验证

**有限理性(Bounded Rationality)** → **满意性原则(Satisficing)**
- 人类无法执行完全理性的优化计算
- 设定可接受性阈值，选择首个满足阈值的选项
- 将决策从"瞬间选择"重新概念化为"问题解决过程"

**满意解研究所方法论的科学基础**：
1. **感知输入的多模态整合** ← 躯体标记假说
2. **认知评估的情境依赖加工** ← 前景理论框架效应
3. **行动选择的适应性执行** ← 漂移-扩散模型

### 5. 合伙人关系感知力决策（核心重点）

**能力-意愿权衡模型**（已整合进合伙人评估）：
- 能力信号：教育背景、过往业绩、技术资产
- 意愿信号：承诺表达、关系投资、价值契合
- 感知偏差：光环效应、幸存者偏差、社会期望偏差

**信任建立的感知循环**：
- 初始信任阈值设定 ← 关系历史、制度保障
- 信任违背的修复 ← 道歉、补偿、制度变革
- 信任升级标志 ← 信息共享深度、相互依赖范围

**决策权对说服信息处理的调节**（牛津大学研究）：
- 有决策权的个体 → 心理理论网络更强功能连接 → 深度评估
- 无决策权的个体 → 表面信息处理
- **启示**：赋予合伙人实质性决策权，提升合作信息评估质量
