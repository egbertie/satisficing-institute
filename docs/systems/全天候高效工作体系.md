## 五、高性价比原则

### 5.1 资源使用优先级

```
1. 本地计算（免费）→ 优先
2. 已付费资源（WPS/Notion会员）→ 充分利用
3. 按量付费API（Kimi）→ 合理使用，避免浪费
4. 订阅服务（Claude）→ 必要时启用
```

### 5.2 成本控制检查清单

每项任务执行前检查：
- [ ] 能用本地解决吗？
- [ ] 能用已付费资源吗？
- [ ] API调用必要吗？
- [ ] 有批量处理方式吗？
- [ ] 结果能复用吗？

### 5.3 效果追踪

每周评估：
- API调用次数 vs 产出价值
- 夜间任务完成率
- 人类工作效率提升度
- 整体成本效益比
