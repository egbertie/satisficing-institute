## 📊 关键创新点

### 1. 双引擎架构
```
左脑引擎（逻辑）    右脑引擎（感知）
    ↓                    ↓
   PFI                  PSI
    ↓                    ↓
    └───────┬────────────┘
            ↓
      冲突检测引擎
            ↓
       整合决策输出
```

### 2. 冲突检测机制
- **类型A**：逻辑高分+感知红灯 → 深度调查
- **类型B**：逻辑低分+感知绿灯 → 重新评估
- **无冲突**：高置信度推进

### 3. 置信度计算公式
```
置信度 = PFI×0.6 + min(PSI,躯体信号)×0.4 - 冲突惩罚

冲突惩罚：
• 类型A：-1.5分（感知警告优先）
• 类型B：-0.5分（需重新评估）
```
