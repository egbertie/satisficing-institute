## 三、Kimi Code集成方案

### 方案A：API直接调用（推荐）

**步骤**：
1. 获取你的Kimi API Key
2. 配置到OpenClaw
3. 创建kimi-code Skill

**配置方式**：
```bash
# 设置环境变量
export KIMI_API_KEY=你的API密钥

# 或配置到OpenClaw
openclaw config set kimi.api_key=你的密钥
```

**使用方式**：
```
你说："用Kimi Code写一个Python脚本"
AI调用Kimi API生成代码
返回结果
```

### 方案B：浏览器自动化（备选）

**步骤**：
1. 登录Kimi网页版
2. 使用Playwright控制浏览器
3. 自动输入提示词、获取结果

**缺点**：
- 速度慢
- 不稳定
- 不推荐
