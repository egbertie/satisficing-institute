# TOOLS.md - 工具使用规则更新

## 新增规则：WPS会员服务联动使用

**生效日期**：2026-03-09  
**适用场景**：文档协作、问卷工具、表格处理等

---

### 规则说明

**Egbertie拥有WPS最高会员级别**，在使用工具时：

1. **优先使用WPS生态**
   - 问卷/表单：优先使用WPS表单（替代问卷星）
   - 文档协作：优先使用WPS云文档（替代飞书文档如有障碍）
   - 表格处理：优先使用WPS表格（替代Excel/Google Sheets）

2. **充分产生联动效应**
   - 问卷数据自动同步到WPS表格
   - 表格数据自动关联到WPS文档
   - 利用WPS会员的高级功能（宏、模板、协作等）

3. **操作原则**
   - 能使用WPS解决的，不使用第三方工具
   - 充分利用会员特权（大容量、高级模板、API等）
   - 保持数据在WPS生态内流转

---

### 具体应用场景

| 场景 | 工具选择 | 联动方式 |
|------|---------|---------|
| TRL自评工具 | WPS表单 | 表单→表格→文档报告 |
| 合伙人评估 | WPS表单 | 评估→自动评分→结果汇总 |
| 数据收集 | WPS表格 | 多表关联+自动图表 |
| 文档协作 | WPS云文档 | 评论、修订、版本管理 |

---

**规则级别**：重要（与飞书协作、最小权限同等重要）
