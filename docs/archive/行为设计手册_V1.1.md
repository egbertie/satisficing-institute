# 行为设计手册 V1.1

## 第一性原理声明
> **信任建立 = 可验证的专业 + 可感知的风险意识 + 可追溯的成功案例**
> 每一个触点都是建立信任的机会，每一个行为都是品牌的一部分。

---

## 一、触点行为设计体系

```

### 1.2 首次接触设计

#### 3秒价值传递

**核心原则**：第一句话必须回答"你能为我解决什么问题"

**设计要素**
| 要素 | 要求 | 示例 |
|------|------|------|
| 问题共鸣 | 立即点出痛点 | "合伙人选择是创业第一大坑" |
| 身份定位 | 明确服务谁 | "专注硬科技创始人" |
| 价值承诺 | 量化结果 | "避开85%的失败风险" |

**场景应用**

**网站Banner**
```
主标题：选对合伙人，避开创业第一大坑
副标题：11年银行风控 + 11年创业经验，专为硬科技创始人服务
CTA：免费获取合伙人风险评估
```

**名片/介绍**
```
满意解研究所
专注硬科技合伙人决策教练
帮你选对合伙人，分好股权，走稳第一步
```

#### 15秒信任建立

**核心原则**：在15秒内提供可信证据

**信任要素序列**
```
1-5秒：专业背景（11年银行风控 + 11年创业）
6-10秒：客户背书（服务XX家硬科技企业）
11-15秒：成果证明（帮助客户避免XX损失）
```

**触点设计**

**首页信任区**
```
┌─────────────────────────────────────┐
│  ✓ 11年银行风控经验                 │
│  ✓ 11年创业实战经验                 │
│  ✓ 服务50+硬科技企业                │
│  ✓ 帮助客户避免数亿元潜在损失       │
└─────────────────────────────────────┘
```

**开场白结构**
```
"我是满意解研究所的创始人，专注帮助硬科技创业者解决合伙人选择问题。
之前我在银行做了11年风控，看过太多企业因为合伙人问题倒下；

```

#### 60秒兴趣激发

**核心原则**：在60秒内让客户想要了解更多

**激发路径**
```
0-15秒：建立信任（如上）
15-30秒：痛点共鸣（讲述一个典型案例）
30-45秒：解决方案预告（我们的方法如何解决）
45-60秒：行动召唤（下一步可以做什么）
```

**60秒电梯演讲完整版**
```
[0-15秒：建立信任]

[45-60秒：行动召唤]
您要是有合伙人选择的困扰，我可以免费做个15分钟的初步诊断，
帮您识别一下潜在风险。这是我的微信，您方便时联系我。

#### 诊断过程可视化

**核心原则**：让客户"看见"专业过程，而不只是结果

**可视化设计**

**诊断流程展示**
```

        └─ 企业背景、团队结构、发展阶段

        └─ 四维度评估：潜力、匹配、动机、风险

#### 评估结果故事化

**核心原则**：用故事包装数据，让客户产生情感共鸣

**故事化框架：SCQA模型**

| 要素 | 说明 | 应用示例 |
|------|------|----------|
| S（情境） | 背景设定 | "您和合伙人张总认识3年，一起从大公司出来创业" |
| C（冲突） | 矛盾点 | "但在技术路线选择上，你们存在分歧" |
| Q（问题） | 核心问题 | "这种分歧如果不解决，可能影响下一轮融资" |
| A（答案） | 解决方案 | "建议设立技术决策委员会，引入外部专家..." |

**评估报告故事化结构**
```
【第一章：旅程回顾】
过去3个月，您和合伙人共同经历了...
（描述共同经历，建立情感连接）

【第二章：十字路口】
根据QPMS评估，你们当前处于关键决策点...
（呈现评估数据，用图表可视化）

【第三章：前路地图】
基于数据分析，我们识别了三条可能的路径...
（提供可选方案）

【第四章：行动指南】
建议分三步走：第一步...第二步...第三步...
（给出可执行建议）
```

---

#### 建议方案可执行化

**核心原则**：给的不是建议，是"明天就能开始做的行动清单"

**可执行化标准**

| 标准 | 要求 | 示例对比 |
|------|------|----------|
| 具体化 | 不说"加强沟通"，说"每周五下午3点一对一会议" | ❌加强沟通 → ✅每周五一对一 |
| 可量化 | 有明确的衡量指标 | ❌提高效率 → ✅决策周期缩短至3天 |
| 有时限 | 明确的deadline | ❌尽快完成 → ✅本月底前完成 |
| 有责任人 | 明确谁来做 | ❌团队完成 → ✅CEO牵头，CTO配合 |

**行动清单模板**
```
【90天行动计划】

第1-30天：基础建设期
├─ [ ] 完成股权协议修订（责任人：CEO，DDL：本月15日）
├─ [ ] 建立周度合伙人沟通机制（责任人：合伙人共同，DDL：下周一开始）
└─ [ ] 引入外部顾问参与重大决策（责任人：CEO，DDL：本月30日）

第31-60天：机制运行期
├─ [ ] 试运行新决策流程（责任人：全体，DDL：持续）
├─ [ ] 收集反馈并优化（责任人：CEO，DDL：第45天评估）
└─ [ ] 完成第一次合伙人复盘（责任人：全体，DDL：第60天）

第61-90天：巩固优化期
├─ [ ] 固化有效机制（责任人：CEO，DDL：第75天）
├─ [ ] 识别并解决遗留问题（责任人：相关方，DDL：第90天）
└─ [ ] 制定下季度合伙人发展计划（责任人：全体，DDL：第90天）
```

### 1.4 长期接触设计

#### 定期价值输出

**核心原则**：即使在没有项目时，也要让客户感受到持续价值

**价值输出节奏**

| 频率 | 形式 | 内容 | 目标 |
|------|------|------|------|
| 每周 | 行业简报 | 硬科技环境周报（摘要版） | 保持连接 |
| 每月 | 深度洞察 | 专题分析/案例分享 | 展示专业 |
| 每季 | 健康检查 | 合伙人关系复盘 | 预防风险 |
| 每年 | 战略规划 | 下年度合伙人策略 | 深度绑定 |

**内容模板：周报摘要**
```
【满意解研究所·硬科技环境周报】
本周值得关注的3件事：

1️⃣ 政策动向
《XX政策》发布，对AI企业融资有影响...

本周硬科技领域发生XX起融资...
【趋势洞察】投资人更关注...

3️⃣ 风险预警
某AI公司合伙人纠纷曝光...
【给我们的启示】建议关注...

---
如需详细版报告或讨论对贵司的影响，随时联系我。
```

#### 关键时刻触发

**核心原则**：在客户最需要的时候出现

**关键时刻地图**

| 客户时刻 | 触发动作 | 提供价值 |
|----------|----------|----------|
| 启动融资 | 发送投资人FAQ + 合伙人故事包装 | 帮助讲好团队故事 |
| 收到TS | 股权架构快速诊断 | 避免条款陷阱 |
| 团队扩张 | 新合伙人评估 | 确保文化匹配 |
| 遭遇危机 | 2小时内响应 + 危机应对指南 | 稳定军心 |
| 年度复盘 | 年度合伙人健康报告 | 系统化复盘 |

**触发机制**
```
监控信号 → 触发条件 → 行动 → 跟进

示例：
监控：客户公司新闻
触发：公布融资消息
行动：发送祝贺 + 融资后合伙人管理指南
跟进：一周后询问是否需要股权架构优化服务
```

### 2.2 异议处理话术

#### "太贵了"

**应对策略：价值对比法**

```
理解您的顾虑。让我帮您算笔账：

我们的深度诊断服务是3万元，
但如果选错合伙人，代价是什么？

• 股权纠纷律师费：10-50万
• 团队分裂重建成本：6-12个月时间
• 公司估值损失：可能腰斩
• 个人精力消耗：无法估量

上个月有个客户，因为没有做评估，
合伙人纠纷导致公司估值从2亿跌到5000万，
损失1.5亿。

花3万避免潜在千万级损失，这个投资回报率如何？

而且，如果我们诊断后发现您没有风险，
这3万就当买个安心，也是值得的，对吧？
```

#### "不需要"

**应对策略：痛点唤醒法**

```
理解，可能您目前的合伙人关系很顺畅。

冒昧问一句：
如果合伙人突然提出不同意见，您有把握能达成一致吗？

我见过太多看似和谐的团队，
在融资条款、技术路线、市场策略上出现分歧，
最后撕破脸。

而且，很多时候问题不是"有没有"，
而是"什么时候暴露"。

我们可以免费做个15分钟的快速诊断，
帮您识别一下潜在的风险点。
即使没有发现问题，也算排除隐患，您说呢？
```

#### "再考虑考虑"

**应对策略：紧迫感营造**

```
当然，重大决策确实需要慎重。

不过我想提醒您一点：
合伙人评估有个最佳时机——
在正式合作之前，或者至少是在股权分配确定之前。

一旦股权定了、协议签了，
后续调整的成本会高很多。

您现在处于什么阶段？
如果还在窗口期，建议尽快做个评估，
别让犹豫成为后悔的开始。

我可以这周就安排时间，
您看周三下午还是周四上午方便？
```

#### "我们自己能解决"

**应对策略：专业分工引导**

```
相信您的能力，能走到今天这一步肯定不简单。

不过，合伙人评估这事儿有个特点——
当局者迷。

就像医生不会给自己做手术，
律师不会给自己辩护，
自己做的评估很难客观。

而且，您的时间应该花在业务上，
合伙人评估我们有成熟的体系，
2周就能出结果，
您自己研究可能要花几个月。

专业的事交给专业的人，
您专心做您最擅长的事，不是更好吗？

我们可以先做个免费的初步诊断，
让您感受一下专业评估和自评的区别，
您再决定要不要深入合作，如何？
