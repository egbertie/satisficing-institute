## 📱 飞书集成配置

### 自动化规则（通过n8n实现）

**规则1：新候选人通知**
- 触发：Notion新增候选人
- 动作：飞书群通知 + 分配给负责人

**规则2：任务到期提醒**
- 触发：截止日期前24小时
- 动作：飞书私信提醒负责人

**规则3：状态变更同步**
- 触发：候选人状态变更
- 动作：更新飞书多维表格 + 通知相关人员

**规则4：周报生成**
- 触发：每周五17:00
- 动作：统计本周进展 → 生成报告 → 发送飞书
