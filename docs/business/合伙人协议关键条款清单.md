# 合伙人协议关键条款清单

> 硬件初创企业合伙人协议核心条款速查  
> 建议：每一条款都需律师审核，本清单仅供参考

---

## 一、基础信息条款

### 1.1 协议主体

```
甲方（创始人）：姓名、身份证号、联系方式
乙方（合伙人）：姓名、身份证号、联系方式
丙方（如有）：姓名、身份证号、联系方式

公司信息：
- 公司名称：_______________
- 注册地址：_______________
- 成立日期：_______________
- 注册资本：_______________
```

### 1.2 合伙目的

```
各方基于共同创业目的，就设立并经营__________公司
（以下简称"公司"）达成如下协议。

公司主营业务：硬件产品研发、生产、销售
（具体范围以工商登记为准）
```

## 二、股权与出资条款（核心）

### 2.1 股权分配

| 股东 | 认缴出资额 | 股权比例 | 出资方式 |
|------|-----------|---------|---------|
| 甲方 | ___万元 | ___% | 现金/知识产权/实物 |
| 乙方 | ___万元 | ___% | 现金/知识产权/实物 |
| 期权池 | - | ___% | 预留 |

**硬件行业特殊约定**：
```
技术出资的估值方式：
- 专利/技术秘密：经第三方评估机构估值
- 技术入股上限：不超过注册资本的 ___%
- 技术出资的权属：入股前已归公司所有
```

### 2.2 股权兑现（Vesting）

**标准条款**：
```
2.2.1 兑现期
各方持有的公司股权分 ___ 年兑现（Vesting），
自本协议签署之日起计算。

2.2.2 兑现方式
- 第一年（12个月）为"悬崖期"，期满兑现 ___%
- 之后按月兑现，每月兑现 1/___
- 满 ___ 年后，全部股权兑现完毕

2.2.3 加速兑现
以下情况触发加速兑现：
- 公司被收购或合并：剩余股权立即兑现
- 公司IPO：剩余股权按锁定期规定处理
- 一方因公司原因被解除职务：已工作时间的股权兑现
```

### 2.3 股权回购

**离职回购**：
```
2.3.1 主动离职
若一方主动离职（提前 ___ 日书面通知）：
- 已兑现股权：保留，但受竞业限制约束
- 未兑现股权：公司按【成本价/净资产值/公允价较低者】回购

2.3.2 被动离职（被开除）
若一方因以下原因被解除职务：
- 严重失职、渎职，给公司造成重大损失（>___万元）
- 违反竞业限制，从事竞争业务
- 刑事犯罪
则：全部股权（含已兑现）按成本价回购

2.3.3 回购程序
- 离职/开除决定作出后 ___ 日内，公司发出回购通知
- 回购价格由双方协商，协商不成委托第三方评估
- 回购款分 ___ 期支付，每期 ___ 个月
```

## 三、股东权利与义务

### 3.1 表决权

**基础表决权**：
```
3.1.1 按股权比例行使表决权，
重大事项需 ___% 以上表决权通过。

重大事项包括：
- 修改公司章程
- 增加或减少注册资本
- 公司合并、分立、解散
- 变更主营业务
- 对外担保（单笔 >___万元）
- 处置核心资产（单笔 >___万元）
```

**特殊表决权安排**：
```
3.1.2 以下事项需全体股东一致同意：
- 引入新合伙人/股东
- 股权比例调整
- 核心技术转让/授权
- 公司出售或控制权变更

3.1.3 技术决策权
以下事项需【技术合伙人】同意：
- 核心技术路线变更
- CTO 任命/罢免
- 重大技术合作（授权费 >___万元）
- 产品停产或重大变更
```

### 3.2 分红权

```
3.2.1 分红条件
公司有可分配利润时，经股东会决议可分红。

3.2.2 分红比例
按股权比例分红，但：
- 需预留 ___% 作为发展基金
- 需优先偿还股东借款（如有）

3.2.3 分红时间
每年 ___ 月进行年度分红决策。
```

### 3.3 知情权

```
3.3.1 各股东有权查阅：
- 公司章程、股东会决议
- 财务账簿、会计报表
- 重大合同（单笔 >___万元）

3.3.2 查阅程序
- 提前 ___ 日书面申请
- 在合理时间和地点查阅
- 可委托会计师/律师协助
```

## 四、分工与职责

### 4.1 职位与分工

| 股东 | 职位 | 主要职责 |
|------|------|---------|
| 甲方 | CEO/总经理 | 战略规划、融资、对外合作、团队管理 |
| 乙方 | CTO/技术合伙人 | 技术研发、产品、供应链、生产 |

### 4.2 全职承诺

```
4.2.1 各方承诺自本协议签署之日起，
全职投入公司经营，不得在其他公司任职或兼职
（经其他股东书面同意的除外）。

4.2.2 若一方违反全职承诺：
- 其他股东有权要求其改正
- 逾期未改正的，视为自动离职，适用股权回购条款
```

### 4.3 薪酬与报销

```
4.3.1 各方薪酬
- 公司成立前 ___ 个月：不领薪酬或每月 ___ 元生活费
- 公司实现收入后：按市场水平发放薪酬
- 具体金额由股东会决定

4.3.2 费用报销
因公司业务产生的合理费用，凭发票报销：
- 差旅费
- 业务招待费（单次 <___元，月度总额 <___元）
- 办公费用
```

## 五、知识产权条款（硬件行业核心）

### 5.1 知识产权归属

```
5.1.1 职务成果
各方在合伙期间，利用公司资源（资金、设备、信息）
产生的所有知识产权归公司所有，包括：
- 专利、实用新型、外观设计
- 软件著作权
- 技术秘密、Know-how
- 商标、域名
- 产品外观设计

5.1.2 既往成果
各方在合伙前已拥有的知识产权，
经书面清单确认后，仍归原所有人所有。
但若在公司业务中使用，需无偿授权给公司使用。

5.1.3 专利申请
- 职务发明创造，由公司决定申请专利或作为技术秘密
- 发明人享有署名权和公司规定的奖励
```

### 5.2 知识产权保护

```
5.2.1 保密义务
各方对公司的技术秘密、商业秘密负有保密义务，
保密期：合伙期间及离职后 ___ 年。

5.2.2 泄密责任
违反保密义务，给公司造成损失的：
- 赔偿全部经济损失
- 公司有权回购其全部股权
- 承担相应法律责任

5.2.3 竞业限制
- 合伙期间：不得从事竞争业务
- 离职后 ___ 年内：不得从事竞争业务
- 竞业限制补偿：离职后每月支付 ___ 元
```

## 六、融资与股权稀释

### 6.1 融资决策

```
6.1.1 融资权限
- 单笔融资 <___万元：CEO 决定
- 单笔融资 ≥___万元：需股东会 ___% 通过

6.1.2 反稀释保护
若公司后续融资估值低于本轮估值（Down Round）：
- 触发反稀释条款
- 具体计算方式：加权平均法/完全棘轮法（二选一）

6.1.3 优先购买权
新股东进入时，现有股东有权按股权比例优先认购
（Right of First Refusal）
```

### 6.2 政府补贴条款

```
6.2.1 补贴归属
公司获得的政府补贴、奖励、税收优惠等归公司所有，
不影响各方股权比例。

6.2.2 补贴申请分工
- 甲方负责：政府关系、材料申报、答辩
- 乙方负责：技术方案、产品演示、技术答辩

6.2.3 补贴使用
单笔使用 >___万元需股东会同意。

6.2.4 补贴与股权
若补贴要求股东同比例增资：
- 各方应按股权比例履行出资义务
- 若一方无法出资，另一方可代为出资
- 代出资金额作为借款，不计入股权
```

## 七、退出机制

### 7.1 自愿退出

```
7.1.1 提前通知
一方拟退出，需提前 ___ 日书面通知其他股东。

7.1.2 股权转让
- 优先转让给其他股东（按持股比例分配）
- 其他股东放弃购买的，可对外转让
- 对外转让需其他股东同意（或放弃优先购买权）

7.1.3 转让价格
- 协商确定
- 协商不成的，委托第三方评估
```

### 7.2 强制退出（除名）

```
7.2.1 除名条件
一方有以下行为，经其他股东 ___% 同意可除名：
- 严重违反本协议义务
- 严重失职给公司造成损失 >___万元
- 刑事犯罪
- 丧失劳动能力超过 ___ 个月

7.2.2 除名后果
- 全部股权按成本价回购
- 已领取的薪酬、报销不需退还
- 知识产权归公司所有
```

### 7.3 公司解散

```
7.3.1 解散条件
- 股东会决议解散
- 被依法吊销营业执照
- 破产

7.3.2 清算程序
- 成立清算组
- 清偿债务
- 剩余财产按股权比例分配

7.3.3 知识产权处置
公司解散后，知识产权按以下方式处理：
- 转让：转让收入按股权比例分配
- 分配：各方按股权比例共有
- 放弃：经全体股东同意放弃
```

## 八、争议解决

### 8.1 协商

```
各方因本协议发生争议，首先通过友好协商解决，
协商期为 ___ 日。
```

### 8.2 调解/仲裁/诉讼

```
协商不成的，选择以下方式之一：

选项 A（调解）：
提交 ______ 调解中心调解

选项 B（仲裁）：
提交 ______ 仲裁委员会仲裁，
仲裁地为 ______，适用中华人民共和国法律

选项 C（诉讼）：
向公司注册地有管辖权的人民法院提起诉讼
```

**建议硬件创业选择**：诉讼（便于财产保全和执行）

## 九、其他条款

### 9.1 协议变更

```
本协议的修改、补充，需经全体股东书面同意。
```

### 9.2 协议生效

```
本协议自各方签字（或盖章）之日起生效。
本协议一式 ___ 份，各方各执一份，公司留存一份。
```

### 9.3 完整协议

```
本协议构成各方就合伙事宜的完整协议，
取代此前所有口头或书面约定。
```

### 9.4 可分割性

```
本协议任何条款被认定为无效，不影响其他条款的效力。
```
