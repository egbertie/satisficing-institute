# 股权分配谈判指南

> 针对硬件初创企业的股权分配实战手册  
> 目标：保护创始人控制权，同时激励合伙人长期投入

---

## 一、股权分配基本原则

### 1.1 黄金比例参考

| 角色 | 建议股权区间 | 适用场景 |
|------|-------------|---------|
| **唯一创始人** | 80-100% | 早期未引入合伙人 |
| **双创始人（技术+商业）** | 60:40 或 55:45 | 双方全职，能力互补 |
| **三创始人** | 50:30:20 或 40:35:25 | 技术+商业+运营 |
| **创始人+核心合伙人** | 创始人 51-70%，合伙人 20-40% | 创始人主导，合伙人补充 |

### 1.2 硬件行业特殊考量

**技术权重较高的原因**：
- 硬件产品技术门槛高，CTO 往往是核心竞争力
- 供应链、生产、品控都需要技术背景
- 但注意：技术合伙人不宜超过 40%，否则失去控制权

**政府补贴的影响**：
- 补贴到账后，公司估值上升，股权稀释需提前规划
- 建议补贴到账后再做新一轮股权分配调整

## 二、谈判前准备

### 2.1 明确你的底线

**写下这三个数字**：
1. **理想股权**（你最希望的比例）：如 60%
2. **可接受股权**（你能接受的最低比例）：如 51%
3. **退出红线**（低于这个比例就不干）：如 45%

**硬件创业的特殊底线**：
- 如果你负责融资和对外，建议保持 51% 以上
- 如果技术合伙人负责核心技术，可以接受 40-45%
- 期权池至少预留 10-15%（用于未来团队激励）

### 2.2 了解对方诉求

**对方要股权，可能是想要**：
- 钱（经济利益）
- 权（决策权/控制权）
- 名（联合创始人头衔）
- 安全感（不被踢出局）

**如何探询**：
> 「如果股权降到 30%，但给你技术决策否决权，你接受吗？」  
> → 如果对方犹豫，说明他要的是「权」不是「钱」  
> → 如果对方拒绝，说明他要的是「钱」或「名」

### 2.3 准备替代方案

**不要只谈股权数字，准备多种组合**：

| 方案 | 股权 | 其他条件 | 适用场景 |
|------|------|---------|---------|
| A | 40% | 联合创始人头衔，技术决策权 | 技术合伙人能力强，但有控制权风险 |
| B | 30% | 25%股权+5%期权（里程碑解锁） | 激励长期投入 |
| C | 25% | 25%股权+高薪资+技术VP头衔 | 对方更看重稳定收入 |
| D | 35% | 股权+表决权分离（你保留51%表决权） | 股权让步但保留控制 |

## 三、谈判话术与技巧

### 3.1 开场定调

**错误开场**：
> ❌ 「我觉得你应该拿 30%」

**正确开场**：
> ✅ 「我们先不聊数字，聊聊你对公司的期待。3年后，你希望在这个公司扮演什么角色？」

**目的**：先了解对方诉求，再谈数字

### 3.2 应对「我要50%」

**第一步：拆解诉求**
> 「50%没问题，但我想理解一下，这个数字对你意味着什么？是觉得贡献值一半，还是担心以后被稀释？」

**第二步：提出替代**
> 「如果给你 35% 股权，但技术上你完全说了算，重大技术决策我不插手，你觉得怎么样？」

**第三步：设置里程碑**
> 「这样，我们先定 30%，如果明年公司收入达到 500万，再给你追加 5%，这样公平吗？」

### 3.3 保护自己控制权的表达

**直接但礼貌**：
> 「我需要保持 51% 的控制权，这不是为了控制你，而是为了让投资人、供应商、客户知道这家公司有主心骨。如果咱俩各50%，外面的人会担心我们内斗。」

**用数据说话**：
> 「我看了行业数据，股权平分（50:50）的创业公司，80%最后因为决策僵局死掉。我们要避免这个陷阱。」

**用期权池转移注意力**：
> 「现在公司估值低，给你 30% 其实没多少钱。不如这样，你先拿 25%，等下一轮融资估值上去了，我们从期权池里给你补，那时候 5% 可能值现在的 15%。」

### 3.4 最后通牒的处理

**对方说：不给XX%我就走**

**不要立即答应或拒绝**：
> 「我理解你的立场，这确实是个大事。给我 24 小时想想，明天我们再聊。这期间你也可以想想，有没有其他方式能让你感到安全，除了股权数字。」

**紧急替代方案**（如果对方真的要撤）：
> 「这样吧，我们先签一个临时协议：你拿 40%，但 6 个月后根据业绩调整到 30-35%。这样你有安全感，我也能控制风险。」

## 四、关键条款设计

### 4.1 股权兑现（Vesting）

**标准条款**：
```
创始合伙人股权分 4 年兑现，满 1 年兑现 25%，
之后按月兑现（1/48 每月）。

若合伙人在兑现完成前离职：
- 已兑现部分：保留
- 未兑现部分：公司按成本价回购
```

**硬件行业建议**：
- 技术合伙人 vesting 期可以延长到 5 年（产品周期长）
- 前 6 个月可以设置「试用期」，试用期离职全部回购
- 产品量产上市后，加速兑现（比如一次性兑现剩余 50%）

### 4.2 退出机制

**三种退出场景**：

| 场景 | 处理方式 | 回购价格 |
|------|---------|---------|
| 主动离职（好聚好散） | 未兑现股权公司回购 | 成本价或公允价较低者 |
| 被开除（严重失职） | 全部股权公司回购 | 成本价 |
| 公司被收购 | 按比例变现 | 按收购价 |

**防止恶意退出**：
```
若合伙人离职后 2 年内从事竞争业务，
已兑现股权中的 50% 自动失效。
```

### 4.3 表决权设计

**股权与表决权分离**：
```
合伙人 A：股权 40%，表决权 30%
合伙人 B：股权 35%，表决权 55%
（差额通过表决权委托协议实现）
```

**适用场景**：
- 技术合伙人股权高，但你不希望他有同等控制权
- 投资人进入后，你想保留控制权

**硬件行业特殊条款**：
```
以下事项需技术合伙人同意：
- 核心技术路线变更
- CTO 人选任命/罢免
- 重大技术合作（授权/转让）

以下事项需 CEO 同意：
- 融资条款
- 公司出售/并购
- 年度预算 > 100万
```

## 五、常见陷阱与应对

### 陷阱 1：股权平分（50:50）

**风险**：决策僵局，公司内耗

**案例**：某硬件创业公司两个创始人各 50%，为产品定价吵了 3 个月，错过窗口期。

**应对**：
- 坚决不接受 50:50
- 如果一定要平等，引入第三方（独立董事/顾问）作为 tie-breaker

### 陷阱 2：口头承诺

**风险**：对方说「我们口头说好了」，但后期翻脸

**案例**：技术合伙人承诺「给我 30% 我就全职加入」，入职后说「我记得是 40%」。

**应对**：
- 任何股权承诺必须有书面文件
- 入职前签订「创始人股权协议」（Founder Agreement）

### 陷阱 3：没有 Vesting

**风险**：合伙人拿了股权就躺平或离职

**案例**：合伙人拿 30% 股权后 3 个月离职，股权还在，后面融资被稀释但仍有投票权。

**应对**：
- 必须有 4 年 vesting
- 离职时未兑现部分按成本价回购

### 陷阱 4：忽略期权池

**风险**：股权分完，后面招不到人

**案例**：两个创始人分了 90%，只剩 10% 给团队，招聘时拿不出期权。

**应对**：
- 早期预留 15-20% 期权池
- 股权分配时把期权池算进去：创始人 A 40% + 创始人 B 35% + 期权池 15% = 90%，留 10% 机动

### 陷阱 5：政府补贴导致的股权纠纷

**风险**：补贴到账后，合伙人认为公司估值上升，要求重新谈股权

**案例**：获得 300万补贴后，技术合伙人要求从 30% 提高到 40%，理由是「公司值钱了」。

**应对**：
- 股权协议中明确：政府补贴不影响股权比例
- 或者约定：补贴到账后，双方按原比例增资，保持股权不变

## 六、谈判 Checklist

### 谈判前
- [ ] 明确自己的底线股权比例
- [ ] 了解对方的核心诉求（钱/权/名/安全感）
- [ ] 准备 2-3 个替代方案
- [ ] 咨询律师了解法律风险

### 谈判中
- [ ] 先聊愿景和角色，再谈数字
- [ ] 记录对方的每一个诉求
- [ ] 不立即答应或拒绝，给自己思考时间
- [ ] 确认双方对「股权」的理解一致（股权=所有权+表决权+分红权）

### 谈判后
- [ ] 48 小时内发送确认邮件，总结共识
- [ ] 1 周内签订书面协议
- [ ] 协议包含：vesting、退出机制、表决权设计
- [ ] 找律师审核协议

## 八、政府补贴相关条款

### 补贴到账后的股权保护

```
特别条款：政府补贴不影响股权结构

1. 公司获得的政府补贴、奖励、退税等，
   不改变现有股东的股权比例。

2. 若补贴要求股东同比例增资，
   各方应按现有股权比例履行出资义务。

3. 若一方无法履行出资义务，
   另一方可代为出资，但股权比例不变，
   代出资金额作为借款处理。
```

### 补贴申请中的合伙人责任

```
特别条款：补贴申请责任分工

1. 创始人 A 负责政府关系对接和补贴申请；
2. 创始人 B 负责技术材料的准备和答辩；
3. 补贴到账后，双方共同决定资金用途，
   单笔支出 >___万需双方同意。
```
