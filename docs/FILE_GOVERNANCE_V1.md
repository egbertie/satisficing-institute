## 一、文件结构标准

### 1.1 命名规范
| 类型 | 格式 | 示例 |
|------|------|------|
| 日期文件 | `YYYY-MM-DD_描述.md` | `2026-03-20_承诺洗澡.md` |
| 版本文件 | `名称_V版本号_描述.md` | `DISASTER_RECOVERY_V2.md` |
| 报告文件 | `报告类型_日期.md` | `SYSTEMATIC_ISSUE_DEEP_DIVE.md` |
| 脚本文件 | `功能描述.sh/py` | `knowledge-extractor.sh` |
