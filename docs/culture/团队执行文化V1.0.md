## 三条铁律

### 1. 时间预估必须激进
**标准：** 原计划的 50%

- 如果任务看似需要 2 小时 → 预估 1 小时
- 如果项目看似需要 1 周 → 预估 3.5 天
- 打破"舒适区"预估，倒逼极限效率

**执行要点：**
- 不接受"时间不够"借口
- 砍掉冗余环节，直奔核心
- 先完成，再完美

---

### 2. 相信极限潜能
**语言切换：** 删除"可能/大概/应该/试试"，只用"一定/必须/完成/交付"

| ❌ 禁用表达 | ✅ 强制表达 |
|------------|------------|
| "我试试看" | "我一定完成" |
| "可能可以" | "一定可以" |
| "大概明天" | "明天交付" |
| "应该没问题" | "保证完成" |

**心理锚点：** 不设退路，只有前行

---

### 3. 完成即启动
**状态切换：** 零停顿、零等待、永向前

- **任务 A 完成** → 立即启动任务 B（不庆祝、不休息）
- **项目交付** → 立即复盘并启动下一个（不停留）
- **目标达成** → 立即设定更高目标（不满足）

**执行节奏：**
```
执行 → 完成 → 立即启动 → 执行 → ...无限循环
```
