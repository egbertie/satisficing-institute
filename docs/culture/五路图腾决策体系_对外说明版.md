## 一、什么是五路图腾决策体系？

五路图腾决策体系是满意解研究所独创的合伙人选择方法论。

它将五个维度的智慧融合为一个完整的决策框架：
- **价值维度**：什么最重要？
- **方法维度**：如何做出选择？
- **状态维度**：在什么状态下做决策？
- **伦理维度**：这个选择是否负责任？
- **直觉维度**：身体告诉你什么？

五路齐备，才能在复杂中找到方向。

## 二、五路图腾详解

### 【第一路】LIU · 惟吾德馨

**源自**：唐代诗人刘禹锡《陋室铭》

**核心精神**：
> "斯是陋室，惟吾德馨。"——简陋的环境不影响内在的价值。

**在决策中的意义**：
- 剥离表象，直击核心价值
- 不因外在光环（大厂背景、名校学历）而迷失
- 关注那些不随股价波动、不随风口转移的本质

**Egbertie的诠释**：
> "在合伙人选择中，最重要的不是对方拥有什么，而是对方是谁。"

---

### 【第二路】SIMON · 满意解

**源自**：诺贝尔经济学奖得主赫伯特·西蒙（Herbert A. Simon）

**核心精神**：
> "不求最优，只求满意。"——在有限信息和有限理性下，追求足够好。

**在决策中的意义**：
- 接受"完美合伙人不存在"的现实
- 设定"满意解标准"，避免过度优化
- 在时限约束下做出最好决策

**Egbertie的诠释**：
> "80分的合伙人，比永远找不到的100分合伙人更有价值。"

---

### 【第三路】GUANYIN · 自在从容

**源自**：观自在菩萨《心经》

**核心精神**：
> "观自在，行深般若波罗蜜多时。"——在觉察中获得自在解脱。

**在决策中的意义**：
- 焦虑会让决策变形，好决策在放松中诞生
- 创造安全的决策空间，不被时间压力裹挟
- 教练状态：不紧绷、不强迫、不急于给答案

**Egbertie的诠释**：
> "当你紧张时，你看到的是恐惧；当你放松时，你看到的是真相。"

---

### 【第四路】CONFUCIUS · 仁者爱人

**源自**：孔子《论语》

**核心精神**：
> "己欲立而立人，己欲达而达人。"——互相成就，共同成长。

**在决策中的意义**：
- 合伙不是签约就结束，而是关系的开始
- 关注对方是否有"共赢心态"而非"零和博弈"
- 长期关系的伦理基础：义利合一

**Egbertie的诠释**：
> "最好的合伙关系，是让双方都成为更好的自己。"

---

### 【第五路】HUINENG · 顿悟直觉

**源自**：六祖慧能《无相颂》

**核心精神**：
> "若能钻木取火，淤泥定生红莲。"——在极限中洞见，在复杂中觉醒。

**在决策中的意义**：
- 数据排除错误选项，直觉指向正确方向
- 身体是直觉的容器，信任身体的信号
- 在信息不完备时，敢于凭借洞见决策

**Egbertie的诠释**：
> "理性的尽头，是直觉的开始。"

## 三、五路如何协同？

```
                    ┌─────────────────┐
                    │   你的决策      │
                    └────────┬────────┘
                             │
              ┌──────────────┼──────────────┐
              │              │              │
              ▼              ▼              ▼
        ┌─────────┐    ┌─────────┐    ┌─────────┐
        │  LIU    │◄───│HUINENG  │───►│  SIMON  │
        │  价值   │    │  直觉   │    │  方法   │
        └────┬────┘    └────┬────┘    └────┬────┘
             │              │              │
             │         ┌────┴────┐         │
             │         │ GUANYIN │         │
             │         │  状态   │         │
             │         └────┬────┘         │
             │              │              │
             └──────────────┼──────────────┘
                            │
                       ┌────┴────┐
                       │CONFUCIUS│
                       │  伦理   │
                       └─────────┘
```

**协同逻辑**：
1. **LIU（价值）** 确定方向——什么最重要
2. **SIMON（方法）** 提供框架——如何系统分析
3. **GUANYIN（状态）** 创造条件——在放松中决策
4. **CONFUCIUS（伦理）** 把关长期——关系是否可持续
5. **HUINENG（直觉）** 最终洞见——身体告诉你答案

## 四、五路图腾咨询流程

### 标准会程：90分钟

| 阶段 | 时长 | 核心内容 | 主导图腾 |
|------|------|----------|----------|
| **根基建立** | 30分钟 | 价值澄清、满意解标准设定 | LIU + SIMON |
| **辩证洞察** | 30分钟 | 信息分析、直觉唤醒 | SIMON + GUANYIN + HUINENG |
| **决策定格** | 30分钟 | 伦理审视、五路校准、行动规划 | CONFUCIUS + 五路综合 |

### 核心特色

**特色1：价值先行**
- 不急于分析人选，先澄清"什么最重要"
- 避免被表象（光环、资源）迷惑

**特色2：双脑并用**
- 左脑：理性分析、决策矩阵
- 右脑：直觉唤醒、身体扫描

**特色3：伦理把关**
- 不只问"有利吗"，更问"有义吗"
- 关注长期关系健康，而非短期利益

**特色4：仪式闭环**
- 决策书写、五路祝福、印记封存
- 让决策从"想法"变成"承诺"

## 五、五路图腾校准清单

在做出最终决策前，用五路图腾逐一校准：

- [ ] **LIU（价值）**：这个选择是否符合我最初的核心价值？
- [ ] **SIMON（方法）**：这是"足够好"的选择吗？我有没有过度优化？
- [ ] **GUANYIN（状态）**：我在做决定时的状态是放松的吗？还是被焦虑驱使的？
- [ ] **CONFUCIUS（伦理）**：这段关系是否能让双方都成长？还是一方的牺牲？
- [ ] **HUINENG（直觉）**：我的身体对这个选择感觉舒展吗？还是紧绷的？

**通过标准**：
- 5项全部通过 → 决策确认
- 4项通过 → 讨论未通过项，调整后确认
- ≤3项通过 → 建议暂缓，补充信息后再议

## 六、常见问题

**Q：五路图腾和普通的决策咨询有什么区别？**

A：普通咨询往往只关注"分析"（SIMON金象），五路图腾则整合了五个维度：
- 价值（避免迷失）
- 方法（系统分析）
- 状态（放松决策）
- 伦理（长期负责）
- 直觉（身体智慧）

这是一个更完整、更人性化的决策框架。

---

**Q：我没有候选人，可以用五路图腾吗？**

A：可以。五路图腾不仅用于"选择A还是B"，也可用于：
- 澄清"我究竟想要什么样的合伙人"
- 评估"我现在的合伙关系健康吗"
- 诊断"为什么我之前的合伙失败了"

---

**Q：五路图腾有科学依据吗？**

A：五路图腾融合了多个学科的研究成果：
- **SIMON（满意解）**：基于诺贝尔经济学奖的有限理性理论
- **GUANYIN（状态）**：基于神经科学的情绪调节研究
- **HUINENG（直觉）**：基于躯体标记假说（Somatic Marker Hypothesis）
- **CONFUCIUS（伦理）**：基于组织行为学的信任研究

同时，它也吸收了东方智慧（儒家、禅宗、道家），是中西融合的决策方法论。

---

**Q：一次咨询能解决问题吗？**

A：取决于问题的复杂程度：
- 简单决策（二选一）→ 一次90分钟咨询通常足够
- 复杂决策（多方权衡）→ 可能需要2-3次咨询
- 诊断性问题（为什么之前失败）→ 一次咨询可完成诊断

我们也会在咨询后提供跟进支持。
