# /话题选择 使用指南

## 什么是 /话题选择？

`/话题选择` 是Discord频道的频道管理工具，用于在频道内创建**独立的话题线程**。当你想讨论某个具体主题，又不想打乱主频道的对话流时，就可以使用这个命令创建一个结构化的话题分支。每个话题有独立的聊天记录，方便后续查找和归档。

---

## 如何使用

1. 在频道输入框输入 `/话题选择`
2. 输入话题名称（建议简明扼要）
3. 选择话题类型（文字/论坛帖子）
4. 点击创建，自动进入话题线程
