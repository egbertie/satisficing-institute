## 使用指南

### 快速开始

```python
from core.entanglement_system import (
    EntanglementMatchingSystem,
    PartnerProfile
)

# 初始化系统
system = EntanglementMatchingSystem()

# 创建合伙人档案
partner_a = PartnerProfile(
    name="李明",
    id="p001",
    cognitive_profile={
        'analytical': 0.7,
        'detail_oriented': 0.8,
        'risk_tolerance': 0.6,
        'independent': 0.7
    }
)

partner_b = PartnerProfile(
    name="王芳",
    id="p002",
    cognitive_profile={
        'analytical': 0.4,
        'detail_oriented': 0.4,
        'risk_tolerance': 0.7,
        'independent': 0.6
    }
)

# 执行匹配
result = system.comprehensive_match(partner_a, partner_b)

# 查看结果
print(f"综合兼容度: {result['overall_compatibility']:.1%}")
print(f"主导协作模式: {result['layer_results']['emergent_matching']['dominant_mode']}")
```

### 运行演示

```bash
cd partner-match-system

```
