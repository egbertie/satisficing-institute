## 二、核心模块详解

### 2.1 L1: BitNet 1.58-bit 推理引擎

**核心原理**:
- 三值量化: {-1, 0, +1}
- 2-bit编码: 00=-1, 01=0, 10=+1
- 无乘法运算: 矩阵乘法简化为加减法

**性能指标**:
```
模型规模: 512×256 = 131,072 weights
压缩前: 512KB (FP32)
压缩后: 32KB (2-bit)
压缩比: 16×
推理延迟: 0.62 ms
吞吐量: 1,611 inferences/sec
```

**应用场景**:
- 边缘设备部署
- 长记忆压缩存储
- 低功耗推理

**代码**: `bitnet_inference.cpp`

---

### 2.2 L3: io_uring + ZeroPool I/O加速

**核心原理**:
- Linux io_uring异步I/O（队列深度256）
- ZeroPool预分配内存池（512MB），消除运行时malloc
- 批量提交（64KB chunks）
- 零拷贝mmap

**性能指标**:
```
测试文件: 100MB
传统同步I/O: ~200ms
io_uring+ZeroPool: 9ms
加速比: 22.2×
吞吐量: 11,111 MB/s (11 GB/s)
```

**应用场景**:
- Checkpoint快速加载
- 大模型权重热加载
- Agent状态持久化

**代码**: `zero_pool_loader/src/main.cpp`
