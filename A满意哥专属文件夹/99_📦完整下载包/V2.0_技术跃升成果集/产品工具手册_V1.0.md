## 第二章：五路图腾决策校准（作业指引）

### 2.1 校准流程总览

五路图腾对应五个决策维度，形成完整的决策支持体系：

```
        SIMON(金) · CONFUCIUS(木)
       GUANYIN(水) · HUINENG(火)
              LIU(土)
```

**相生关系**: 土 → 金 → 水 → 木 → 火 → 土

### 2.2 LIU校准（土）——价值纯度与心性修炼

**核心问题**: 这个合伙人是否符合我的核心价值观？

**操作步骤**:

1. **价值观对齐访谈** (30分钟)
   - 列出创始人核心价值观（3-5条）
   - 设计价值观冲突场景提问
   - 记录候选人反应与价值观匹配度

2. **心性修炼评估表**

| 评估维度 | 低(1-3) | 中(4-6) | 高(7-10) | 评分 | 证据 |
|----------|---------|---------|----------|------|------|
| 长期主义 | 只看短期利益 | 兼顾短长期 | 专注长期价值 | | |
| 抗压能力 | 易崩溃 | 能应对 | 压力下更清醒 | | |
| 学习能力 | 固步自封 | 被动学习 | 主动迭代 | | |
| 诚信度 | 有明显瑕疵 | 基本可信 | 高度可信 | | |

3. **关键判断**: 如果合伙人在核心利益冲突时的选择与我价值观相悖 → **No-Go**

### 2.3 SIMON校准（金）——理性决策框架

**核心问题**: 如何在信息不全时做出可辩护的决策？

**操作步骤**:

1. **满意标准定义**
   - 与创始人共同定义"足够好"的标准
   - 必须量化的指标（如：融资能力、行业经验年限）
   - 可妥协的维度（如：年龄、毕业院校）

2. **PFI指数计算**

```
PFI (Partner Fit Index) = Σ(维度权重 × 维度得分) / 总权重

维度设置（可根据案例调整）:
┌──────────────┬────────┬─────────────────────────────┐
│ 维度         │ 权重   │ 评分标准                     │
├──────────────┼────────┼─────────────────────────────┤
│ 能力互补性   │ 25%    │ 1-10分：技术/商业/运营互补程度│
│ 价值观契合   │ 25%    │ 1-10分：核心价值观匹配度     │
│ 压力测试表现 │ 20%    │ 1-10分：72小时极限评估表现   │
│ 承诺可信度   │ 15%    │ 1-10分：过往承诺兑现率       │
│ 成长潜力     │ 15%    │ 1-10分：学习能力与迭代速度   │
└──────────────┴────────┴─────────────────────────────┘

PFI ≥ 7.5: 强烈推荐
PFI 6.0-7.4: 条件推荐（需明确风险点）
PFI < 6.0: 不推荐
```

3. **风险热力图绘制**
   - X轴：风险发生概率
   - Y轴：风险影响程度
   - 标注：价值观冲突、能力缺口、股权分歧、退出机制

### 2.4 GUANYIN校准（水）——压力管理与状态调节

**核心问题**: 在高压场景下，创始人能否保持清醒判断？

**操作步骤**:

1. **压力场景模拟** (贯穿72小时)
   - 模拟融资谈判破裂场景
   - 模拟技术路线分歧场景
   - 模拟股权争议场景

2. **状态观察清单**

| 观察点 | 负面信号 | 正面信号 |
|--------|----------|----------|
| 情绪反应 | 易怒/逃避 | 冷静/直面 |
| 决策速度 | 冲动/拖延 | 审慎/及时 |
| 沟通方式 | 攻击性/沉默 | 建设性/开放 |
| 身体语言 | 紧张/回避 | 放松/直视 |

3. **调节能力评估**
   - 候选人是否有自我调节的方法？
   - 从压力中恢复需要多长时间？

### 2.5 CONFUCIUS校准（木）——感知力与直觉训练

**核心问题**: 如何训练决策直觉？如何在商业中践行儒家伦理？

**操作步骤**:

1. **感知力训练设计**
   - 微表情识别测试
   - 非语言信号观察练习
   - "直觉-验证"循环训练

2. **儒商伦理评估**

| 儒家原则 | 评估问题 | 候选人表现 |
|----------|----------|------------|
| 仁（关怀）| 如何对待团队成员？ | |
| 义（正义）| 面对利益冲突如何选择？ | |
| 礼（尊重）| 沟通中的尊重程度？ | |
| 智（智慧）| 问题解决能力？ | |
| 信（诚信）| 承诺兑现记录？ | |

3. **直觉验证机制**
   - 记录第一印象（第1分钟）
   - 深度访谈后对比
   - 验证直觉的准确率

### 2.6 HUINENG校准（火）——极限压力测试与重生

**核心问题**: 通过极限测试，验证合伙人的真实状态

**操作步骤**:

1. **72小时极限测试设计**

| 阶段 | 时间 | 测试内容 | 观察重点 |
|------|------|----------|----------|
| 高压连续 | 24h | 高强度工作+信息轰炸 | 抗压能力、专注力 |
| 意外中断 | 8h | 突然暂停，观察恢复 | 调节能力、心态 |
| 危机模拟 | 16h | 最坏情况推演 | 应变能力、价值观 |
| 深度对话 | 24h | 连续深度访谈 | 真实自我暴露 |

2. **顿悟时刻识别**
   - 候选人在哪个时刻表现出真实的自我？
   - 什么情况下会突破防御机制？

3. **可控失败设计**
   - 故意设置一个小失败场景
   - 观察候选人如何面对失败
   - 从失败中学习的能力

### 3.2 数字替身使用指南

#### 黎红雷教授（拟邀）（儒商顾问）

**使用场景**: 伦理对齐、文化框架、价值观评估

**提问模板**:
```
"创始人提出[具体情境]，候选人的选择是否符合儒商伦理？
请从仁义礼智信五个维度分析。"
```

**输出要求**: 伦理评分 + 文化适配建议

#### 罗汉教授（拟邀）（数学护法）

**使用场景**: 模型验证、算法优化、风险评估

**提问模板**:
```
"当前PFI计算模型如下[模型描述]，是否存在统计偏差？
建议如何优化权重设置？"
```

**输出要求**: 数学验证 + 模型优化建议

#### 谢宝剑研究员（拟邀）（地理自在官）

**使用场景**: 地域战略、政策解读、深港资源对接

**提问模板**:
```
"创始人项目[项目名称]位于[地区]，候选人[背景]，
在深港战略框架下的匹配度如何？"
```

**输出要求**: 地域适配分析 + 资源对接建议

#### 方翊沣博士（拟邀）（感知力导师）

**使用场景**: CEO感知力训练体系设计

**提问模板**:
```
"基于[候选人评估数据]，请设计感知力训练方案，
帮助创始人提升合伙人选择的直觉准确率。"
```

**输出要求**: 训练方案 + 效果评估方法

#### 陈国祥博士（拟邀）（能量导师）

**使用场景**: 催眠引导、能量调节（暂定）

**提问模板**:
```
"创始人当前状态[描述]，候选人能量场[描述]，
两者能量匹配度及调节建议？"
```

**输出要求**: 能量评估 + 调节方案

#### 李泽湘教授（拟邀）（行业导师）

**使用场景**: 硬件科技创业前沿引导、行业资源对接

**提问模板**:
```
"硬件科技项目[描述]，候选人[背景]，
在李泽湘体系下的创业成功率评估？"
```

**输出要求**: 行业适配度 + 资源对接建议

## 第五章：不满意退款机制

### 5.1 退款条件

**全额退款条件**（满足任一）:
1. 6个月内合伙人关系破裂且原因在评估时已存在
2. PFI指数预测准确率低于70%
3. 重大风险提示未被识别

**部分退款条件**:
- 根据服务完成度协商

### 5.2 退款流程

1. 创始人提出退款申请（书面）
2. 7个工作日内完成原因调查
3. 15个工作日内完成退款
