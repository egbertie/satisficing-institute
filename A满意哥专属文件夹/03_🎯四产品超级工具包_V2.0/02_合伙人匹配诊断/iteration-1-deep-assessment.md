### 1.3 潜意识测试：内隐联想测试(IAT)

**测试设计原理：**
采用Greenwald(1998)经典IAT范式，测量合伙人对核心概念的潜意识关联强度。

| 测试维度 | 目标概念对 | 属性词对 | 测量目标 |
|---------|-----------|---------|---------|
| **权力态度** | 我/合伙人 | 领导/服从 | 权力分配潜意识偏好 |
| **风险偏好** | 冒险/保守 | 积极/消极 | 决策风格潜意识倾向 |
| **信任模式** | 信任/怀疑 | 安全/危险 | 信任建立机制 |
| **冲突态度** | 冲突/和谐 | 必要/有害 | 冲突处理方式偏好 |
| **成功归因** | 能力/运气 | 内部/外部 | 归因风格匹配度 |

**操作流程：**
```
Phase 1: 基线练习 (20 trials)
Phase 2: 相容任务测量 (40 trials) - 典型配对
Phase 3: 反转练习 (20 trials)
Phase 4: 不相容任务测量 (40 trials) - 反转配对
Phase 5: 数据清洗与D分数计算
```

**结果解读框架：**
- D分数 > 0.65：强烈内隐偏好，需关注显隐不一致
- D分数 0.35-0.65：中等偏好，存在调节空间
- D分数 < 0.35：偏好较弱，内隐态度开放

### 1.4 生理压力监测：心率变异性(HRV)

**HRV指标体系：**

| 指标 | 含义 | 合伙匹配解读 |
|-----|------|-------------|
| **SDNN** | 全部窦性心搏R-R间期标准差 | 整体自主神经张力 |
| **RMSSD** | 相邻R-R间期差值均方根 | 副交感神经活性（恢复力） |
| **LF/HF比值** | 低频/高频功率比 | 交感-副交感平衡状态 |
| **pNN50** | 相邻R-R差>50ms占比 | 心率变异性健康度 |

**压力反应同步性分析：**
```python
# 伪代码：压力同步指数计算
def calculate_stress_sync(partner_a_hrv, partner_b_hrv, task_timeline):

    高同步 = 共同承压能力强
    低同步 = 压力传导存在障碍
    """
    a_stress_phases = extract_stress_phases(partner_a_hrv)
```
