## 🎨 设计规格

### 版本A：横向时间线/流程图
**用途**：演示文稿、网页横幅、流程说明  
**尺寸**：1200×400px（横向3:1）  
**特点**：
- 清晰展示五行相生循环：土→金→水→木→火→土
- 适合横向滚动展示
- 带循环回连箭头

### 版本B：五边形/星形布局
**用途**：海报、宣传册、品牌展示  
**尺寸**：800×800px（正方形1:1）  
**特点**：
- 中心聚焦：满意解研究所
- 五路图腾围绕中心呈五角星布局
- 相生关系标注于外圈

## 📖 使用指南

### Mermaid渲染
在支持Mermaid的平台（GitHub、Notion、Obsidian等）直接粘贴`.mmd`文件内容：

```markdown
```mermaid
[粘贴mmd内容]
```
```

### SVG使用
- **网页**：直接嵌入`<img src="totem_timeline.svg">`
- **印刷**：矢量无损，可任意缩放
- **编辑**：使用Figma/Illustrator/Sketch打开

### PNG使用
- **社交媒体**：直接上传使用
- **文档插入**：Word/PPT/PDF
- **预览展示**：快速查看效果

## 🛠️ 技术说明

### SVG生成方式
- 手写SVG代码，确保语义清晰
- 使用CSS渐变增强视觉层次
- 添加阴影滤镜提升立体感

### 可编辑源文件建议
如需深度编辑，建议使用以下工具：

| 工具 | 用途 | 导出格式 |
|------|------|----------|
| **Figma** | 界面设计、协作 | SVG, PNG, PDF |
| **Adobe Illustrator** | 专业矢量编辑 | AI, SVG, EPS |
| **Sketch** | Mac端设计 | Sketch, SVG |
| **Excalidraw** | 手绘风格 | SVG, PNG |

### 导入Figma步骤
1. 打开Figma，新建文件
2. 拖拽SVG文件到画布
3. 右键 → "Outline Stroke" 转曲线
4. 调整字体为思源宋体/黑体
5. 导出所需格式
