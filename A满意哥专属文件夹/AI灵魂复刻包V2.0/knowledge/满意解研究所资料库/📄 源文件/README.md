## 🎭 角色体系

| 角色 | 类型 | 定位 | 核心职责 |
|------|------|------|----------|
| **创始人/CEO** | 人类 | 核心决策者 | 战略决策、一线交付、品牌建设 |
| **项目进化官（PEO）** | AI | 战略中枢 | 版本管理、战略校准、内容审核 |
| **经验萃取官（EEO）** | AI | 知识转化 | 经验萃取、方法论提炼、案例库 |
| **虚拟专家团** | AI暂代 | 外部智库 | 方法论、行业、法律、命理（后续转真人） |

**人员配置**：人类1人 + AI角色2个（同一Kimi）+ 虚拟专家4个（AI暂代）

**协作流程**：
```
创始人（人类）
    ↓
AI支持角色（PEO/EEO）—— 分别激活
    ↓
虚拟专家团（AI暂代→逐步真人化）
```
