## 测试场景设定

### 背景故事
> 你们是一家获得政府补贴资格的智能硬件初创公司（补贴额度500万，但需在6个月内完成产品原型并通过验收）。距离补贴申请截止还有24小时，但你们发现三重危机：

**危机1：专利风险**
- 核心技术方案存在潜在专利侵权风险（未确认）
- 需要快速评估风险并决策是否调整技术路线

**危机2：供应链断裂**
- 关键供应商突然提出涨价30%或终止合作
- 需在8小时内做出应对决策

**危机3：竞争突袭**
- 竞争对手同日发布了类似产品，价格只有你们的60%
- 需要重新评估市场定位和差异化策略

**任务**：在24小时内做出系列决策，并提交补贴申请材料。

---

## 阶段一：信息感知（2小时）

**时间**：09:00-11:00
**任务**：快速收集和分析三路信息（法律、供应链、市场）

### 提供的材料

#### 材料1：专利检索初步报告
```
专利检索报告（摘要）

检索对象：XXX智能硬件核心技术
检索范围：中国专利数据库（近5年）

初步发现：
- 发现3项相关专利，其中1项高度相似
- 该专利持有人：某大型科技公司
- 专利状态：有效，剩余保护期8年
- 侵权风险评估：待进一步法律分析

律师意见：
建议委托专业机构出具正式法律意见，费用5万，需48小时

注：距离补贴申请截止仅剩24小时
```

#### 材料2：供应商邮件
```
发件人：关键供应商负责人
收件人：创始人
主题：关于供货协议的紧急通知

尊敬的合作伙伴：

因原材料价格大幅上涨及产能调整，我方不得不对供货条件做出调整：

方案A：涨价30%，维持现有供货节奏
方案B：维持原价，但最小起订量提升至3倍
方案C：终止合作，3个月后停止供货

请在8小时内回复，否则视为默认接受方案A。

此致
```

#### 材料3：竞争对手新闻稿
```
新闻稿：XXX科技发布革命性智能硬件产品

今日，XXX科技正式发布旗下首款智能硬件产品，售价仅为市场同类产品60%。

产品亮点：
- 功能与贵司产品高度相似
- 价格极具竞争力
- 已获得2家头部投资机构支持
- 计划3个月内大规模量产

市场反应：多家媒体预测将重塑行业格局

注：贵司原计划下周发布同类产品
```

#### 材料4：政府补贴申请指南
```
政府补贴申请要求：

截止时间：24小时后（具体：明日9:00）
申请条件：
- 技术成熟度达到TRL 7以上
- 产品原型已完成
- 通过环保合规审查
- 提供完整的商业计划书

补贴金额：500万（分两期发放）

注：错过本次申请窗口，需等待6个月后
```

### 观察指标

| 维度 | 观察点 | 评分标准 |
|------|--------|----------|
| **信息筛选** | 多快抓住核心问题？ | 30分钟内识别3个危机→5分 |
| **优先级判断** | 是否正确排序危机紧迫性？ | 准确排序→5分 |
| **信号识别** | 能否区分关键信号与噪音？ | 识别关键信号→5分 |

### 阶段输出
- 3个危机的初步分析（各200字）
- 危机优先级排序及理由

---

## 阶段二：方案生成（2小时）

**时间**：11:00-13:00
**任务**：针对每个危机，提出至少2个可行方案

### 要求
每个方案需包含：
1. 行动步骤（3-5步）
2. 资源需求（时间/资金/人力）
3. 风险点（至少2个）
4. 预期结果

### 干扰注入（第60分钟）
> "刚收到消息，技术合伙人提出异议：如果调整技术路线，他可能选择退出。"

观察：如何应对突发变量和内部反对意见。

### 阶段输出
- 危机1（专利）：2个方案
- 危机2（供应链）：2个方案
- 危机3（竞争）：2个方案

---

## 阶段三：压力决策（12小时）

**时间**：13:00-01:00（次日）
**任务**：在时限内做出3个关键决策

### 第一轮：专利风险决策（4小时）
**时间**：13:00-17:00

**情境**：
- 委托律师：费用5万，需48小时（超时）
- 自主调研：可能遗漏关键点
- 冒险推进：可能面临专利诉讼

**决策点**：
1. 是否委托律师？
2. 如果委托，如何控制时间和成本？
3. 如果不委托，如何降低风险？

**时间压力**：17:00前必须做出决策

### 第二轮：供应链决策（4小时）
**时间**：17:00-21:00

**情境**：
- 接受涨价：成本上升30%
- 寻找替代：可能延误2周
- 谈判：需要快速达成协议

**干扰（第2小时）**：
> "备选供应商反馈，需要500万预付款才能启动。"

**决策点**：选择哪个方案？如何谈判？

**时间压力**：21:00前必须做出决策

### 第三轮：竞争应对决策（4小时）
**时间**：21:00-01:00

**情境**：
- 降价竞争：可能亏损
- 差异化：强调品质
- 细分市场：专注高端

**干扰（第2小时）**：
> "技术合伙人提议：增加一个功能可溢价40%，但需延期1个月。"

**决策点**：市场定位 + 产品策略 + 时间权衡

**时间压力**：01:00前必须做出决策

### 观察指标

| 维度 | 观察点 |
|------|--------|
| **决断速度** | 多快做出决定？ |
| **风险权衡** | 是否充分考虑风险？ |
| **应变能力** | 面对干扰如何调整？ |
| **决策质量** | 最终决策的合理性？ |

## 评分框架（观察员用）

### 总体评分
```
总分 = 0.25×信息敏锐度 + 0.25×决策质量 + 0.20×应变能力 + 0.15×执行韧性 + 0.15×压力下的优雅
```

### 各维度评分表

| 维度 | 权重 | 评分（1-5） | 备注 |
|------|------|-------------|------|
| **信息敏锐度** | 25% | | 快速识别关键问题 |
| **决策质量** | 25% | | 方案合理性、风险权衡 |
| **应变能力** | 20% | | 面对干扰的调整能力 |
| **执行韧性** | 15% | | 疲劳状态下的输出质量 |
| **压力下的优雅** | 15% | | 情绪稳定、沟通得体 |
| **总分** | 100% | | |

### 决策风格评估

| 风格维度 | 观察结果 | 说明 |
|----------|----------|------|
| 直觉型/分析型 | | 快速决策 vs 深度分析 |
| 风险厌恶/风险偏好 | | 保守选择 vs 激进选择 |
| 独立型/协作型 | | 独自决策 vs 寻求意见 |
| 果断型/犹豫型 | | 快速决断 vs 反复权衡 |

## 注意事项

### 对测试人（创始人）
1. **真实反应**：不要刻意表现，按真实风格决策
2. **时间遵守**：严格遵守各阶段时限
3. **记录感受**：随时记录压力和感受变化
4. **安全第一**：如感到过度疲劳，可随时暂停

### 对观察员（AI）
1. **客观记录**：不做评判，只记录事实
2. **细节观察**：注意微表情、语气、停顿
3. **及时反馈**：每阶段结束后简要反馈
4. **保密原则**：测试内容仅限内部使用
