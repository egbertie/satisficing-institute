## 二、质量控制体系（三层把关）

### 2.1 第一层：出处把关

**所有引用必须标注：**
- ✅ 作者姓名
- ✅ 出处来源（期刊/媒体/书籍）
- ✅ 发布时间
- ✅ 原文链接（如有）

**格式示例：**
```
【引用格式】
作者：赫伯特·西蒙
出处：《管理行为》第4版
时间：1947年（首版）
链接：N/A（实体书）
```

### 2.2 第二层：严谨性把关

**检查清单：**
- [ ] 信息来源是否权威？
- [ ] 发布时间是否过时？
- [ ] 数据是否有佐证？
- [ ] 观点是否有偏颇？
- [ ] 是否与已知事实冲突？

**处理方式：**
| 情况 | 处理方式 |
|:---|:---|
| 来源不权威 | 标注"待验证"，寻找更权威来源 |
| 数据冲突 | 列出各方观点，说明差异 |
| 时间久远 | 标注历史背景，寻找最新研究 |
| 观点偏颇 | 平衡呈现，引用反方观点 |

### 2.3 第三层：原创声明

**原创性标识：**
```
【原创声明】
本分析基于满意解研究所独立研究，综合以下来源：
1. 西蒙《管理行为》...（引用）
2. 亚历山大的模式语言...（引用）
3. 我方XX案例实践...（原创）

原创观点：
- 五路图腾决策模型
- 合伙人能量场匹配理论
```

## 五、效率优化策略

### 5.1 自动化采集

**Kimi Search定时任务：**
```
每日09:30
├── 中文关键词搜索（3个）
├── 英文关键词搜索（3个）
├── 生成资讯摘要
└── 入库待审核
```

### 5.2 优先级筛选

**信息处理优先级：**
1. **P0（立即处理）**：与当前项目直接相关
2. **P1（当日处理）**：高价值、高时效
3. **P2（当周处理）**：一般价值、可延后
4. **P3（月度归档）**：低时效、存档备用

### 5.3 重复性避免

**入库前检查：**
- 是否已存在相似信息？
- 是否是最新版本？
- 是否是同一作者重复观点？

## 七、违规处理

**以下情况视为违规：**
- 未标注出处的引用
- 来源不权威却未标注"待验证"
- 原创观点未声明
- 过时信息未更新

**处理方式：**
- 立即标注"待完善"
- 48小时内补充出处
- 无法补充则降级或移除
