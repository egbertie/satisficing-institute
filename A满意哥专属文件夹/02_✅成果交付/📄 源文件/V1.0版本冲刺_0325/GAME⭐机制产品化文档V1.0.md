## 一、产品概述

### 1.1 产品定位

**产品名称**：满意解决策力训练小程序（GAME⭐版）

**核心定位**：将"合伙人选择"决策方法论转化为可体验、可量化、可传播的游戏化训练产品

**目标用户**：
- **核心用户**：硬科技创业者、准创业者
- **扩展用户**：MBA学生、投资人、创业服务从业者
- **大众用户**：对决策科学感兴趣的知识人群

### 1.2 产品愿景

> 让每一位创业者都能在游戏中体验满意解的力量，在实战中做出不后悔的决策

### 3.2 排行榜算法

#### 3.2.1 积分排名算法

```python
# 基础排名规则
def calculate_ranking(user_id):
    """
    排名计算逻辑
    """
    # 1. 获取用户总积分
    user_score = get_user_total_score(user_id)
    
    # 2. 计算排名
    rank = count_users_with_higher_score(user_score) + 1
    
    # 3. 计算击败百分比
    total_users = get_total_active_users()
    beat_percentage = (total_users - rank) / total_users * 100
    
    return {
        "rank": rank,
        "score": user_score,
        "beat_percentage": round(beat_percentage, 1),
        "tier": get_tier_by_rank(rank, total_users)
    }

# 排名分段
def get_tier_by_rank(rank, total):
    percentile = rank / total * 100
    if percentile <= 1:
        return "🏆 传奇"
    elif percentile <= 5:
        return "💎 钻石"
    elif percentile <= 10:
        return "🥇 黄金"
    elif percentile <= 25:
        return "🥈 白银"
    elif percentile <= 50:
        return "🥉 青铜"
    else:
        return "⭐ 新手"
```

### 6.2 关键数据模型

**用户表 (user)**
```sql
CREATE TABLE user (
    id BIGINT PRIMARY KEY,
    openid VARCHAR(64) UNIQUE,
    nickname VARCHAR(64),
    avatar_url VARCHAR(255),
    level INT DEFAULT 1,
    total_score INT DEFAULT 0,
    current_score INT DEFAULT 0,
    created_at TIMESTAMP,
    updated_at TIMESTAMP
);
```

**积分记录表 (score_log)**
```sql
CREATE TABLE score_log (
    id BIGINT PRIMARY KEY,
    user_id BIGINT,
    action_type VARCHAR(32),
    action_desc VARCHAR(255),
    score_change INT,
    balance INT,
    created_at TIMESTAMP
);
```

**排行榜表 (ranking)**
```sql
CREATE TABLE ranking (
    id BIGINT PRIMARY KEY,
    user_id BIGINT,
    rank_type VARCHAR(32),
    rank_value INT,
    score INT,
    period VARCHAR(32),
    updated_at TIMESTAMP
);
```

**徽章表 (badge)**
```sql
CREATE TABLE user_badge (
    id BIGINT PRIMARY KEY,
    user_id BIGINT,
    badge_code VARCHAR(32),
    badge_type VARCHAR(16),
    acquired_at TIMESTAMP,
    is_showing BOOLEAN DEFAULT FALSE
);
```
