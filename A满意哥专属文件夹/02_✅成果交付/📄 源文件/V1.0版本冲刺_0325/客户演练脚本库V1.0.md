## 剧本A：技术 founder 找商业合伙人

### 客户画像卡

| 项目 | 内容 |
|------|------|
| **姓名** | 李明（化名） |
| **年龄** | 34岁 |
| **背景** | 清华电子系博士，AI芯片方向，8年技术研发经验 |
| **创业阶段** | 技术验证完成，有原型Demo，无营收 |
| **核心痛点** | 有技术但不懂商业，需要找商业合伙人 |
| **历史创伤** | 半年前合作过一个销售合伙人，3个月后分手 |
| **性格特点** | 理性、谨慎、完美主义、对"画饼"敏感 |
| **咨询诉求** | 学习识别靠谱商业合伙人的方法 |

---

### 开场白（扮演者使用）

**版本1（标准版）**：
> "我做了8年AI芯片，技术上我是有信心的，我们的原型已经做出来了，性能比市面上同类产品好30%。但商业这块我确实不懂，之前找了一个做销售的合伙人，合作了3个月发现完全不在一个频道上。他每天都在说认识这个老板那个投资人，但落不了地。
> 
> 我现在很困惑的是，怎么识别一个真正靠谱的商业合伙人？技术人看商业人，感觉都在画饼，不知道怎么判断。"

**版本2（焦虑版）**：
> "我们的技术真的挺好的，但是我已经创业1年半了，还没找到合适的商业合伙人。钱快烧完了，如果再找不到，可能就要回去打工了。
> 
> 我之前找的那个合伙人，聊的时候感觉特别好，什么资源都有，但真正合作起来发现完全不是那么回事。我现在都不敢相信人了，但又不得不找..."

---

### 常见问题清单（扮演者备用）

#### 【基础问题】（前15分钟会问）

**Q1**: 技术合伙人和商业合伙人股权怎么分合理？
- *预期顾问反应*：不直接给数字，先了解具体情况

**Q2**: 怎么判断一个销售总监能不能做合伙人？
- *预期顾问反应*：引导思考能力vs资源的权重

**Q3**: 面试商业合伙人应该问哪些问题？
- *预期顾问反应*：提供结构化面试清单

#### 【进阶问题】（15-30分钟会问）

**Q4**: 如果合伙人能力不行，怎么体面地分开？
- *预期顾问反应*：讨论退出机制设计

**Q5**: 你们有没有技术背景找到好商业合伙人的成功案例？
- *预期顾问反应*：分享案例，但强调个体差异

**Q6**: 商业合伙人要不要懂技术？
- *预期顾问反应*：分析不同模式的利弊

#### 【刁难问题】（30分钟后可能抛出）

**Q7**: 你说的这些我都在网上看过了，有什么不一样的？
- *压力点*：测试顾问的价值主张
- *应对参考*：强调方法论的系统性和实战验证

**Q8**: 你们自己做咨询的，有实际创业经验吗？
- *压力点*：质疑顾问资质
- *应对参考*：坦诚背景 + 强调研究方法论

**Q9**: 如果我按你们的方法选了合伙人，结果还是失败了怎么办？
- *压力点*：测试过度承诺
- *应对参考*：诚实说明风险 + 强调满意解不是保证

**Q10**: 你们的收费标准是什么？
- *压力点*：突然转向商业
- *应对参考*：介绍服务层次，但不强行推销

**场景：面试中的两难选择**

```
【背景】

• 优点：大厂销售总监背景，资源丰富，谈吐得体
• 疑点：之前创业失败过，负债200万
• 你的感觉：聊得来，但总觉得哪里不对劲

候选人B - 王总  
• 优点：连续创业者，上次成功退出，财务自由
• 疑点：年龄偏大（52岁），对新技术接受慢
• 你的感觉：能力强，但担心精力跟不上

【时间压力】
两人都表示如果这周不决定，就会转向其他机会。

【你的任务】
请使用7维度评估框架，做出选择并说明理由。
```

### 观察员重点观察清单

#### 顾问表现检查点

| 时间点 | 检查项 | 优秀表现 | 需改进 |
|--------|--------|----------|--------|
| 0-5min | 开场破冰 | 自然、建立信任 | 生硬、流程化 |
| 5-15min | 问题诊断 | 问到核心痛点 | 流于表面 |
| 15-30min | 工具介入 | 恰当使用7维度 | 工具使用生硬 |
| 30-45min | 应对刁难 | 从容、专业 | 慌乱、防御 |
| 45-60min | 总结收尾 | 有明确行动项 | 仓促结束 |

#### 客户体验检查点

- [ ] 客户是否感到被理解？（观察肢体语言、语气）
- [ ] 客户的困惑是否得到解答？
- [ ] 客户是否有"Aha moment"（恍然大悟）？
- [ ] 客户是否愿意继续深入？

## 剧本B：资源 founder 找技术合伙人

### 客户画像卡

| 项目 | 内容 |
|------|------|
| **姓名** | 张伟（化名） |
| **年龄** | 38岁 |
| **背景** | 前阿里P9，有丰富的资源和行业人脉 |
| **创业阶段** | 有商业模式，有种子客户，缺技术落地 |
| **核心痛点** | 技术合伙人迟迟不到位，产品无法开发 |
| **历史创伤** | 找过3个技术合伙人，都因各种原因没谈成 |
| **性格特点** | 结果导向、急躁、善于社交、重视效率 |
| **咨询诉求** | 技术合伙人应该给多少股权才合理 |

---

### 开场白（扮演者使用）

> "我在阿里做了10年，资源这块是不缺的，商业模式也跑通了，有3个种子客户等着签约。但就是没有技术合伙人！
> 
> 我已经找了3个技术候选人了，第一个要70%股权，我直接走了；第二个技术不错，但不愿意全职；第三个聊得挺好，但临签约前说家里不同意。
> 
> 我现在最大的困惑是，技术合伙人到底该给多少股权？为什么我找的人要么狮子大开口，要么不靠谱？"

---

### 常见问题清单

#### 【股权问题】

**Q1**: 技术合伙人一般给多少股权？
- *陷阱*：期望一个标准答案
- *应对参考*：解释股权分配的多个变量

**Q2**: 技术合伙人非要控股怎么办？
- *压力点*：两难选择
- *应对参考*：分析控股背后的动机

**Q3**: 股权是按贡献分还是按角色分？
- *深度问题*：涉及分配哲学
- *应对参考*：介绍动态股权机制

#### 【招募问题】

**Q4**: 怎么判断一个技术合伙人的真实水平？
- *常见困惑*：非技术背景看不懂技术
- *应对参考*：提供技术尽调框架

**Q5**: 技术合伙人不愿意全职怎么办？
- *现实难题*：优秀人才难找
- *应对参考*：讨论阶段性合作模式

#### 【刁难问题】

**Q6**: 你们说的这套方法太理论了，实战中根本用不了。
- *压力点*：否定方法论价值
- *应对参考*：用具体案例说明

## 剧本C：已有合伙人出现分歧

### 客户画像卡

| 项目 | 内容 |
|------|------|
| **姓名** | 王芳（化名） |
| **年龄** | 36岁 |
| **背景** | 与合伙人共同创业1年半，公司营收300万 |
| **创业阶段** | A轮前，产品已上线，有付费客户 |
| **核心痛点** | 与合伙人价值观冲突、利益分配不均 |
| **当前危机** | 合伙人想接一个大客户项目，但违背公司战略 |
| **性格特点** | 理性、重原则、不善表达情绪 |
| **咨询诉求** | 该修 relationship 还是该分手 |

---

### 开场白（扮演者使用）

> "我和合伙人一起创业1年半了，公司做得还行，但我们的关系越来越差。
> 
> 他是销售出身，我是产品出身。刚开始互补挺好的，但现在越来越觉得价值观不一样。他为了成单什么都答应客户，我觉得这样会毁掉产品。
> 
> 最近一次争吵是关于一个大客户项目。那个项目金额很大，但要我们做一个完全不在规划里的功能。我想拒绝，他说我有钱不赚是傻子。
> 
> 我现在真的很痛苦，不知道是该想办法修好这段关系，还是干脆分手。但分手的话，股权怎么分？客户怎么分？想想都头大..."

---

### 常见问题清单

#### 【关系修复问题】

**Q1**: 价值观不一样还能继续合作吗？
- *核心痛点*：深层分歧
- *应对参考*：区分核心价值观vs操作层面

**Q2**: 怎么和合伙人谈敏感问题？
- *实操需求*：沟通技巧
- *应对参考*：提供对话框架

#### 【分手问题】

**Q3**: 如果必须分手，怎么算公平？
- *复杂性*：涉及法律、情感、商业
- *应对参考*：介绍退出机制设计

**Q4**: 分手的话客户资源怎么分？
- *实操难题*：灰色地带
- *应对参考*：分析不同方案利弊

## 通用刁难话术库

### 质疑类

| 话术 | 压力级别 | 应对建议 |
|------|----------|----------|
| "你们说的这些我都在网上看过了" | ⭐⭐⭐ | 强调系统性和案例验证 |
| "你们自己有创业经验吗？" | ⭐⭐⭐⭐ | 坦诚背景 + 强调研究方法论 |
| "听起来都是理论，实战有用吗？" | ⭐⭐⭐ | 用具体案例和数据说话 |
| "如果按你们的方法还是失败了怎么办？" | ⭐⭐⭐⭐ | 诚实说明风险，不承诺结果 |

### 比较类

| 话术 | 压力级别 | 应对建议 |
|------|----------|----------|
| "XX咨询公司也是这么说的" | ⭐⭐ | 询问对方遇到的问题 |
| "我朋友说合伙人选择看缘分" | ⭐⭐ | 承认直觉作用，补充系统方法 |
| "你们和市面上其他服务有什么区别？" | ⭐⭐⭐ | 清晰的价值主张 |

### 拖延类

| 话术 | 压力级别 | 应对建议 |
|------|----------|----------|
| "我再考虑考虑" | ⭐⭐ | 了解具体顾虑 |
| "等我有时间再联系你" | ⭐⭐ | 设定具体跟进时间 |
| "我要和合伙人商量一下" | ⭐⭐ | 提供合伙人一起看的内容 |

### 价格类

| 话术 | 压力级别 | 应对建议 |
|------|----------|----------|
| "你们的收费太贵了" | ⭐⭐⭐ | 解释价值和ROI |
| "有免费的服务吗？" | ⭐⭐ | 介绍免费内容和付费区别 |
| "能不能先免费试一次？" | ⭐⭐ | 介绍试用产品/内容 |

## 观察员评分参考

### 优秀顾问表现（5分标准）

- ✅ 开场自然，3分钟内建立信任
- ✅ 提问精准，15分钟内触及核心问题
- ✅ 倾听到位，不打断客户，有确认复述
- ✅ 工具使用恰当，不生搬硬套
- ✅ 应对刁难从容，不防御不攻击
- ✅ 有明确行动项，客户知道下一步做什么
- ✅ 时间把控精准，不仓促不拖沓

### 需改进表现（3分以下）

- ❌ 开场生硬，像背书
- ❌ 问不到点上，客户反复说同样的话
- ❌ 频繁打断客户
- ❌ 工具使用生硬，为了用而用
- ❌ 被刁难后慌乱或防御
- ❌ 结束仓促，没有总结
- ❌ 严重超时或过早结束
