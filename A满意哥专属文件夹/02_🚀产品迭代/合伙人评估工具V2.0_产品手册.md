# 合伙人评估工具V2.0 产品手册

## 产品概述

基于满意解研究所10个案例数据构建的专业级合伙人匹配评估工具。

## 核心功能

### 1. 7维度深度评估
- **价值观契合度** (25%权重) - 3道问题
- **能力互补性** (20%权重) - 3道问题
- **沟通效率** (15%权重) - 3道问题
- **承诺可信度** (15%权重) - 3道问题
- **利益一致性** (10%权重) - 3道问题
- **退出可接受性** (10%权重) - 3道问题
- **成长匹配度** (5%权重) - 2道问题

**总计**: 18道精心设计的问题

### 2. 智能评分系统
- 每道题5个选项，对应1-10分
- 维度加权平均计算
- 综合得分0-10分

### 3. 风险等级判定
| 综合分 | 等级 | 建议 |
|--------|------|------|
| ≥8.0 | EXCELLENT | 强烈推荐 |
| 7.0-7.9 | LOW | 可以推进 |
| 6.0-6.9 | MEDIUM | 需谨慎 |
| 5.5-5.9 | HIGH | 需深度尽调 |
| <5.5 | CRITICAL | 建议否决 |

### 4. 个性化建议
根据各维度得分，自动生成针对性改进建议

### 5. 可视化报告
生成专业HTML评估报告，包含：
- 综合得分卡片
- 7维度进度条可视化
- 风险等级标识
- 建议行动清单

## 使用方法

### 快速评估（直接输入分数）
```python
from skills.assessment-tool-v2.tool import quick_assessment

scores = {
    "values_alignment": 8,
    "capability_complementarity": 9,
    "commitment_credibility": 8,
    "communication_efficiency": 7,
    "interest_alignment": 7,
    "exit_acceptability": 7,
    "growth_matching": 7
}

result = quick_assessment(scores)
print(f"综合得分: {result['overall_score']}")
print(f"风险等级: {result['risk_level']}")
```

### 交互式评估（完整18题）
```python
from skills.assessment-tool-v2.tool import run_assessment

result = run_assessment()  # 交互式答题
```

### 生成HTML报告
```python
tool = PartnershipAssessmentTool()
result = tool.quick_assessment(scores)
html_path = tool.generate_html_report(result)
```

## 技术特性

- **数据驱动**: 基于10个真实案例验证
- **科学权重**: 7维度权重基于案例统计
- **风险预警**: 内置10+风险信号识别
- **专业输出**: 可视化HTML报告

## 应用场景

1. **合伙前评估**: 正式合伙前的尽职调查
2. **磨合期诊断**: 发现问题，及时干预
3. **定期复盘**: 每季度评估关系健康度
4. **危机处理**: 冲突发生时的客观评估
