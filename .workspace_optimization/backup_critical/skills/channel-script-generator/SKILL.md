# 渠道话术生成器 (Channel Script Generator)

## 核心功能

为6大渠道类型生成定制化沟通话术，覆盖初次接触、深度交流、合作邀约全场景。

## 渠道类型

| 渠道 | 核心诉求 | 决策特点 |
|------|----------|----------|
| 律所 | 增值服务、客户粘性 | 注重风险控制 |
| 猎头 | 优质客户、返佣 | 注重效率 |
| 会所 | 转介价值、专业背书 | 注重合规 |
| 孵化器 | 服务增值、满意度 | 注重入驻体验 |
| 家办 | 风险控制、回报 | 注重长期价值 |
| 科学家 | 信任、专业性 | 注重方法论 |

## 话术结构

每套话术包含：
1. **开场白** (破冰、建立连接)
2. **痛点共鸣** (挖掘需求)
3. **价值展示** (解决方案)
4. **案例背书** (信任建立)
5. **合作邀约** (下一步行动)

## 使用方式

```bash
# 生成全部渠道话术
generate-scripts --all

# 生成指定渠道话术
generate-scripts --channel law-firm
generate-scripts --channel headhunter
generate-scripts --channel incubator

# 生成指定场景
generate-scripts --channel law-firm --scene first-contact
generate-scripts --channel law-firm --scene deep-talk
generate-scripts --channel law-firm --scene cooperation
```

- [ ] 开场白能在10秒内抓住注意力

- [ ] 案例背书具体可信
