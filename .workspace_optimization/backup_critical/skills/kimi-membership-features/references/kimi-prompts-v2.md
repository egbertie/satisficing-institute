## 🔑 关键认知: Agent池共享规则

**Agent功能共享同一个额度池**（网站+文档+表格）：
- Allegretto: 40次/月 ≈ **10次/周**
- **策略**: 批量处理、优先高价值任务

**独立额度**（不共享）：
- 深度研究: 40次/月 ≈ **10次/周**
- PPT: 40次/月 ≈ **10次/周**

**WPS大会员**（充足额度）：
- PPT制作、PDF转换、文档润色、云同步
- **策略**: PPT交给WPS，节省Kimi额度

## 功能1: 网站 (Web浏览/解析)

### 使用场景
- 快速获取政策原文、学术论文
- 提取关键信息，无需完整阅读

### 提示词模板（Agent池优化版）

```
【满意解研究所 - 网站解析任务】

## 目标网页
[粘贴URL]

## 解析目标
请提取以下信息：
1. **核心观点** (3-5条，带原文引用)
2. **关键数据** (数字、百分比、时间点)
3. **政策条款** (如适用，原文摘录)
4. **与满意解关联点** (如何支持合伙人决策研究)

## 输出格式
| 类别 | 内容 | 原文位置 |
|------|------|----------|
| 核心观点1 | ... | 第X段 |
| 关键数据 | ... | 第X段 |
| ... | ... | ... |

## 特殊要求
- 不确定的内容标注[待核实]
- 无法找到的内容标注[未提及]
- 如有PDF下载链接，请提取

## 后续处理
解析完成后，请将关键内容发送给OpenClaw进行深度分析和归档。
```

### 💡 Agent池节省技巧
- **批量查询**: 一次提供多个URL，合并处理
- **优先政策原文**: 仅用于获取无法手动下载的文档
- **替代方案**: 简单网页可用OpenClaw web_fetch（免费）

## 功能2: 文档 (文档处理/解析)

### 使用场景
- 解析PDF学术论文、政府白皮书
- 批量提取多个文档核心内容
- **优先使用此功能**（Agent池最高效用法）

### 提示词模板（批量优化版）

#### 单文档解析
```
【满意解研究所 - 文档解析任务】

## 目标文档
[上传文档]

## 解析目标
请按以下结构提取信息：

### 1. 元数据
- 标题:
- 作者/机构:
- 发布日期:
- 文档类型: [论文/报告/政策/其他]

### 2. 核心观点 (5-7条)
每条包含：观点摘要 + 原文引用 + 证据等级[P1/P2/P3/P4]

### 3. 关键数据 (表格形式)
| 数据项 | 数值 | 时间 | 来源(文中位置) |
|--------|------|------|----------------|

### 4. 与满意解关联点
- 可直接引用的内容:
- 需要验证的内容:
- 与五路图腾的关联: [土/水/木/金/火]

### 5. 需人工验证清单
- [ ] ...

## 输出要求
- 所有数据标注来源
- 不确定内容标注[待核实]
- 如有引用，使用[^ID^]格式
```

#### 批量文档解析（Agent池高效用法）⭐
```
【满意解研究所 - 批量文档解析任务】

## 目标文档（请一次性上传所有相关文档）
[上传多个相关文档，建议3-5个/批次]

## 解析目标
请对每个文档提取：
1. 核心观点 (3条)
2. 关键数据 (2-3个)
3. 与满意解关联点
4. 五路图腾对应维度 [土/水/木/金/火]

## 汇总要求
最后请生成对比矩阵：
| 文档 | 核心观点 | 关键数据 | 证据等级 | 关联度 | 图腾维度 |
|------|----------|----------|----------|--------|----------|
| 文档1 | ... | ... | P1/P2/P3 | 高/中/低 | 土/水/木/金/火 |
| 文档2 | ... | ... | ... | ... | ... |

## 特殊要求
- 相互矛盾的观点标注⚠️
- 重复内容只保留最早来源
- 不确定内容标注[待核实]

## 后续处理
请将对比矩阵发送给OpenClaw进行知识图谱整合。
```

### 💡 Agent池节省技巧
- **批量上传**: 一次处理3-5个文档，节省多次调用
- **优先级**: 学术论文>政府白皮书>行业报告
- **替代方案**: 简单文档可用WPS AI解析

## 功能3: PPT（WPS协同版）⭐

### 策略变更
**不再使用Kimi PPT功能，改为WPS大会员制作**
- 节省Kimi PPT额度（10次/周）
- WPS大会员PPT功能充足
- 模板库更丰富

### 协作流程

```
Egbertie提供要点
    ↓
OpenClaw生成素材和内容大纲（消耗Claw Token）
    ↓
WPS大会员制作PPT（消耗WPS额度，充足）
    ↓
最终交付
```

### 我给WPS的素材提示词模板

```
【WPS PPT制作素材 - 满意解研究所】

## PPT类型
[ ] 投资人路演  [ ] 专家讲座  [ ] 研究汇报  [ ] 其他

## 目标受众
[填写受众类型]

## 核心信息（我会为你生成）
1. 标题页: [主标题 + 副标题]
2. 内容大纲: [每页标题 + 要点]
3. 关键数据: [需要展示的数字/图表]
4. 金句引用: [需要高亮显示的内容]
5. 视觉建议: [配色/图片/图表类型]

## 格式要求
- 每页不超过50字
- 多用图表/数据可视化
- 配色: 专业稳重（深蓝+金色）
- 字体: 清晰易读

## 输出
请基于以上素材，使用WPS AI生成完整PPT。
```

### 我为你生成的素材包含

| 内容大纲 | 每页标题+要点 | WPS直接套用 |

## 功能5: 深度研究（周期配额管理版）

### 使用场景
- 每日深度研究主题（10次/周）
- 核心学术支持

### 提示词模板（配额优化版）

```
【满意解研究所 - 深度研究任务】

## 研究主题
[填写具体研究主题]

## 研究背景
满意解研究所专注于硬科技初创企业家合伙人选择决策教练：
- 理论基础：赫伯特·西蒙满意解理论 + 前景理论 + 儒商哲学
- 方法论：左脑逻辑 + 右脑直觉（感知力）综合决策
- 框架：五路图腾（土水木金火）× 五大专家

## 执行指令

### 阶段1: 情报网格构建
1. 执行≥3个独立搜索查询，覆盖：
   - 学术数据库（arXiv/Google Scholar/JSTOR）
   - 灰色文献（政府白皮书/智库报告）
   - 行业数据源（SEC/Patents/统计局）
   - 多语种信源（英语+中文）

2. 对关键主张执行"5层溯源"
3. 主动检索 contradictory evidence（反方证据）

### 阶段2: 证据链锻造
- 学术引用微格式: ^ID^ 作者. (年份). 标题. 期刊/出版社. DOI/URL. [检索日期]
- 证据等级: [P1/P2/P3/P4]

### 阶段3: 五路图腾关联分析
| 图腾 | 关联点 | 应用方式 |
|------|--------|----------|
| 土-信实 | ... | ... |
| 水-变通 | ... | ... |
| 木-生长 | ... | ... |
| 金-精准 | ... | ... |
| 火-能量 | ... | ... |

### 阶段4: 逻辑架构
引言 → 理论框架 → 方法论 → 多维度发现 → 讨论 → 满意解应用点 → 批判性反思

### 阶段5: 迭代优化
信息饱和 → 逻辑压力测试 → 元认知审计

## 输出要求

### 1. 执行摘要（500字）
- 研究背景
- 核心发现（3-5条）
- 满意解应用价值

### 2. 详细研究报告（6000-8000字）
- 按上述逻辑架构组织
- 所有数据标注来源
- 所有案例标注真实性

### 3. 核心观点/金句库（10-15条）
每条包含：金句原文 + 来源标注 + 满意解应用场景

### 4. 可引用数据清单（表格）
| 数据项 | 数值 | 时间 | 来源 | 证据等级 | 置信度 |
|--------|------|------|------|----------|--------|

### 5. 五路图腾关联矩阵
| 图腾 | 关联点 | 应用方式 | 优先级 |
|------|--------|----------|--------|

### 6. 需人工验证清单
- [ ] ...

## 禁忌
- ❌ "多项研究表明..."（不指明具体研究）
- ❌ "众所周知..."（必须提供溯源）
- ❌ 将推测包装为结论
- ❌ 编造不存在的引用

## 置信度编码
High/Medium/Low/Speculative

## 后续流程
生成完成后，请将论文发送给OpenClaw进行验证优化，最终输出2.0学术严谨版。

请生成完整研究报告。
```
