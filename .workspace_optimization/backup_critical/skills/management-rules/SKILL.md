# 📜 管理规则手册 V1.0.0

## 🎯 功能概述

### 五大规则体系
1. **沟通规则** - 汇报层级、沟通准则、响应时效
2. **任务管理** - 生命周期、优先级、截止日期
3. **报告机制** - 晨报/日报/周报/里程碑报告
4. **记忆更新** - 文件体系、更新频率、内容标准
5. **安全规则** - 权限分级、敏感操作、数据安全

## 🚀 使用方法

### 规则查询
```
用户: 查询报告机制规则

AI: 📊 报告机制规则
   
   报告类型与频率:
   ├─ 晨报: 每日09:00 → 今日计划+昨日完成
   ├─ 日报: 每日23:59 → 33角色全景
   ├─ 周报: 每周日 → 周进展+下周计划
   ├─ 里程碑: 达成时 → 成果+教训
   └─ 风险报告: 发现时 → 风险+应对方案
   
   晨报模板必含:
   ├─ 今日重点(Top 3)
   ├─ 昨日完成
   ├─ 进行中任务
   ├─ 风险/阻塞
   └─ 需要决策
```

### 合规检查
```
用户: 检查记忆更新合规性

AI: 🔍 记忆更新合规检查
   
   检查项目:
   ├─ MEMORY.md更新: ✅ 已更新
   ├─ memory/YYYYMMDD.md: ✅ 已创建
   ├─ 重大决策记录: ⚠️ 缺少[RULE]标签
   └─ 截止日期记录: ✅ 已记录
   
   合规率: 75%
   建议: 为重大决策添加[RULE]标签
```
