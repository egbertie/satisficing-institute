# 🧠 第一性原理工作法 V1.0.0

## 🎯 功能概述

### 核心流程
1. **质疑** - 挑战现有假设
2. **拆解** - 找到最基本的事实
3. **重构** - 从本质建立新方案

### 适用场景
- 复杂问题分析
- 创新方案设计
- 决策质量提升

## 🚀 使用方法

### 本质拆解
```
用户: 用第一性原理分析合伙人匹配问题

AI: 🧠 第一性原理分析
   
   问题: 如何找到合适的合伙人？
   
   第一步：质疑假设
   ├─ ❌ "合伙人要能力强"
   ├─ ❌ "合伙人要志同道合"
   └─ ❌ "合伙关系靠感觉"
   
   第二步：拆解到本质
   ├─ 合伙人 = 资源互补 + 风险共担 + 目标一致
   ├─ 匹配 = 能力匹配 + 价值观匹配 + 时机匹配
   └─ 成功 = 结构合理 + 沟通顺畅 + 退出明确
   
   第三步：重构方案
   ├─ 建立7维度评估模型
   ├─ 设计风险预警机制
   └─ 预设退出条款
```
