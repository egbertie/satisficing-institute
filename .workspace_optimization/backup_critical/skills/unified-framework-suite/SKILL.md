# Unified Framework Suite

**统一框架与方法论套件** - 整合决策框架、管理哲学、合作模式。

> 🎯 替代: decision-frameworks + management-philosophy-framework + partnership-model

---

## 核心能力

| 功能模块 | 覆盖场景 |
|---------|---------|
| **决策框架** | 决策树、矩阵分析、情景规划 |
| **管理方法** | 敏捷、精益、OKR、平衡计分卡 |
| **合作模式** | 伙伴关系、联盟策略、生态构建 |
| **战略规划** | SWOT、五力模型、蓝海战略 |
| **框架应用** | 模板库、案例库、实施指南 |

---

## 快速开始

```bash
# 决策框架
framework-suite decision --method "满意解" --options options.json --criteria criteria.json

# 管理方法
framework-suite management --method OKR --objectives "目标1,目标2" --key-results "KR1,KR2"

# 合作模式
framework-suite partnership --type strategic --evaluation "互补性,信任度,目标一致性"

# 战略分析
framework-suite strategy --tool swot --subject "新产品线"
```
