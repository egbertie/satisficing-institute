## 采集阶段检查

### 语言覆盖检查（灵活平衡）

**基本原则：**
- 不要为了双语而双语！
- 质量 > 数量，深度 > 广度

**具体策略：**

| 领域 | 语言策略 | 原因 |
|:---|:---|:---|
| **技术研究** | 优先英文 | 技术迭代源头在硅谷 |
| **中国文化/国学** | 优先中文 | 英文资源有限，中文一手资料 |
| **管理方法论** | 中英双搜 | 西方理论+东方实践 |
| **行业动态** | 中英并重 | 全球视野，中美双主导 |
| **法律/政策** | 适用语言优先 | 中国法律用中文，美国用英文 |

**例外处理：**
- 如果某领域英文/中文确实稀缺，不必强行凑数
- 专注高质量的单语来源，胜过低质量的双语拼凑
- 中国文化研究（儒商、五路图腾、六祖等）以中文为核心，英文可选

### 来源权威性检查
- 学术信息：优先期刊、著作、权威媒体
- 行业信息：优先头部媒体、官方渠道
- 政策信息：必须政府官方来源

## 入库阶段检查

### 必填字段（缺一不可）
```yaml
必填:
  - 信息标题
  - 作者姓名
  - 出处来源（期刊/媒体/书籍）
  - 发布时间
  - 原文链接（如有）
  - 关键摘录
  - 质量评级（1-5星）
  - 入库人
  
可选:
  - 中文明细来源
  - 英文明细来源
  - 应用场景
  - 审核状态
```

### 引用格式标准
```
【标准引用】
作者：[姓名]
出处：[期刊/媒体/书籍名称]
时间：[发布年份/日期]
链接：[URL或N/A]
```

### 原创声明格式
```
【原创声明】
本分析为满意解研究所原创，综合以下来源：
1. [引用来源1]
2. [引用来源2]
...
原创观点：
- [原创观点1]
- [原创观点2]
```

## 质量评级标准

| 评级 | 标准 | 处理方式 |
|:---:|:---|:---|
| ⭐⭐⭐⭐⭐ | 权威来源、多次验证可靠 | 直接入库，优先使用 |
| ⭐⭐⭐⭐ | 可靠来源、偶有偏差 | 入库使用，标注注意 |
| ⭐⭐⭐ | 一般来源、特定场景 | 入库备用，谨慎使用 |
| ⭐⭐ | 待观察、新发现 | 待审核，暂不正式使用 |
| ⭐ | 不可靠、已淘汰 | 拒绝入库 |

---

## 禁止事项（红线）

❌ **严禁以下行为：**
1. 未标注出处的引用
2. 来源不权威却未标注"待验证"
3. 原创观点未声明
4. 过时信息未标注历史背景

✅ **但允许（灵活处理）：**
- 中国文化研究以中文为主，不强求英文
- 技术研究以英文为主，不强求中文
- 不要为了凑双语而降低信息质量
- 单语高质量 > 双语低质量

## 违规处理流程

```
发现违规
    ↓
立即标注"待完善"
    ↓
48小时内补充
    ↓
无法补充 → 降级或移除
```
