## 事件响应机制

### 用户指令触发 (人工即主权)

| 指令 | 响应 |
|------|------|
| "检查状态" | 全面状态报告 |
| "任务队列" | 当前任务列表 |
| "Token情况" | Token使用分析 |
| "启动补位" | 分析并建议可补位任务 |
| "复盘建议" | 效率分析+改进建议 |

### 关键事件自动通知

| 事件 | 优先级 | 响应 |
|------|--------|------|
| 任务超时 (>24h) | 🔴 紧急 | 立即通知 |
| Token < 10% | 🔴 紧急 | 立即通知 |
| 任务阻塞 (>30min) | 🟡 高 | 10分钟内通知 |
| Token < 30% | 🟡 高 | 下次交互提示 |
