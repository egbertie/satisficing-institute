## 任务1: 深度研究 - 脑科学决策神经机制 ⭐ P0

**关联**: 今晚方翊沣博士知识图谱构建

### 提示词（复制到Kimi深度研究）

```
【满意解研究所 - 深度研究任务】

## 研究主题
"创业决策中的神经机制：基于证据累积理论的合伙人信任判断研究"

## 研究背景
满意解研究所专注于硬科技初创企业家合伙人选择决策教练，采用左脑逻辑+右脑直觉（感知力）综合决策方法论。

方翊沣博士（马来西亚理科大学脑科学系、前中国科学院深圳先进技术研究院副研究员）将作为金图腾（精准）的感知力训练导师，为本研究提供学术支持。

## 研究目标
1. 为感知力决策建立神经科学基础
2. 提取可引用的脑科学研究结论
3. 与满意解五路图腾方法论建立关联
4. 识别可落地的感知力训练方法

## 执行指令

### 阶段1: 情报网格构建
搜索范围：
- 学术数据库：PubMed/Google Scholar/Neuroscience期刊
- 关键词：decision making neuroscience, trust neural mechanisms, evidence accumulation theory, somatic marker hypothesis, interoceptive awareness
- 重点关注：近5年（2020-2025）顶刊研究

### 阶段2: 核心研究问题
1. 信任判断的神经机制是什么？（脑区激活模式：杏仁核、前额叶皮层、岛叶）
2. 直觉决策与理性决策的神经差异？（双系统理论神经基础）
3. 高压情境下的决策偏差有哪些神经基础？（皮质醇对决策的影响）
4. 神经反馈训练如何提升决策质量？（已有实证研究）
5. 证据累积理论在商业决策中的应用？

### 阶段3: 五路图腾关联分析
- 金（精准）- 神经机制与决策精准度的关系
- 火（能量）- 压力状态下的神经调节
- 水（变通）- 环境变化时的神经可塑性

### 阶段4: 输出要求
1. 执行摘要（500字）
2. 详细研究报告（6000-8000字）
3. 决策神经机制图谱（脑区功能图）
4. 可引用研究清单（带DOI，≥20条）
5. 感知力训练实用方法（≥5种）
6. 五路图腾关联矩阵
7. 需人工验证清单

### 特殊要求
- 区分"已验证的科学结论"和"探索性应用"
- 标注每种证据的置信度（High/Medium/Low）
- 不确定内容标注[待核实]

请生成完整研究报告。

8. 需人工验证清单

```

## 收集目标
感知力训练（Intuition/Somatic Markers/Embodied Cognition）的实用方法

## 理论基础
- 达马西奥躯体标记假说（Somatic Marker Hypothesis）
- 具身认知（Embodied Cognition）
- 内感受知觉（Interoceptive Awareness）
- 正念冥想（Mindfulness）
- 神经反馈（Neurofeedback）

   - 生理指标监测（HRV/GSR/EEG）

2. 实践方法：
   - 正念冥想（MBSR/MBCT）
   - 身体扫描（Body Scan）
   - 决策日记（Decision Journal）
   - 72小时极限测试
   - 红色 teaming（红军模拟）

3. 商业应用：
   - 高管教练（Executive Coaching）方法
   - 军事/应急决策训练
   - 投资直觉训练

| 维度 | 具体内容 |
|------|----------|
| 方法名称 | 中英文 |
| 理论基础 | 支撑该方法的科学理论 |

| 效果评估 | 如何衡量进步？|
| 科学依据 | 支持该方法的学术研究（≥1篇）|
| 适用场景 | 硬科技创业合伙人选择的哪个阶段？|

4. 满意解感知力训练体系草案

## 收集数量目标
- 初步目标：15-20种方法
- 深度方法：5-8种（有完整操作指南）

## 特殊要求
- 区分"有科学证据支持"和"实践者经验"
- 标注每种方法的证据等级（高/中/低）
- 不确定内容标注[待验证]
- 引用关键学术文献

## 后续处理
收集完成后，请将方法清单发送给OpenClaw进行体系化整合，纳入满意解工具包。
