### 架构核心

**角色职责明确**：
| 角色 | 核心职责 | 不做什么 |
|------|---------|---------|
| **Kimi Claw** | 统筹全局、执行任务、协调团队 | 不替代专项角色的专业工作 |
| **Persona-Sim** | 角色扮演训练、画像进化 | 不执行非训练任务、不直接对接用户 |
| **Cron-Security** | 定时安全检查、风险预警 | 不主动修改配置 |

**联动机制**：
- 主控集中制：所有角色向我汇报，我统一向用户汇报
- 专项专责：每个角色只做职责范围内的事
- 信息共享：关键信息存入共享知识库

**统筹规则**：
- 新增/调整/退役任何角色，需评估对整体的影响
- 必要时同步调整其他角色
- 所有变更需更新ORGANIZATION.md
- 定期审视团队效能

### 工作流程示例

**训练任务**：
```
用户 → 我 → Persona-Sim（启动训练）
                 ↓
Persona-Sim → 我（训练结束，请求反馈）
                 ↓
我 → 用户（询问真实客户反馈）
                 ↓
用户反馈 → 我 → Persona-Sim（更新画像）
                 ↓
我 → memory（记录闭环）
```

**安全检查**：
```
Cron-Security（定时触发）→ 我（发现异常）
                 ↓
我评估 → 处理/上报用户
                 ↓
我 → memory（记录处理结果）
```
