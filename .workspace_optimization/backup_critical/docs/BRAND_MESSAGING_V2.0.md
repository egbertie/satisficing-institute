> 满意解——用科学方法，帮你找到最合适的合伙人。

**核心痛点切入**:
```
"创业路上，你可能听过很多决策建议，但有没有发现：

❌ 很多建议是基于'完美理性人'假设，而你面对的是信息不完备的现实
❌ 很多建议告诉你'要理性'，但你的直觉往往是对的
❌ 很多建议只讲方法，但忽略了决策背后的心理机制

满意解研究所结合了诺贝尔经济学奖得主西蒙的'满意解'理论
和认知心理学的前沿成果，帮你：

✓ 在有限信息下找到足够好的合伙人
✓ 识别并克服决策中的认知偏差
✓ 校准直觉与理性，做出明智决策

这不是玄学，是科学。"
```

## 三、核心概念对外表述

### 3.1 概念对照表

| 内部概念 | 对外表述 | 避免表述 | 说明 |
|----------|----------|----------|------|
| 西蒙满意解 + 前景理论 | "满意决策科学" | "前景理论满意解" | 统一品牌话语 |
| 有限理性 | "理解边界，科学决策" | "人是非理性的" | 通俗化、正面表达 |
| 认知偏差管理 | "识别盲区，校准判断" | "纠正你的错误" | 价值导向、非评判 |
| 五路图腾 | "儒商决策智慧" | "儒家管理" | 文化锚定、去宗教化 |
| 感知力 | "决策直觉的科学训练" | "第六感" | 去神秘化 |
| 损失厌恶 | "保守倾向管理" | "你怕损失" | 温和表达 |
| 参考点 | "决策锚点" | "心理参照系" | 简化术语 |
| 抱负水平 | "可接受标准" | "最低要求" | 避免负面联想 |
| 满意解 | "最适合的方案" | "将就的方案" | 品牌核心，正面表达 |
| 明智决策 | "超越理性的智慧" | "非理性决策" | 品牌升华 |

### 3.2 禁用词汇清单

**品牌保护类**:
- ❌ "卡尼曼满意解" - 混淆理论归属
- ❌ "前景理论匹配" - 喧宾夺主
- ❌ "行为经济学合伙人" - 过于学术

**负面暗示类**:
- ❌ "非理性" - 暗示客户不理性
- ❌ "认知错误" - 过于负面
- ❌ "偏差纠正" - 居高临下
- ❌ "将就" - 与满意解理念冲突

**过度承诺类**:
- ❌ "保证找到" - 过度承诺
- ❌ "100%成功" - 无法兑现
- ❌ "完美合伙人" - 违背满意解理念

### 3.3 概念解释话术

**当有人问"什么是满意解？"**:
```
标准回答:
"满意解是诺贝尔经济学奖得主西蒙提出的理论。
它告诉我们：人类是有限理性的，不可能找到'完美'方案，
所以应该在信息、时间、认知的约束下，
找到一个'足够好'的方案就果断行动。

我们用这个理论帮助创业者：
不用追求完美的合伙人，只要找到最适合当前阶段的，
在正确的时间做出决定，然后全力投入。"
```

**当有人问"和猎头有什么区别？"**:
```
标准回答:
"猎头帮你找人，我们帮你决策。

猎头问的是'这个人好不好'，
我们问的是'这个人适不适合你，现在是不是做决定的时候'。

我们解决的不是信息不对称问题，
而是决策偏差问题——
帮你识别什么时候该冒险、什么时候该保守，
什么时候该继续找、什么时候该停下来决定。"
```

**当有人问"为什么要引入前景理论？"**:
```
标准回答:
"西蒙的理论告诉我们人类有认知边界，
卡尼曼的前景理论告诉我们人类有哪些系统性偏差。

比如，人天生更怕损失而不是追求收益，
这会导致你在该引入合伙人的时候因为怕'失去控制权'而犹豫不决。

我们整合这两个理论，
不是为了让你知道这些概念，
而是为了在决策时帮你识别并克服这些偏差，
做出真正明智的决定。"
```

## 五、视觉与语言风格指南

### 5.1 语言风格

**整体调性**: 专业但不学术，科学但不冰冷，理性但有温度

**DO**:
- ✓ 用类比解释复杂概念
- ✓ 讲故事而非讲道理
- ✓ 承认复杂性而非过度简化
- ✓ 用"我们"拉近距离
- ✓ 提供具体行动建议

**DON'T**:
- ✗ 堆砌学术术语
- ✗ 居高临下的说教
- ✗ 过度承诺或恐吓
- ✗ 负面攻击竞争对手
- ✗ 空洞的鸡汤

### 5.2 语气参考

| 场景 | 语气 | 示例 |
|------|------|------|
| **专业咨询** | 权威、可信 | "根据我们的研究和实践，建议..." |
| **科普内容** | 亲和、易懂 | "你知道吗？其实我们在决策时..." |
| **危机处理** | 冷静、专业 | "我们理解您的担忧，让我们一起..." |
| **成交推动** | 真诚、紧迫 | "时间窗口有限，我们可以马上..." |

### 5.3 品牌声音检查清单

```
发布任何内容前，问自己：

□ 是否自然融入了"满意解"品牌？
□ 是否让核心理论（西蒙满意解）保持主角地位？
□ 对外表述是否通俗易懂？
□ 是否符合五路图腾的价值观？
□ 是否避免了禁用词汇？
□ 语气是否专业而有温度？
```
