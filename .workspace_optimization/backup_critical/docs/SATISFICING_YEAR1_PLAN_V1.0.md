## 一、核心目标（唯一目标）

### 1.1 第一年的唯一目标

**验证商业模式 + 积累案例资产**

| 维度 | 具体目标 |
|-----|---------|
| **案例数** | 完成30个深度合伙人评估案例 |
| **标杆客户** | 建立3个可脱敏展示的标杆案例 |
| **收入目标** | 健康现金流（覆盖时间成本） |
| **验证结论** | 确认"合伙人评估服务"是否成立 |

### 1.2 明确的非目标（不做）

| 非目标 | 为什么不追求 |
|-------|-------------|
| 收入最大化 | 验证期优先模式验证，而非利润 |
| 客户数量 | 深度案例比数量更重要 |
| 品牌知名度 | 口碑比曝光更有价值 |
| 团队扩张 | OPC形态下不招人 |
| 产品多元化 | 第一年只有一个产品 |
| 市场覆盖 | 仅服务大湾区硬件初创 |

## 二、唯一核心产品

### 2.1 产品定义

| 要素 | 定义 |
|-----|------|
| **产品名称** | 合伙人决策评估服务 |
| **产品定位** | 帮助初创期硬件企业家做出不后悔的合伙人选择 |
| **目标客户** | 获得政府补贴的初创期智能硬件企业家 |
| **交付周期** | 2-4周 |
| **建议定价** | 3-8万元/项目（根据复杂度浮动） |

### 2.2 产品交付（极简三件套）

```
┌─────────────────────────────────────────────────────┐
│           合伙人决策评估服务                         │
├─────────────────────────────────────────────────────┤
│  交付1：3次深度面谈                                  │
│  ├─ 第1次：需求澄清（60分钟）                        │
│  ├─ 第2次：候选人评估（90分钟）                      │
│  └─ 第3次：决策建议（60分钟）                        │
├─────────────────────────────────────────────────────┤
│  交付2：1份评估报告                                  │
│  ├─ 格式：飞书/Notion文档                            │
│  ├─ 内容：画像+风险+建议+行动                        │
│  └─ 页数：≤5页                                      │
├─────────────────────────────────────────────────────┤
│  交付3：1次专家会诊（可选/另收费）                    │
│  ├─ 场景：复杂案例需要专家介入                       │
│  └─ 定价：1-2万元/次                                │
└─────────────────────────────────────────────────────┘
```

### 2.3 无其他产品（第一年）

- ❌ 无免费引流产品
- ❌ 无低价体验产品
- ❌ 无会员订阅产品
- ❌ 无培训课程产品
- ❌ 无工具/软件产品
- ❌ 无咨询陪跑产品

**唯一变体**：专家会诊服务（仅针对已购买评估服务的客户）

## 三、唯一客户画像

### 3.1 客户画像（唯一权威定义）

> ⚠️ **重要**：本文档包含满意解第一年客户画像的**唯一权威定义**。其他文档引用时，请直接链接到本节。

```
满意解第一年的唯一客户：

┌─ 基本信息 ──────────────────────────────┐
│ 阶段：初创期（获得政府补贴）              │
│ 领域：智能硬件（机器人/芯片/传感器等）     │
│ 地域：深圳/大湾区                         │
│ 角色：创始人/CEO亲自决策                  │
└─────────────────────────────────────────┘

┌─ 痛点特征 ──────────────────────────────┐
│ • 正在考虑引入合伙人，但不确定人选          │
│ • 之前有过合伙人失败的痛苦经历              │
│ • 听说过合伙人撕逼案例，担心自己也会遇到    │
│ • 意识到合伙人选择的重要性，但缺乏判断方法  │
└─────────────────────────────────────────┘

┌─ 决策特征 ──────────────────────────────┐
│ • 决策紧迫：有明确的时间压力               │
│ • 预算可控：3-8万元在可接受范围内          │
│ • 亲自参与：创始人愿意投入时间面谈         │
│ • 重视信任：更相信人脉推荐而非广告         │
└─────────────────────────────────────────┘
```

### 3.2 明确不服务的客户

| 不服务类型 | 原因 |
|-----------|------|
| 非硬件领域 | 专业能力边界外 |
| 非初创期（已过A轮） | 决策复杂度不同，需求不匹配 |
| 非深圳/大湾区 | 面谈成本过高 |
| 无明确合伙人选择需求 | 需求不明确，难以交付 |
| 非创始人亲自决策 | 信息传递损耗，效果打折 |
| 预算低于3万元 | 无法覆盖时间成本 |

## 四、极简运营方式

### 4.1 获客运营（极简）

**获客动作**：
1. 维护专家网络关系（月度咖啡/微信）
2. 定期更新案例进展（让专家知道你在做什么）
3. 标杆客户主动请求推荐

**获客物料**：
- 1页介绍文档（飞书文档即可）
- 3个标杆案例（脱敏后）
- 客户推荐信/反馈截图

### 4.2 交付运营（极简）

**交付工具**：
| 环节 | 工具 | 说明 |
|-----|------|------|
| 面谈 | 飞书/腾讯会议 | 线上为主，线下为辅 |
| 报告 | Notion/飞书文档 | 不做PDF设计 |
| 日程 | 飞书日历 | 客户自助预约 |
| 收款 | 个人银行账户 | 第一年不搞公司化 |

**交付流程**：
```
客户引荐 → 15分钟初步沟通 → 确认需求 → 签约付款
                                              ↓
结案归档 ← 交付报告 ← 3次面谈 ← 准备评估 ← 启动
```

### 4.3 合作运营（极简）

**合作管理**：
- 专家网络：微信分组管理，定期同步
- 外包设计：按件付费，不签长期
- 外包研究：按项目付费，不养团队

### 4.4 不做的运营

- ❌ 不做内容矩阵
- ❌ 不做社群运营
- ❌ 不做活动策划
- ❌ 不做品牌投放
- ❌ 不做 SEO/SEM
- ❌ 不做邮件营销

## 五、季度里程碑

### 5.1 Q1（3-4月）：验证期

**核心目标**：完成3个案例，验证服务模式

| 周次 | 关键动作 | 目标 |
|-----|---------|------|
| W1-2 | 专家网络激活 | 获取首批3个潜在客户 |
| W3-6 | 完成3个评估 | 完成3个合伙人评估 |
| W7-8 | 反馈收集 | 收集客户反馈，优化流程 |

**不做的事**：
- 不追求收入（甚至可以免费/低价）
- 不追求数量（3个足够）
- 不扩张服务范围

**成功标准**：
- 3个客户愿意付费（哪怕低价）
- 客户反馈正向（愿意推荐）
- 流程基本跑通

### 5.2 Q2-Q3（5-9月）：积累期

**核心目标**：完成20个案例，建立标杆

| 月份 | 目标 | 关键动作 |
|-----|------|---------|
| 5月 | 3个案例 | 每月稳定获客 |
| 6月 | 3个案例 | 筛选第1个标杆 |
| 7月 | 4个案例 | 优化评估模板 |
| 8月 | 4个案例 | 筛选第2个标杆 |
| 9月 | 6个案例 | 筛选第3个标杆 |

**不做的事**：
- 不扩张团队
- 不增加产品
- 不扩大客户范围

**成功标准**：
- 20个案例完成
- 3个标杆案例确定（可脱敏展示）
- 客户转介绍率>30%

### 5.3 Q4（10-12月）：模式固化期

**核心目标**：完成30个案例，商业模式跑通

| 月份 | 目标 | 关键动作 |
|-----|------|---------|
| 10月 | 5个案例 | 总结案例规律 |
| 11月 | 5个案例 | 撰写年度报告 |
| 12月 | 决策月 | 决定是否第二年扩张 |

**不做的事**：
- 不提前扩张
- 不增加投入
- 不启动新业务

**成功标准**：
- 30个案例完成
- 年度报告输出
- 第二年战略明确

## 六、财务规划（极简）

### 6.1 收入预估

| 场景 | 案例数 | 平均单价 | 年收入 |
|-----|-------|---------|-------|
| 保守 | 20 | 3万 | 60万 |
| 中性 | 30 | 5万 | 150万 |
| 乐观 | 40 | 8万 | 320万 |

**第一年目标**：中性偏保守（100-150万）

### 6.2 成本结构

| 成本项 | 预估 | 说明 |
|-------|------|------|
| 时间成本 | 主成本 | Egbertie时间投入 |
| 外包设计 | <2万 | 报告模板/品牌物料 |
| 外包研究 | <3万 | 行业研究支持 |
| 专家会诊分成 | 10-20% | 专家会诊收入分成 |
| 其他杂费 | <1万 | 交通/会议等 |

### 6.3 现金流原则

- 预付款：签约收50%，结案收50%
- 无垫资：不垫付任何费用
- 轻资产：不购买固定资产
- 快周转：单个项目周期<1个月

## 七、成功指标

### 7.1 关键指标（北极星指标）

**北极星指标：深度案例数**

原因：
- 案例是商业模式验证的核心
- 案例是专家网络背书的依据
- 案例是第二年扩张的基础

### 7.2 辅助指标

| 指标 | 目标值 | 说明 |
|-----|-------|------|
| 客户满意度 | >80% | 愿意推荐 |
| 转介绍率 | >30% | 口碑传播 |
| 复购/增购率 | >20% | 专家会诊转化 |
| 按时交付率 | >90% | 运营效率 |

### 7.3 失败指标（预警）

| 指标 | 预警线 | 应对措施 |
|-----|-------|---------|
| 案例数<15 | 9月底 | 延长验证期或止损 |
| 满意度<60% | 任何时间 | 立即复盘调整 |
| 转介绍率<10% | 6月底 | 重新评估价值主张 |
| 精力分散>30% | 任何时间 | 回归边界审视 |
