## 第一性原理拆解

### 原理1：客户价值第一性
```
客户购买的不是：
❌ 评估工具
❌ 理论体系  
❌ 咨询时间

客户购买的是：
✅ "我能做出不后悔的合伙人选择"（状态改变）
✅ "我有可执行的信任判断"（能力获得）
✅ "我降低了决策风险"（安全感）
```
