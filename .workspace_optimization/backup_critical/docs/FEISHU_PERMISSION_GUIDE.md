# 飞书权限申请指南

> 本指南帮助您申请飞书开放平台所需的文档权限，以恢复飞书文档同步功能。

## 当前权限状态

- ✅ **已有权限**：drive:file:title:read（读取文件标题）
- ❌ **缺少权限**：文档内容读写权限

## 需要申请的权限

| 权限名称 | 权限代码 | 用途 |
|---------|---------|------|
| 读取文件内容 | `drive:file:read` | 读取飞书云文档内容 |
| 写入文件 | `drive:file:write` | 修改/更新飞书文档 |
| 读取文档 | `docx:document:read` | 读取 docx 文档内容 |
| 写入文档 | `docx:document:write` | 写入 docx 文档 |

## 申请步骤

### 步骤1：登录飞书开放平台

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 使用管理员账号登录

### 步骤2：进入应用管理

1. 点击右上角 **"开发者后台"**
2. 找到并点击需要配置的应用（或创建新应用）

### 步骤3：配置权限

1. 左侧菜单选择 **"权限管理"**
2. 点击 **"添加权限"** 或 **"申请权限"**
3. 在搜索框中输入需要添加的权限代码
4. 勾选以下权限：
   - ✅ `drive:file:read`
   - ✅ `drive:file:write`
   - ✅ `docx:document:read`
   - ✅ `docx:document:write`

### 步骤4：提交审核（如需）

- 部分权限需要企业管理员审核
- 填写申请理由示例：

```
本应用用于企业内部文档自动化同步管理，需要读取和写入文档内容以实现：
1. 本地知识库与飞书文档的双向同步
2. 文档版本管理和备份
3. 团队协作内容的自动化更新

不涉及敏感数据，仅用于企业自有文档管理。
```

### 步骤5：发布版本（重要！）

**权限变更后必须发布新版本才能生效：**

1. 左侧菜单选择 **"版本管理"**
2. 点击 **"创建版本"**
3. 填写版本信息：
   - 版本号：如 `1.0.1`
   - 更新说明：如 "添加文档读写权限"
4. 点击 **"发布"** 或 **"申请发布"**

## 验证权限

权限生效后，运行以下命令验证：

```bash
# 检查当前应用权限
openclaw tools feishu_app_scopes

# 测试读取文档
openclaw tools feishu_doc read --doc_token YOUR_DOC_TOKEN
```

## 常见问题

### Q1: 申请被拒绝怎么办？
- 联系企业管理员说明用途
- 提供更详细的应用场景说明
- 考虑使用"飞书多维表格"替代方案（见方案2）

### Q2: 发布后权限仍不生效？
- 确保应用已安装到需要使用的企业/组织
- 检查是否使用了正确的 App ID 和 App Secret
- 尝试重新获取 tenant_access_token

### Q3: 没有开发者权限？
- 联系飞书管理员获取开发者权限
- 或请管理员代为配置应用权限

## 替代方案

如果权限申请困难，可考虑：

1. **多维表格方案**（推荐）：使用 `scripts/feishu_bitable_sync.py`
2. **手动导入方案**：使用 `scripts/feishu_doc_import.py`

详见相应脚本的使用说明。
