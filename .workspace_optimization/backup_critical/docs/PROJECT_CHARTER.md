## 1. 项目愿景

### 1.1 愿景陈述

> **打造满意解研究所，成为中国领先的硬科技创业合伙人匹配决策系统提供商**

我们致力于通过AI驱动的决策科学，帮助硬科技创业者找到最适合的投资人和合伙人，显著提高创业成功率，降低错配风险。

### 1.2 使命宣言

- **短期**: 在2026年Q1完成产品MVP验证，服务首批10个硬科技创业项目
- **中期**: 在2026年实现首年80万收入，建立行业口碑
- **长期**: 在3年内成为中国硬科技创业服务领域的知名品牌

### 1.3 核心价值观

| 价值观 | 解释 | 行为准则 |
|--------|------|----------|
| **数据驱动** | 一切决策基于数据 | 收集、分析、验证，拒绝拍脑袋 |
| **专业深度** | 硬科技领域专精 | 深入理解技术和市场 |
| **客户成功** | 客户成功即我们成功 | 结果导向，长期陪伴 |
| **敏捷迭代** | 快速试错，快速学习 | 每周复盘，持续优化 |

## 2. 项目范围

### 2.1 业务范围

**核心业务**: 硬科技创业合伙人匹配决策教练

#### 服务内容
1. **投资人匹配评估**
   - 投资人画像分析
   - 匹配度算法评分
   - 尽职调查辅助

2. **合伙人匹配评估**
   - 联合创始人适配性分析
   - 核心团队组建建议
   - 股权架构设计咨询

3. **决策支持系统**
   - 多维度决策模型
   - 风险评估与预警
   - 决策执行跟踪

### 2.2 技术范围

**核心产品**: 硬科技合伙人匹配决策系统 (TechPartner AI)

| 模块 | 功能 | 技术栈 |
|------|------|--------|
| 数据采集 | 市场/投资人/项目数据 | Python + Jina AI |
| 匹配引擎 | 算法模型/评分系统 | Python + ML |
| 对话系统 | 客户交互/咨询 | Kimi API + OpenClaw |
| 知识库 | 案例库/方法论 | Notion + Markdown |
| 报告生成 | 决策报告/匹配建议 | Python + 模板 |

### 2.3 目标客户

**主要客户群体**:
- 硬科技领域天使轮-A轮创业公司
- 有融资需求的科研院所成果转化项目
- 寻求联合创始人的连续创业者

**目标市场**: 
- 地理: 初期聚焦中国（北京/上海/深圳/杭州）
- 领域: 人工智能、半导体、新材料、生物医药

### 2.4 项目边界

**包含**:
- 合伙人匹配决策服务
- 投资人对接辅助
- 决策报告生成
- 方法论输出

**不包含**:
- 直接融资撮合（仅提供匹配建议）
- 法律/财务尽职调查（推荐专业机构）
- 投后管理服务
