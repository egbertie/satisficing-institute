1. 打开企业微信，进入 `liuhuabin` 群聊
2. 点击右上角「...」→「群机器人」
3. 点击「添加机器人」
4. 选择「新创建一个机器人」
5. 填写机器人名称：**满意解小助手**
6. 上传头像（可选）
7. 点击「确定」

### 步骤2：获取Webhook地址

1. 创建完成后，会显示Webhook地址
2. 格式类似：
   ```
   https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
   ```
3. **复制这个地址发给我**

### 安全提示

- Webhook地址相当于机器人的"密码"
- 不要泄露给无关人员
- 如泄露，可在群设置中重置
