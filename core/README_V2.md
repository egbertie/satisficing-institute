# 满意解知识管理系统 V2.0 - 完整版

> **基于第一性原理的全域知识基础设施**
> 
> 核心理念：单一数据源（Single Source of Truth）+ 自动派生渲染 + 全域覆盖

---

## 架构设计（三阶金字塔 - 扩展版）

```
┌─────────────────────────────────────────────────────────────────┐
│  第三阶：发布层 (Publish) - 自动生成，只读                       │
│  ├── 官网文档/                                                  │
│  ├── 客户交付包/ (V2.0下载包等)                                 │
│  ├── 营销素材/                                                  │
│  └── API文档/                                                   │
│  【生成方式】脚本自动渲染，禁止手动修改                            │
├─────────────────────────────────────────────────────────────────┤
│  第二阶：工作层 (Working) - 自动同步，半只读                      │
│  ├── V1.3完全版本.md (从Core同步)                               │
│  ├── 战略定位1.1.md (从Core同步)                                │
│  ├── 产品工具手册.md (从Core同步)                               │
│  ├── 案例库/ (从Core案例库.yaml生成)                            │
│  ├── 专家档案/ (从Core专家档案.yaml生成)                        │
│  └── 技术文档/ (从Core技术架构.yaml生成)                        │
│  【生成方式】脚本从Core同步，人工可覆盖特殊场景                    │
├─────────────────────────────────────────────────────────────────┤
│  第一阶：核心层 (Core) - 人工维护，唯一真相源                     │
│  ├── 五路图腾.yaml           # 核心理念定义                      │
│  ├── 专家网络.yaml           # 专家基础信息                      │
│  ├── 产品体系.yaml           # 产品矩阵定义                      │
│  ├── 品牌资产.yaml           # 品牌核心资产                      │
│  ├── 案例库.yaml             # 15个案例结构化数据                │
│  ├── 专家档案.yaml           # 6位专家详细档案                   │
│  ├── 技术架构.yaml           # V2.0技术架构定义                  │
│  ├── 运营体系.yaml           # 运营文档核心定义                  │
│  ├── 技能注册.yaml           # 189个Skill索引                    │
│  └── 记忆索引.yaml           # 历史记忆归档索引                  │
│  【维护方式】人工编辑，变更触发自动同步                           │
└─────────────────────────────────────────────────────────────────┘
```

## 使用方式

### 标准流程
```bash
# 1. 修改Core定义
vim core/案例库.yaml

# 2. 运行对应同步脚本
python3 scripts/sync_cases.py

# 3. 运行全局检查
python3 scripts/consistency_check.py
```

### 一键全量同步
```bash
./scripts/sync_all.sh
```

### 历史记忆迁移
```bash
python3 scripts/migrate_memory.py
```
