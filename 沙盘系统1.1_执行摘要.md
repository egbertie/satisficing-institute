## 一、系统核心价值

## 五、复盘与迁移工具

### 5.1 四维度复盘法

```
决策复盘：
├── 数据层：数据使用是否准确？遗漏了什么？
├── 直觉层：直觉信号是否准确？是否被倾听？
├── 执行层：执行是否到位？环节出了什么问题？
└── 迁移层：经验如何迁移？需要调整什么？
```

### 5.2 行动清单生成器

```
从案例到行动：
1. 核心洞察：_________________
2. 立即行动（3个）：
   - _________________
   - _________________
   - _________________
3. 需要学习：_________________
4. 检查清单：□ □ □
```

### 5.3 个人决策风格诊断

**四种类型**：
- A型（数据驱动）：需加强直觉训练
- B型（直觉驱动）：需加强数据分析
- C型（关系驱动）：需建立独立判断
- 均衡型（双脑决策）：需保持平衡

## 六、使用路径

### 6.1 个人训练路径（4-6周）

**Week 1-2：熟悉案例**
- 阅读全部20个案例
- 选择3个相关场景深度研究
- 完成决策风格诊断

**Week 3-4：场景演练**
- 选择3-5个场景模拟决策
- 记录决策日志
- 完成感知力训练

**Week 5-6：实战应用**
- 应用到真实决策
- 建立复盘习惯
- 持续优化

### 6.2 教练引导路径

**案例教学**：
1. 选择匹配场景
2. 讲解背景与决策点
3. 引导学员做出选择
4. 对比Egbertie决策
5. 完成复盘总结

**角色扮演**：
1. 学员扮演决策者
2. 教练扮演顾问
3. 模拟完整决策流程
4. 即时反馈与指导
