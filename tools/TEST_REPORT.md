## 4. 已知限制

1. **无数据持久化**: 评估结果不保存到数据库，刷新页面后丢失
2. **单用户设计**: 无并发用户隔离，适用于个人评估
3. **CORS限制**: 跨域请求需要额外配置（当前仅支持同源访问）
4. **无身份验证**: 公开访问，无需登录
