# 决策基因体检 - 终极版产品手册

## 一、产品定位

### 1.1 产品愿景
**成为中国创业者首选的决策能力科学测评与提升平台**

### 1.2 核心价值主张
- **科学**：基于心理测量学标准，信度>0.85，多维度效度验证
- **智能**：AI实时解读，动态报告，预测性建议
- **全面**：从测评到诊断到教练到工具，完整服务闭环
- **极致**：游戏化+沉浸式+无障碍的顶级体验

### 1.3 目标用户
| 用户类型 | 特征 | 核心需求 |
|---------|------|---------|
| 早期创业者 | 0-1阶段，决策经验少 | 了解自己的决策风格，避免踩坑 |
| 成长期创业者 | A轮前后，决策复杂度提升 | 优化决策流程，提升决策质量 |
| 成熟创业者 | B轮+，团队决策挑战 | 团队决策协同，培养决策能力 |
| 投资机构 | VC/PE，投前尽调 | 评估创始人决策能力，降低投资风险 |

### 2.3 核心流程

#### 2.3.1 用户测评流程

```
开始测评
    ↓
环境检测（设备、网络、环境）
    ↓
引导说明（3分钟视频/图文）
    ↓
练习题目（2题，熟悉操作）
    ↓
正式测评（58题，约25-35分钟）
    ├─ 经典决策题（24题）
    ├─ 认知反应题（12题，测反应时）
    ├─ 陷阱题（8题）
    ├─ 一致性检验题（6题）
    └─ 认知风格题（8题）
    ↓
数据质量评估
    ↓
报告生成（AI实时生成，3秒内）
    ↓
报告呈现（交互式网页）
    ↓
AI解读对话（可选）
    ↓
后续服务推荐（产品2/3/4）
```

#### 2.3.2 报告生成流程

```
用户答题数据
    ├─ 答案数组
    ├─ 反应时数据
    ├─ 鼠标轨迹
    └─ 陷阱题触发
    ↓
评分计算引擎
    ├─ 8维度原始分计算
    ├─ 反应时分析（直觉/理性）
    ├─ 数据质量评分
    └─ 常模百分位转换
    ↓
内容组装引擎
    ├─ 根据画像匹配内容块
    ├─ 匹配成功案例
    ├─ 生成对比分析
    └─ 预测建议
    ↓
AI解读生成
    ├─ 构建Prompt
    ├─ 调用Claude/GPT
    ├─ 结果校验
    └─ 格式化输出
    ↓
可视化渲染
    ├─ 雷达图生成
    ├─ 对比图表
    ├─ 风格矩阵
    └─ 时间轴
    ↓
输出报告
```
