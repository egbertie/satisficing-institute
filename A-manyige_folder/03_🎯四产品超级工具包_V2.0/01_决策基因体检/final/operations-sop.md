## 五、增值运营SOP

### 5.1 复购与升级

#### 5.1.1 复购引导策略

```
复购时机：

首次复购（3-6个月）：
• 触发条件：收到积极反馈/决策困境咨询
• 策略：推荐诊断服务或教练服务
• 优惠：老客户专属9折

年度复购（12个月）：
• 触发条件：年度追踪提醒
• 策略：决策基因体检复测
• 优惠：复测5折

升级复购：
• 免费→专业：免费版完成7天后
• 专业→尊享：使用3个月后
• C端→B端：团队扩张时
```

### 5.2 交叉销售

#### 5.2.1 产品1→2→3→4转化

```
产品1（体检）→ 产品2（诊断）：
• 触发：测评结果显示高风险
• 推荐话术："您的决策风格存在3个潜在风险点，深度诊断可以帮您..."
• 目标转化率：40%

产品2（诊断）→ 产品3（教练）：
• 触发：诊断发现问题需要长期解决
• 推荐话术："针对您的XX问题，建议通过1v1教练服务进行系统提升"
• 目标转化率：30%

产品3（教练）→ 产品4（SaaS）：
• 触发：教练服务结束，需要持续工具支持
• 推荐话术："为了保持决策能力的持续提升，推荐您使用我们的决策工具"
• 目标转化率：60%
```

## 八、危机处理SOP

### 8.1 常见危机场景

| 危机类型 | 触发条件 | 响应时间 | 处理流程 |
|---------|---------|---------|---------|
| 数据泄露 | 疑似/确认泄露 | 1小时 | 技术封锁→法务评估→用户通知→公关应对 |
| 负面舆情 | 社交媒体发酵 | 2小时 | 监测→评估→回应→跟进 |
| 服务中断 | >30分钟 | 立即 | 技术抢修→用户通知→补偿方案 |
| 投诉升级 | 多渠道投诉 | 4小时 | 专人跟进→快速解决→回访确认 |

### 8.2 舆情监控

```
监控渠道：
• 微博、知乎、小红书
• 创业社群、行业论坛
• 媒体报道

预警关键词：
```
