# 年度决策顾问 - 第5轮迭代：社会影响力扩展

## 迭代概述

**轮次**: 第5轮 - 社会影响力扩展  
**核心目标**: 将决策顾问服务从商业领域扩展至社会领域，创造更广泛的社会价值  
**参考对象**: Bridgespan Group（非营利咨询）+ Dalberg（发展咨询）+ 政府智库模式

## 3. 非营利组织决策支持

### 3.1 服务定义
为非营利组织（基金会、NGO、行业协会等）提供决策支持，提升其治理水平与决策质量。

### 3.2 非营利组织决策特点

| 特点 | 描述 | 对决策的影响 |
|------|------|--------------|
| **多元利益相关者** | 捐赠者、受益者、志愿者、社区、政府 | 决策需平衡多方利益，程序比结果更重要 |
| **使命驱动** | 以社会使命为核心，非利润最大化 | 决策标准复杂，难以量化 |
| **资源约束** | 资源有限且不稳定 | 决策容错率低，需特别谨慎 |
| **治理结构** | 理事会决策、秘书处执行 | 决策与执行分离，沟通成本高 |
| **信任基础** | 依赖公众信任与捐赠 | 声誉风险是重大决策风险 |

### 4.4 案例教学法

#### 决策案例开发流程

```
决策案例开发流程:

1. 案例选题
   └── 识别重大决策事件 → 评估教学价值 → 获得企业授权

2. 案例研究
   ├── 资料收集（公开资料 + 访谈）
   ├── 决策过程重构
   ├── 关键决策点识别
   └── 决策后果追踪

3. 案例撰写
   ├── 案例正文（情境、决策点、数据）
   ├── 教学笔记（教学目标、讨论问题、理论链接）
   └── 多媒体素材

4. 案例测试
   ├── 试讲
   ├── 反馈收集
   └── 案例修订

5. 案例发布
   ├── 案例库收录
   └── 持续更新
```
