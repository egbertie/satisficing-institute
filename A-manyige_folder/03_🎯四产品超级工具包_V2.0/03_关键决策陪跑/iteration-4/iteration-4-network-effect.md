# 关键决策陪跑 - 第4轮迭代：网络效应构建

## 迭代目标
从"一对一服务"升级为"网络化服务"，通过构建同行支持网络、专家会诊体系、校友互助网络，实现服务价值的指数级放大。

## 一、同行决策支持小组

### 1.1 小组设计原则

**小组规模**：5-8人（最佳6人）
**组建标准**：
- 行业相关但非直接竞争（如都是SaaS但不同细分领域）
- 发展阶段相近（融资轮次、营收规模、团队规模）
- 地理位置相对集中（便于线下聚会）
- 决策者层级匹配（创始人对创始人，CEO对CEO）

**小组类型**：

| 类型 | 成员特征 | 讨论重点 |
|-----|---------|---------|
| 早期创业组 | 种子-A轮，0-50人 | 产品方向、融资策略、早期团队 |
| 成长期组 | B-C轮，50-300人 | 规模化、组织建设、市场扩张 |
| 成熟期组 | D轮+/Pre-IPO，300人+ | 战略转型、资本运作、退出规划 |
| 行业专题组 | 同行业不同细分领域 | 行业趋势、政策影响、生态合作 |

### 1.4 小组规则与公约

**保密原则**：
- 小组内分享的所有信息严格保密
- 不得向外泄露他人公司敏感信息
- 案例学习时使用脱敏信息

**参与承诺**：
- 每月例会出席率≥80%
- 连续2次缺席视为自动退出
- 请假需提前48小时通知

**互动准则**：
- 真诚分享，不隐藏失败
- 积极倾听，不打断他人
- 提供建设性反馈，不评判
- 尊重不同观点

**退出机制**：
- 成员可主动申请退出
- 连续3次违反公约将被请退
- 毕业机制：企业上市后"毕业"成为校友

## 四、决策导师匹配

### 4.1 导师体系设计

**导师类型**：

| 类型 | 背景要求 | 服务形式 | 适合客户 |
|-----|---------|---------|---------|
| 创业导师 | 成功创业者（B轮以上或盈利） | 月度深度交流 | 早期创业者 |
| 阶段导师 | 经历过相同阶段的创业者 | 关键节点辅导 | 特定阶段困惑 |
| 行业导师 | 10年+行业经验 | 专项问题咨询 | 行业特定问题 |
| 职能导师 | 特定职能专家（CFO/CTO/CMO） | 职能建设指导 | 职能建设需求 |

**匹配算法**：

```python
def mentor_matching(client, mentor_pool):
    """
    导师匹配算法
    """
    scores = []
    
    for mentor in mentor_pool:
        score = 0
        
        # 行业匹配（25%）
        if client.industry == mentor.industry:
            score += 25
        elif is_related_industry(client.industry, mentor.industry):
            score += 15
        
        # 阶段匹配（25%）
        stage_diff = abs(client.stage - mentor.stage_experience)
        score += max(0, 25 - stage_diff * 5)
        
        # 决策风格匹配（20%）
        style_similarity = calculate_similarity(
            client.decision_style, 
            mentor.coaching_style
        )
        score += 20 * style_similarity
        
        # 需求匹配（20%）
        if client.primary_need in mentor.expertise_areas:
            score += 20
        elif is_related_area(client.primary_need, mentor.expertise_areas):
            score += 10
        
        # 性格匹配（10%）
        personality_fit = assess_compatibility(
            client.personality,
            mentor.personality
        )
        score += 10 * personality_fit
        
        scores.append((mentor, score))
    
    # 返回Top 3匹配结果
    return sorted(scores, key=lambda x: x[1], reverse=True)[:3]
```
