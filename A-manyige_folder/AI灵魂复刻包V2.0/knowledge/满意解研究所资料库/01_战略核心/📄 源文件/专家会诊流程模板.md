# 满意解研究所 · 专家会诊流程模板
## 跨学科专家委员会 · 复杂案例决策支持

---

## 一、会诊目的

**官方定位**：
> 针对复杂、高风险、多维度交织的合伙人匹配决策，启动跨学科专家委员会进行多维度交叉验证，提升决策质量。

**内部定位**：
> 整合方法论专家、行业专家、命理顾问等多方视角，为创始人决策提供全面支持。

---

## 二、会诊触发条件

### 必须启动会诊的情形
- [ ] 客户为年度战略级客户（合同金额>50万）
- [ ] 候选人背景复杂（涉及跨境、多段创业经历、法律纠纷史）
- [ ] 创始人与候选人存在潜在价值观冲突信号
- [ ] 技术转移风险高（TRL 6-7关键节点）
- [ ] 涉及二代接班、家族企业治理等敏感场景

### 建议启动会诊的情形
- [ ] 首次接触的新行业（缺乏案例库积累）
- [ ] 客户表现出高度焦虑或决策困难
- [ ] 评估结果呈现"高能力+高风险"特征
- [ ] 需要快速决策（时间压力<1周）

---

## 三、专家委员会构成

### 3.1 核心成员（必须参加）

| 角色 | 职责 | 对外身份 | 内部身份 |
|------|------|----------|----------|
| **主持人** | 流程把控、综合判断 | 决策科学专家 | 创始人/PEO |
| **方法论专家** | 评估框架审查 | 哲学/数学教授 | 原有师生关系 |
| **行业专家** | 技术/市场可行性 | 西丽湖技转专家 | 技术转移顾问 |

### 3.2 特邀成员（按需参加）

| 角色 | 职责 | 对外身份 | 内部身份 |
|------|------|----------|----------|
| **法律顾问** | 合规风险、股权架构 | 律所合伙人 | 渠道合作律师 |
| **财务顾问** | 估值模型、对赌条款 | 会计师事务所 | 合作会计师 |
| **心理学顾问** | 认知风格、行为分析 | 组织心理学专家 | 合作心理咨询师 |
| **命理顾问** | ⚠️ 仅限内部参考 | 不对外披露 | 内部命理专家 |

### 3.3 专家网络维护

**合作方式**：
- 前期：免费合作（联合署名论文、学术声誉共建）
- 后期：按项目分成（不固定薪资）
- 年度：专家答谢会+案例集赠送

**年度成本**：0元（首年），后续视合作深度调整

## 四、会诊流程

### 4.1 会前准备（T-3天）

**由PEO负责**：

- [ ] **案例材料包准备**
  - 客户背景（脱敏处理）
  - 候选人评估报告（TRL+PFI）
  - 关键访谈记录
  - 风险提示清单

- [ ] **专家邀约**
  - 确认核心成员时间
  - 按需邀请特邀成员
  - 发送案例摘要（不含命理信息）

- [ ] **命理预审**（内部）
  - 命理顾问提前审查（如有需要）
  - 将命理发现转换为"认知风格"术语
  - 准备内部备忘录（不对外）

### 4.2 会诊会议（2-3小时）

#### 第一阶段：背景介绍（20分钟）
**主持人（创始人/PEO）**：
```
"今天我们讨论的案例是：
- 客户背景：XX领域科学家，TRL 6阶段
- 候选人概况：XX背景，5C评级卡显示...
- 核心矛盾：创始人关注X，候选人倾向Y
- 需要解决的问题：是否建议引入？关键风险点？"
```

#### 第二阶段：专家发言（每人15分钟）
**发言顺序**：
1. 行业专家（技术可行性、市场前景）
2. 方法论专家（评估框架、逻辑一致性）
3. 心理学顾问（认知风格匹配、行为风险）
4. 法律顾问（合规风险、股权架构建议）
5. **命理顾问（如有）→ 用"直觉判断""模式识别"表述**

**发言模板**：
```
"从[我的专业领域]角度，我观察到：
- 关键发现1...
- 关键发现2...
- 风险提示...
- 建议..."
```

#### 第三阶段：交叉讨论（30分钟）
**主持人引导**：
- "各位专家的意见是否有冲突？"
- "需要补充什么信息才能做出判断？"
- "如果必须给出一个建议，会是什么？"

#### 第四阶段：综合判断（20分钟）
**主持人总结**：
```
"基于各位专家的输入，我的综合判断是：
- 主要结论：建议引入/不建议引入/需进一步验证
- 关键依据：专家A的X观点+专家B的Y发现
- 风险提示：...
- 后续建议：..."
```

### 4.3 会后输出（T+1天）

**PEO负责生成**：

1. **《专家会诊纪要》（对外版）**
   - 参会专家（对外身份）
   - 讨论要点（已转换话术）
   - 专家建议汇总
   - 综合结论

2. **《内部备忘录》（内部版）**
   - 命理相关发现（如有）
   - 专家间分歧点
   - 主持人的直觉判断
   - 后续跟踪要点

3. **《客户反馈建议》**
   - 如何向客户呈现结论
   - 需要客户补充的信息
   - 下一步行动建议

## 五、会诊记录模板

### 5.1 会诊基本信息

```yaml
会诊编号：C-2026-001
日期：2026-03-07
地点：融创中心/线上
客户：（脱敏编号）
主持人：
参会专家：
  - 方法论专家：XXX（哲学教授）
  - 行业专家：XXX（技转专家）
  - 心理学顾问：XXX（组织心理学）
  - 命理顾问：XXX（内部）
案例类型：科学家引入商业合伙人
```

### 5.2 专家观点摘要

| 专家 | 核心观点 | 风险提示 | 建议 |
|------|----------|----------|------|
| 方法论专家 | 评估框架完整，但权重需调整 | 过度依赖定量指标 | 增加PFI权重至40% |
| 行业专家 | 技术前景良好，TRL 7可达 | 市场时机不确定 | 建议分阶段引入 |
| 心理学顾问 | 认知风格互补，但冲突风险高 | 价值观差异未充分评估 | 增加价值观对齐工作坊 |
| 命理顾问（内部） | 火土相生，但需防... | ... | ... |

### 5.3 综合结论

```
【会诊结论】
建议：谨慎引入，分阶段验证

【关键依据】
1. 行业专家：技术可行性高（TRL 7可达）
2. 心理学顾问：认知风格互补，但需价值观对齐
3. 方法论专家：建议增加右脑评估权重

【风险提示】
- 高：价值观冲突可能导致决策分歧
- 中：市场时机不确定性
- 低：技术实现风险

【后续建议】
1. 安排创始人与候选人共同参加价值观对齐工作坊
2. 设计3个月试用期，明确退出机制
3. 90天后复诊评估
```

## 六、命理信息的特殊处理

### 6.1 命理顾问参与规则

**会前**：
- 提前1天收到案例材料（含真实信息）
- 准备内部备忘录（使用命理术语）
- 将发现转换为"认知风格""行为倾向"等表述

**会中**：
- 最后发言（避免影响其他专家）
- 使用转换后的话术：
  - ❌ "八字不合"
  - ✅ "决策时序偏好存在显著差异"

**会后**：
- 内部备忘录存档于07_命理内部/
- 不出现在对外纪要中

### 6.2 命理发现的话术转换示例

| 命理发现 | 对外表述 |
|----------|----------|
| 两人八字相冲 | "在高压决策场景下，双方的应激反应模式可能产生冲突" |
| 命格互补 | "认知风格呈现高度互补，有利于覆盖决策盲区" |
| 流年不利 | "当前处于业务关键转型期，建议分阶段推进" |
| 财运旺盛 | "资源配置能力和风险承担意愿较强" |

## 七、质量控制

### 7.1 会诊有效性评估

每次会诊后，由PEO评估：
- [ ] 专家意见是否充分表达了？
- [ ] 是否存在重大遗漏？
- [ ] 综合结论是否有充分依据？
- [ ] 命理信息是否妥善处理？

### 7.2 会诊结果跟踪

3个月后回访：
- 实际决策结果 vs 会诊建议
- 专家判断的准确性
- 案例入库（匿名化处理）

### 7.3 专家绩效评估

年度评估各专家：
- 参与会诊次数
- 建议被采纳比例
- 预测准确性（跟踪验证）
- 内外话术合规性

## 八、快速启动清单

### 启动会诊前（PEO执行）
- [ ] 确认触发条件
- [ ] 准备案例材料包
- [ ] 邀约专家（至少3位核心成员）
- [ ] 预订会议室/设置线上会议
- [ ] 准备命理预审（如有需要）

### 会诊当天（主持人执行）
- [ ] 开场介绍（5分钟）
- [ ] 背景说明（15分钟）
- [ ] 专家发言（每人15分钟）
- [ ] 交叉讨论（30分钟）
- [ ] 综合判断（20分钟）
- [ ] 总结散会（5分钟）

### 会诊后（PEO执行）
- [ ] 24小时内生成纪要
- [ ] 审核命理信息是否外泄
- [ ] 提交主持人确认
- [ ] 存档（对外版+内部版）
- [ ] 设置3个月跟踪提醒
