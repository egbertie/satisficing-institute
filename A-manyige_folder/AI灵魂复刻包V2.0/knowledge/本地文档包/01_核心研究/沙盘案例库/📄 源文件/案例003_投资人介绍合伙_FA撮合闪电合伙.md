# 投资人介绍合伙沙盘案例
> **案例编号**: CASE-003  
> **类型**: FA撮合的闪电合伙（投资人介绍）  
> **领域**: 生物医药（CAR-T细胞治疗）  
> **难度**: ★★★★★

## 一、背景

### 1.1 撮合方：某头部医疗FA机构
- **角色**: 不仅撮合交易，更撮合团队
- **服务模式**: "投后团队搭建"增值服务
- **收费模式**: 融资额的2%+团队搭建成功费

### 1.2 合伙人A：Dr.林（42岁）
- **身份**: 前FDA审评员，回国科学家
- **技术背景**: 15年细胞治疗监管经验，熟悉中美双报
- **核心资产**: 监管策略know-how，FDA人脉
- **诉求**: 找CEO快速推进临床，不愿自己管运营

### 1.3 合伙人B：赵总（45岁）
- **身份**: 前某上市公司生物医药板块负责人
- **背景**: 成功主导过2个新药上市，熟悉临床运营
- **资源**: 临床医院资源、CRO合作网络
- **诉求**: 寻找有差异化技术的平台，做一番事业

## 二、关键决策点

### 2.1 决策点1：闪电信任建立（权重35%）
**核心矛盾**: 30天内要建立深度信任 vs 正常需要6-12个月

**风险点**:
- 信息茧房：FA为促成交易可能选择性披露
- 蜜月期偏差：快速推进掩盖潜在分歧
- 背景尽调不足：无法验证过往业绩真实性

**满意解策略**:

#### 策略1：分阶段信任建立
```
阶段1（0-30天）：商业信任
├── 签署NDA + 竞业禁止
├── 第三方背景调查（雇佣专业机构）
├── FA提供双方过往项目参考人
└── 里程碑：共同出席投资人路演

阶段2（30-90天）：战略信任
├── 共同制定18个月路线图
├── 暴露脆弱性：各自说出最大担忧
├── 小规模冲突测试（刻意制造分歧观察反应）
└── 里程碑：完成首轮融资

阶段3（90-180天）：情感信任
├── 共同经历危机（不可避免会到来）
├── 非工作场景深度交流
└── 里程碑：顺利度过首个重大分歧
```

#### 策略2：FA责任绑定
| FA承诺 | 具体条款 | 违约后果 |
|--------|----------|----------|
| 信息真实 | 双方背景信息经FA核实 | 退还全部费用 |
| 匹配质量 | 提供3组备选方案 | 费用折扣50% |
| 后续支持 | 合伙后6个月持续跟进 | 免费延长服务 |

### 2.2 决策点2：估值与股权时序（权重30%）
**特殊挑战**:
- Dr.林的技术平台尚未形成专利
- 赵总以未来全职加入作为投入
- FA催促快速确定估值以便启动融资

**满意解策略**:

#### 股权设计：时间加权法
```
初始股权结构（公司注册时）:
├── Dr.林：40%（技术方）
├── 赵总：30%（预留，按全职入职进度解锁）
├── 期权池：20%
└── FA顾问费转股：10%

赵总股权解锁条件:
├── 入职即解锁10%
├── 6个月后解锁10%
├── 首轮融资完成后解锁5%
└── IND获批后解锁5%
```

#### 估值策略：两步走
1. **第一步（种子轮）**: 淡化估值，强调里程碑
   - 采用SAFE协议，估值封顶
   - 重点展示团队组合价值

2. **第二步（A轮）**: 数据说话
   - 临床前数据出来后重新定价
   - 此时团队磨合度已验证

### 2.3 决策点3：FA角色边界（权重20%）
**潜在风险**:
- FA过度介入公司运营决策
- FA为追求下一轮融资佣金，推动不理性扩张
- 信息泄露：FA同时服务竞争对手

**满意解策略**:

#### FA服务范围界定
| 服务项 | 是否包含 | 边界说明 |
|--------|----------|----------|
| 团队撮合 | ✅ | 已完成 |
| 首轮融资 | ✅ | 限定3家以内深度对接 |
| 战略建议 | ⚠️ | 仅建议，不决策 |
| 运营介入 | ❌ | 不进入董事会 |
| 后续融资 | 可选 | 按业绩决定是否续约 |

#### 信息防火墙
- 要求FA签署排他性保密协议
- 核心技术信息分期披露
- 设立"FA沟通窗口人"，统一信息出口

### 2.4 决策点4：闪电合伙后的减速带（权重15%）
**问题描述**: 30天完成的合伙，需要用180天来夯实

**满意解策略**:

#### 强制磨合期设计
```markdown
《闪电合伙后180天特别条款》

第1-30天：观察期
- 每周一次深度1:1（非工作议题50%时间）
- 共同参加合伙关系辅导（专业机构）
- 不做不可逆重大决策（>100万支出需双方同意）

第31-90天：测试期
- 引入第三方项目复盘机制
- 刻意暴露一个分歧，测试解决能力
- 启动首轮融资，共同面对投资人拷问

第91-180天：固化期
- 完成公司章程、股东协议细化
- 建立正式决策流程
- 确定长期股权调整机制
```

## 四、满意解策略

### 4.1 核心原则
> **"闪电开始，慢火熬汤；用制度弥补信任时间不足"**

### 4.2 关键机制设计

#### 机制1：信任加速协议
```markdown
《FA撮合特别信任协议》

第1条 背景核实条款
双方同意在14天内完成：
- 雇佣第三方进行背景调查
- 提供至少3个可核实的项目参考人
- FA对信息真实性承担连带责任

第2条 减速带条款
注册公司后180天内：
- 任何一方可随时提出"暂停键"，无理由暂停合伙7天
- 暂停期间进行深度反思和沟通
- 暂停结束后决定是否继续

第3条 退出保护条款
180天内退出视为"无过错退出"：
- 股权按原始出资或技术评估值回购
- 不触发竞业限制
- FA退还部分撮合费用
```

#### 机制2：FA制衡机制
| 制衡点 | 具体措施 |
|--------|----------|
| 信息披露 | FA需披露同期撮合的所有类似项目 |
| 利益冲突 | 设立"冷静期"，确认无冲突后再推进 |
| 服务质量 | 分期支付费用，与合伙稳定性挂钩 |
| 持续责任 | FA承诺12个月内提供团队关系支持 |

#### 机制3：闪电合伙复盘机制
- **每周**: 15分钟"合伙健康度"快速检查
- **每月**: 正式复盘会议，评估信任进展
- **每季**: 第三方辅导介入
- **第180天**: 全面评估，决定是否进入"正常合伙模式"

### 4.3 成功指标（180天）
- [ ] 首轮融资成功（最低目标）
- [ ] 完成背景尽调，无重大负面发现
- [ ] 成功解决至少2次重大分歧
- [ ] 双方信任评分>4/5
- [ ] 无"暂停键"被触发或触发后成功解决

## 五、案例复盘要点

### 5.1 适用于哪些场景
✅ FA/猎头撮合的高管组合  
✅ 跨境团队搭建（海归+本土）  
✅ 时间窗口紧迫（竞对追赶、政策窗口）  

### 5.2 不适用于哪些场景
❌ 技术过于早期，需要长期磨合方向  
❌ 一方有复杂历史（多次创业失败、法律纠纷）  
❌ FA收费模式存在严重利益冲突  

### 5.3 咨询要点清单
- [ ] FA背景调查（FA的FA）
- [ ] 双方背景深度核实
- [ ] 闪电合伙动机分析（是真急还是假急）
- [ ] 信任建立加速方案设计

### 5.4 特别关注信号
🚨 **红灯信号**（建议暂缓）:
- FA催促过急（"明天必须签"）
- 一方拒绝背景调查
- 双方对成功定义差异巨大
- 任何一方有"先上车后补票"心态

🟡 **黄灯信号**（需加强保护机制）:
- 双方此前从未共事过
- 行业周期处于下行阶段
- 技术路线存在重大争议
