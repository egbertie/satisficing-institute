## 五行相生关系

```
    土（LIU-价值）
         │
         ▼ 土生金
    金（SIMON-方法）
         │
         ▼ 金生水
    水（GUANYIN-状态）
         │
         ▼ 水生木
    木（CONFUCIUS-伦理）
         │
         ▼ 木生火
    火（HUINENG-顿悟）
         │
         ▼ 火生土
    回到 土（价值验证）
```

**相生含义**:
- **土生金**: 价值清晰才能方法得当
- **金生水**: 方法得当才能心态从容
- **水生木**: 心态从容才能培育合伙伦理
- **木生火**: 伦理根基成熟触发顿悟决策
- **火生土**: 决策验证回归价值本质

### 1. LIU - 刘禹锡（土）
- **核心精神**: 惟吾德馨

### 2. SIMON - 西蒙（金）
- **核心精神**: 满意解
- **定位**: 决策工具包、有限理性框架
- **校准问题**: 是否接受不完美？

### 3. GUANYIN - 观自在（水）

## 使用指南

### 社交媒体发布
- 使用 PNG 格式文件
- 适合平台: 微信公众号、微博、LinkedIn、Twitter/X
- 建议配文: 五路图腾体系介绍 + 五行相生关系说明

### 专家头像替换
1. 打开 SVG 源文件
2. 定位到每个图腾卡片中的头像占位区域
3. 替换 `<g transform="translate(90, -50)">` 内的占位图形
4. 使用圆形裁剪 (`clipPath`) 保持头像圆形

### 颜色定制
- 修改 `<defs>` 部分的渐变色值
- 五行主色变量:
  - `earthGradient`: 土黄
  - `metalGradient`: 金银
  - `waterGradient`: 水黑
  - `woodGradient`: 木青
  - `fireGradient`: 火红
