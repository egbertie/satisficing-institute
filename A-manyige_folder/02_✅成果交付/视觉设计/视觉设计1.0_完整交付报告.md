## 使用指南

### SVG文件导出

**PNG导出（屏幕使用）**:
```bash
# 使用Inkscape命令行导出
inkscape input.svg --export-type=png --export-dpi=72 -o output.png
```

**PDF导出（打印使用）**:
```bash
# 使用Inkscape导出
inkscape input.svg --export-type=pdf -o output.pdf
```

### PPT使用

1. 直接插入SVG文件（Office 2016+支持）
2. 或先导出高分辨率PNG再插入
