## 📋 使用说明

1. **勾选方式**：每个检查项勾选【是 / 否 / 待核实】
2. **评分规则**：
   - 🟢 **绿色（通过）**：该部分80%以上勾选"是"
   - 🟡 **黄色（关注）**：该部分50%-80%勾选"是"或存在"待核实"
   - 🔴 **红色（否决/暂缓）**：该部分不足50%勾选"是"或触发红色警报

## 第一部分：基础信息核查（5项）

| 序号 | 检查项 | 是 | 否 | 待核实 | 备注/证据 |
|:----:|--------|:--:|:--:|:------:|-----------|
| B1 | **股权结构清晰度**<br>创始人股权比例明确，无代持/隐名股东 | ☐ | ☐ | ☐ | |
| B2 | **过往创业经历**<br>已核实过往创业项目及退出情况 | ☐ | ☐ | ☐ | |
| B3 | **技术/商业背景匹配度**<br>核心能力与项目需求高度契合 | ☐ | ☐ | ☐ | |
| B4 | **资金来源稳定性**<br>个人财务状况健康，无重大债务 | ☐ | ☐ | ☐ | |
| B5 | **家庭支持度**<br>配偶/家庭成员了解并支持创业决定 | ☐ | ☐ | ☐ | |

**本部分评分**：□ 绿（4-5项是） | □ 黄（2-3项是或有待核实） | □ 红（0-1项是）

## 第二部分：早期风险信号（6项）

| 序号 | 检查项 | 是 | 否 | 待核实 | 备注/证据 |
|:----:|--------|:--:|:--:|:------:|-----------|
| E1 | **股权设计合理性**<br>避免"正浩式稀释"（承诺5-15%→实际不足1%）<br>⚠️ 参考：正浩创新案例 | ☐ | ☐ | ☐ | |
| E2 | **决策机制明确性**<br>已书面约定重大决策流程（一致同意/多数决/CEO最终决策） | ☐ | ☐ | ☐ | |
| E3 | **价值观对齐度**<br>对产品底线、客户标准、发展节奏达成共识 | ☐ | ☐ | ☐ | |
| E4 | **退出条款清晰度**<br>股东协议中包含vesting（建议4年+1年cliff）、回购条款 | ☐ | ☐ | ☐ | |
| E5 | **利益分配公平性**<br>股权/期权分配方案经所有创始人签署确认 | ☐ | ☐ | ☐ | |
| E6 | **沟通模式健康度**<br>已建立定期同步机制（建议每周核心会+每月深度会） | ☐ | ☐ | ☐ | |

**本部分评分**：□ 绿（5-6项是） | □ 黄（3-4项是或有待核实） | □ 红（0-2项是）

> 💡 **案例警示**：
> - **正浩创新**：VIE架构下股权承诺与实际严重不符，创始人股权从5-15%稀释至不足1%
> - **纳微科技**：技术路线分歧导致董事会频繁2:1僵局，最终分道扬镳

## 📎 附录：使用场景说明

### 场景一：首次接触（Coffee Chat阶段）
- **目标**：快速筛选，识别明显风险
- **使用方式**：优先检查红色警报（R1-R4），同步了解基础信息（B1-B5）
- **时间**：30分钟内完成

### 场景二：尽职调查（DD阶段）
- **目标**：全面评估，形成准入决策
- **使用方式**：完整填写所有4部分，收集证据支撑每项判断
- **时间**：1-2周完成

### 场景三：定期复盘（季度评估）
- **目标**：动态监控，预防风险演化
- **使用方式**：重点复查第二部分（早期风险信号）和第三部分（关键人风险）
- **时间**：每季度1次，每次2-3小时
