# 满意解理论体系 V2.0 速查卡

## 四层次架构
```
价值层(儒商哲学) → 认知层(前景理论) → 方法论层(满意解) → 执行层(工具)
```

## 核心理论公式
```
决策质量 = f(有限理性, 认知偏差管理, 信息完备度, 时间约束, 价值对齐)
```

## 三支柱
- **理性基石**：西蒙满意解 + 博弈论基础
- **认知校准**：前景理论 + 双系统理论
- **价值锚定**：儒商哲学 + 五路图腾
