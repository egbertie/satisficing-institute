## 一、律所合作话术包

### 场景1: 初次接触 (电话/微信)

**时长**: 2-3分钟
**目标**: 引起兴趣，获得面谈机会

**完整话术**:

```
【开场白】
王主任您好，我是Egbertie。咱们之前可能没打过交道，但我关注贵所很久了。
今天冒昧联系，是想聊聊一个咱们律所客户普遍面临、但现有服务很难解决的问题。

【停顿，等待回应】

【痛点共鸣】
您服务的科技型企业客户里，有没有遇到过这种情况——
两个创始人一开始情同手足，股权五五开，结果公司做起来后闹分家，
撕逼打官司，最后公司垮了、项目黄了？

【停顿，观察反应】

【案例佐证】
我上周刚接触一个案例：南山这边一个AI团队，天使轮拿了500万，
两个创始人大学同学，一个做技术、一个做商务，股权对半分。
一年后，技术创始人觉得商务合伙人"不懂技术瞎指挥"，
商务创始人觉得技术合伙人"太理想化不懂市场"。
最后闹到对簿公堂，公司散伙，投资人的钱也打了水漂。

【价值展示】
我专门做"硬科技创业团队合伙人匹配决策"，
就是在他们找合伙人、分股权的时候介入，
用银行风控的方法帮他们把把关，避免后期的股权纠纷。

【合作邀约】
您手里有没有那种技术很牛、但公司治理一塌糊涂的初创团队？
我可以免费给他们做一次"合伙人健康体检"，就当是给您客户的增值服务。
如果后续他们有需要深度服务的，咱们可以谈合作分成。

【下一步】
您这周有时间吗？我想当面跟您详细聊聊这个模式。
```

### 场景2: 深度交流 (面谈)

**时长**: 30-45分钟
**目标**: 建立信任，明确合作模式

**完整话术**:

```
【开场破冰】
王主任，感谢您抽时间见面。我刚才在路上还在想，
律所和合伙人决策服务，看似是两个领域，其实解决的是同一个问题——
都是在"人"的层面上帮企业规避风险。

【深入痛点】
您处理的股权纠纷案子里，有没有发现什么规律？
我觉得大部分问题其实不是在纠纷发生时才出现的，
而是在最开始找合伙人、分股权的时候，就已经埋下了隐患。

【等对方分享案例】

【方法论介绍】
我做了一套TRL-PFI评估模型：
- TRL是技术成熟度，判断公司现在处于什么阶段
- PFI是合伙人匹配度，从技术理解力、阶段匹配、文化适配、风险共担、市场资源五个维度评估

举个例子：一个TRL 4-5阶段的技术团队（刚有原型），
需要的是一个"能听懂技术的工程化合伙人"，
而不是"大厂出来的销售总监"。
找错人，后期一定会出问题。

【合作模式】
我想跟贵所建立三种合作：
1. **增值服务**：您推荐客户来免费体检，我给您客户的增值服务
2. **深度服务**：如果客户需要深度评估，我给您10-15%的服务费返佣
3. **联合服务**：复杂案件可以联合服务，您负责法律、我负责决策评估

【案例数据】
我已经服务了X个案例，其中Y个是股权设计阶段介入，
到目前为止还没有出现后期纠纷的。当然，样本还不够大，
但我相信这个方向是对的。

【下一步】
您觉得这个模式可行吗？我们可以先从一个试点客户开始。
```

### 场景3: 合作推进 (跟进)

**时长**: 10-15分钟
**目标**: 促成首次合作

**完整话术**:

```
【开场】
王主任，上次聊完后我一直在想咱们说的那个合作。
我这周整理了一个具体的合作方案，想跟您汇报一下。

【具体方案】
我准备这样启动：
1. 我先给咱们律所的3-5个重点客户免费做"合伙人健康体检"
2. 体检报告发您一份，您看看质量
3. 如果有客户需要深度服务，咱们再谈具体分成

【降低门槛】
第一次合作，我不收任何费用，就当是建立信任。
您看这周方便安排吗？

【处理异议】
如果对方说"考虑一下"：
完全理解，毕竟这是个新模式。您看这样行不行——
我先发几个案例报告给您看看，您评估一下质量，
觉得有价值咱们再推进，没价值就当交个朋友。

如果对方说"客户可能不需要"：
我觉得不是不需要，是不知道有这个方法。
很多创始人找合伙人全靠感觉，出了问题才想起来找律师。
咱们是在前置环节帮他们把风险挡住，这对客户是价值，
对律所也是增值服务。

【收尾】
您看咱们先从哪个客户试点？
```

## 七、通用话术模块

### 7.1 处理常见异议

**异议1: "我不需要这个服务"**
> "我理解，大部分创始人觉得自己能搞定。但数据显示，硬科技创业团队找合伙人的平均周期是3-6个月，找到的人一年流失率高达40%。您确定要把时间浪费在试错上吗？"

**异议2: "价格太贵了"**
> "3万块确实不是小数目。但您想想，找错合伙人的成本是多少？股权纠纷诉讼费、时间成本、机会成本，可能几百万都不止。3万块是帮您规避几百万的风险。"

**异议3: "我考虑考虑"**
> "完全理解，毕竟这是个重要决策。您看这样行不行——我先免费给您做一次'合伙人健康体检'，您体验一下服务质量，觉得有价值咱们再谈后续，没价值就当交个朋友。"

**异议4: "我现在不急"**
> "很多创始人都是等到出问题了才想起来找我们。但合伙人决策就像治病，预防的成本远低于治疗。现在花3万块做评估，可能帮您省下将来300万的纠纷成本。"

### 7.2 金句库

- "数据越来越便宜，判断越来越贵。"
- "找错合伙人的成本，远高于找对合伙人的成本。"
- "银行风控防的是'钱'的风险，我们防的是'人'的风险。"
- "不是算命，是风控。"
- "在信息不全时做出有据可依的决策。"
- "满意解，不是最优解，是足够好的解。"
- "你的直觉知道答案，我们只是帮你听见它。"
