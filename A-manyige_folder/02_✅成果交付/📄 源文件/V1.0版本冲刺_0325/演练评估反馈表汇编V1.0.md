## 📊 使用说明

### 谁填写哪张表？

| 表单 | 填写人 | 填写时机 |
|------|--------|----------|
| 顾问表现评估表 | 观察员/教练 | 演练结束后 |
| 客户扮演者反馈表 | 客户扮演者 | 演练结束后 |
| 观察员记录表 | 观察员 | 演练过程中+结束后 |
| 演练场次综合评估表 | 主持人/PM | 复盘后 |
| FAQ更新记录表 | 记录员 | 每场结束后 |

### 数据如何使用？

1. **个人成长**：顾问查看自己的评估表，针对性改进
2. **团队复盘**：综合评估表用于集体讨论
3. **流程优化**：FAQ表持续更新服务标准
4. **上岗认证**：顾问评分≥4.0方可服务真实客户
