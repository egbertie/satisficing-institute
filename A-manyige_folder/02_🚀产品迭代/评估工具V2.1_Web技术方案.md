# 合伙人评估工具V2.1 - Web界面化技术方案

## 产品目标
将V2.0命令行/脚本工具转化为Web应用，支持：
- 浏览器直接访问
- 交互式问卷填写
- 实时结果展示
- 报告下载分享

## 技术栈选型

### 方案A: 纯前端静态站点（推荐）
**技术栈**: HTML5 + Tailwind CSS + Vanilla JavaScript
**优点**:
- 零后端，部署简单（GitHub Pages/Netlify/Vercel）
- 加载速度快
- 免费托管
- 可离线使用（PWA）

**缺点**:
- 无持久化存储
- 无用户系统

**适用场景**: MVP验证，快速上线

### 方案B: 轻量级后端
**技术栈**: Python Flask/FastAPI + SQLite + Tailwind
**优点**:
- 支持数据持久化
- 可保存历史评估记录
- 支持用户账号

**缺点**:
- 需要服务器
- 部署成本

**适用场景**: 正式产品化，需要数据积累
