# 创意提案管理

> **创意收集、审议、采纳流程**

---

## 🎯 提案流程

```
创意产生 → 提交提案 → 初步审议 → 深度评估 → 决策 → 执行/归档
```

---

## 📋 提案状态

| 状态 | 含义 | 图标 |
|------|------|------|
| 待审议 | 刚提交，等待评估 | ⏳ |
| 审议中 | 正在评估可行性 | 🔍 |
| 已采纳 | 通过审议，准备执行 | ✅ |
| 已归档 | 不采纳或已执行完毕 | 📦 |

## 📝 提案模板

提交新提案时，请在 `待审议/` 文件夹创建文件：

文件名格式: `提案_YYYYMMDD_简称.md`

内容模板:
```markdown
# 创意提案: [标题]

## 提案人
- 姓名:
- 日期:

## 创意描述
简要描述创意内容

## 预期价值
- 对项目的价值:
- 对客户的价值:

## 可行性评估
- 技术可行性:
- 资源需求:
- 时间估算:

## 建议下一步
建议采取的行动
```
