## 🖼️ 文件使用技巧

### 1. Markdown文件 (.md)
- **用途：** 文案、报告、方案
- **打开方式：** 直接点击预览，或复制内容到任何文本编辑器
- **转PDF：** 用Typora、VS Code等工具导出

### 2. SVG图片 (.svg)
- **用途：** 五路图腾等矢量图（无限放大不模糊）
- **打开方式：**
  - **方法A：** 用浏览器直接打开（把文件拖到浏览器窗口）
  - **方法B：** 复制SVG代码，粘贴到 [SVG Viewer](https://www.svgviewer.dev/) 在线预览
  - **方法C：** 用AI设计软件（Illustrator、Figma）打开编辑

### 3. CSV表格 (.csv)
- **用途：** 数据、进度表、监控看板
- **打开方式：**
  - **Excel：** 双击直接用Excel打开
  - **Numbers：** Mac用户可用
  - **在线：** 粘贴到 [CSV Viewer](https://www.convertcsv.com/view-csv-online.htm)
