## 📐 五行相生关系

```
    土生金 ───→ 金生水
    ↑              │
    │              ↓
  火生土 ←──── 木生火
```

### 相生原理详解

| 关系 | 原理 | 精神寓意 |
|------|------|----------|
| **土生金** | 大地孕育矿藏 | 品德修养孕育智慧结晶 |
| **金生水** | 金属凝结水露 | 理性分析滋养感性智慧 |
| **水生木** | 水分滋养草木 | 智慧慈悲滋养仁爱之心 |
| **木生火** | 草木燃烧生火 | 仁爱之心激发顿悟智慧 |
| **火生土** | 火烬归于大地 | 智慧实践回归品德修养 |

## 🧩 图腾详情

### 1. LIU - 刘禹锡

```
🏛️
LIU
刘禹锡
━━━━━
🟫 土 | Earth
━━━━━
惟吾德馨
```

- **出处**: 《陋室铭》"斯是陋室，惟吾德馨"
- **核心精神**: 品德高尚，不以外物喜忧
- **现代启示**: 在物质匮乏中保持精神富足

### 2. SIMON - 赫伯特·西蒙

```
⚖️
SIMON
赫伯特·西蒙
━━━━━
⬜ 金 | Metal
━━━━━
满意解
```

- **出处**: 诺贝尔经济学奖得主，提出"满意解"(Satisficing)理论
- **核心精神**: 在现实约束下寻求足够好的解决方案
- **现代启示**: 避免过度优化，追求实用主义

### 3. GUANYIN - 观自在菩萨

```
🪷
GUANYIN
观自在菩萨
━━━━━
🔵 水 | Water
━━━━━
自在从容
```

- **出处**: 《心经》"观自在菩萨，行深般若波罗蜜多时"
- **核心精神**: 观察内在，获得自在解脱
- **现代启示**: 在变化中保持内心平静

### 4. CONFUCIUS - 孔子

```
🎋
CONFUCIUS
孔子
━━━━━
🟢 木 | Wood
━━━━━
仁者爱人
```

- **出处**: 《论语》"仁者爱人"
- **核心精神**: 以仁爱之心对待他人
- **现代启示**: 构建和谐人际关系的基石

### 5. HUINENG - 六祖慧能

```
🔥
HUINENG
六祖慧能
━━━━━
🔴 火 | Fire
━━━━━
顿悟/知行合一
```

- **出处**: 《六祖坛经》"菩提本无树，明镜亦非台"
- **核心精神**: 顿悟见性，知行统一
- **现代启示**: 理论与实践相结合

## 🔧 使用方法

### 查看信息图

1. **直接查看**: 使用浏览器打开 `.svg` 文件
2. **编辑修改**: 使用文本编辑器修改 `.mmd` 文件

### 重新生成SVG

```bash
# 安装 mermaid-cli
npm install -g @mermaid-js/mermaid-cli

# 生成SVG
mmdc -i 五路图腾信息图.mmd -o 五路图腾信息图.svg -b white -w 1200 -H 1200

# 生成PNG
mmdc -i 五路图腾信息图.mmd -o 五路图腾信息图.png -b white -w 1200 -H 1200
```
